# 八卦属性详解

> **核心原则**：八卦是万物之象的原型——天地水火雷风山泽，8种基本象涵盖了宇宙万物的基本运动模式。64卦是八卦的两两组合，共64种原型模式。

> **设计思想**：八卦 = 类（8种基本类），属性德 = 属性，自然象/方位/季节 = 属性展开，取象判定 = 方法，应用场景 = 路由

## 一、☰ 乾（天·健）

### 基本属性
- **卦名**：乾（qián）
- **自然象**：天
- **属性德**：健（刚健不息）
- **五行**：金
- **方位**：西北
- **季节**：秋冬之交
- **阴阳**：阳卦（三爻皆阳，至阳之象）

### 象征体系
| 维度 | 对应 |
|------|------|
| 身体 | 首（头） |
| 动物 | 龙 |
| 人物 | 君、父、大人、创始人 |
| 家庭 | 父亲 |
| 器物 | 圆形物、金属、玉石、钟表 |
| 颜色 | 大赤、金白 |

### 取象判定特征
**取乾象的条件**（满足2条以上）：
- 事物具有**创造/创始**的性质
- 事物表现出**刚健不息、自强向上**的状态
- 事物处于**主导/领导**地位
- 事物具有**纯粹、圆满**的特征
- 事物的运动是**周而复始、永不停息**的

### 现代应用场景
- 领导力与创始（创业、开创性项目）
- 强盛期的组织或个人
- 刚健直行的品格
- 周期性运转的系统
- 纯粹的理论或原则

### 卦德详解
**健**：乾之德为健，"天行健"——天的运行刚健不息，从不停顿。健不是刚硬对抗，而是**持续不断的创造力**。乾卦六爻皆阳，象征纯粹的阳气，代表宇宙间最根本的创造力。

## 八卦之间的关系

### 先天八卦（伏羲八卦）—— 对立统一
```
       ☰乾（南）
   ☱兑        ☴巽
 ☲离            ☵坎
   ☳震        ☶艮
       ☷坤（北）
```
- 乾↔坤（天↔地）：创造与承载
- 坎↔离（水↔火）：深沉与光明
- 震↔巽（雷↔风）：触发与渗透
- 艮↔兑（山↔泽）：止息与交流

### 后天八卦（文王八卦）—— 流转循环
```
       ☴巽（东南）  ☲离（南）   ☷坤（西南）
 ☳震（东）                         ☱兑（西）
       ☶艮（东北）  ☵坎（北）   ☰乾（西北）
```
- 东方震（春·动）→ 南方离（夏·丽）→ 西方兑（秋·悦）→ 北方坎（冬·陷）→ 回到东方
- 代表事物的生长收藏循环

### 八卦的生克关系
- **相生**：乾坤→震巽→坎→离→艮兑→乾坤（木火土金水循环）
- **相克**：坎克离（水克火），离克乾兑（火克金），乾兑克震巽（金克木），震巽克坤艮（木克土），坤艮克坎（土克水）
