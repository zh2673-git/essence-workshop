# 选题基因规范（Topic Selection Gene System）

> 基于三层基因系统的选题诊断与生成系统。
> 适用于公众号管线和视频号管线的选题环节，覆盖政策/财经、知识科普、学术分析、社会情绪等多类内容。

---

## 核心公式

```
爆款选题 = 传播层（让人点进来并转发） × 内容层（让人读完有获得感） × 风格适配层（能按目标风格展开）

传播层 = T1(与我有关) + T2(情绪杠杆) + T3(社交货币) + T4(信息缺口)
内容层 = C1(钱袋子) + C2(制度变迁) + C3(权属之争) + C4(空白→填补) + C5(时效性) + C6(逻辑拆解)
风格适配层 = S1(问题意识) + S2(证据链) + S3(反直觉空间)  [论文风格必备]
```

> **设计说明**：原 6 基因系统作为内容层保留，新增传播层覆盖打开与转发机制，新增风格适配层衔接 `writing-style-guide.md` 的论文风格输出约束。选题阶段只判断"能不能写成论文风格"，不写形式规范。

---

## 步骤 -1b：前置三维定位

在基因评分之前，先完成以下三维定位，避免选题脱离读者与平台。

```
步骤 -1b：三维定位
├── 读者维度：目标读者是谁？核心焦虑与身份诉求是什么？
├── 平台维度：内容主要在微信哪个场景传播？（朋友圈 / 群聊 / 推荐流 / 搜一搜）
└── 情绪维度：选题希望触发哪种情绪？（焦虑 / 愤怒 / 好奇 / 认同 / 优越感 / 希望）
```

| 维度 | 关键问题 | 输出格式 |
|---|---|---|
| **读者维度** | 谁需要读这篇文章？他们的职业、年龄、痛点、身份标签是什么？ | 1-2 句话读者画像 |
| **平台维度** | 朋友圈传播需要社交货币，群聊传播需要实用信息，推荐流传播需要强钩子 | 选择主要传播场景 |
| **情绪维度** | 这篇文章想让读者感到什么？情绪是转发的重要燃料 | 选择 1-2 种主情绪 |

## 三层基因定义

### 第一层：传播基因（传播层）

传播基因决定选题是否能被看见、被点击、被转发，适用于所有内容领域。

| 基因 | 一句话 | 触发机制 | 检验标准 |
|------|--------|---------|---------|
| **T1 与我有关** | 读者 0.5 秒内感到"这事跟我有关系" | 自我相关性优先于一切 | 读者看标题就能判断相关性 |
| **T2 情绪杠杆** | 触发可被转发放大的集体情绪 | 情绪降低分享决策成本 | 读完后有"必须让 XX 也看看"的冲动 |
| **T3 社交货币** | 转发能帮助读者表达"我是谁" | 身份认同与形象管理 | 读者愿意把它作为社交资本分享 |
| **T4 信息缺口** | 读者知道一点但想知道更多 | 好奇心驱动点击 | 能制造"原来如此"的认知增量 |

### 第二层：内容基因（内容层）

内容基因决定选题是否有足够的骨架支撑成文，原 6 基因系统归入此层。

| 基因 | 一句话 | 触发机制 | 检验标准 |
|------|--------|---------|---------|
| **C1 钱袋子** | 选题直击读者钱包 | 损失厌恶 > 获得偏好 | 读者会觉得"这事关我的钱" |
| **C2 制度变迁** | 从旧制度到新制度的演化叙事 | 长时间跨度制造戏剧张力 | 能画出10年以上的时间线 |
| **C3 权属之争** | "到底归谁"的归属争议 | 争议性天然引发站队和讨论 | 存在至少两方对立立场 |
| **C4 空白→填补** | 从无法可依到有法可依的进程 | "原来现在有法了"的认知增量 | 能找到"从无到有"的里程碑 |
| **C5 时效性** | 绑定新规/新法时间节点 | FOMO（错失恐惧）驱动点击 | 有明确的新政策/新法规锚点 |
| **C6 逻辑拆解** | 把复杂制度拆成可理解的逻辑链 | 降低认知门槛，获得掌控感 | 能用3-5步把复杂制度讲清楚 |

### 第三层：论文风格适配基因（风格适配层）

风格适配基因用于判断选题是否能支撑论文风格输出，与 `writing-style-guide.md` 形成衔接。

| 基因 | 一句话 | 检验标准 | 不满足的处理 |
|------|--------|---------|-------------|
| **S1 问题意识** | 能提出一个值得被回答的具体问题 | 能用一句话说清"这篇文章在回答什么问题" | 改为科普/随笔风格，或换题 |
| **S2 证据链** | 能找到 2-3 个可溯源的核心证据 | 有论文、数据、法条、案例等可引用材料 | 降级为观点文，不用论文风格 |
| **S3 反直觉空间** | 结论与常识有张力，值得论证 | 能说出"看似 XX，实则是 YY"的核心判断 | 用平铺直叙风格，或增加论证深度 |

> **注意**：风格适配层只在用户明确选择论文风格时启用。若选择对话风格或蒸馏 Skill 风格，此层可跳过。

## 选题诊断流程

### 触发条件

- 用户说「写公众号」但没有给具体话题
- 用户说「帮我选题」「不知道写什么」
- 用户给了一个模糊方向（如"想写医保""想写教育"）
- 用户明确指定论文风格输出

### 诊断步骤

```
步骤-1a：判断用户是否有明确话题
├── 有明确话题 → 跳到步骤-1b
└── 没有明确话题 → 步骤-1b 中追加 1-2 轮追问

步骤-1b：前置三维定位
├── 读者维度：目标读者是谁？
├── 平台维度：主要传播场景是什么？
└── 情绪维度：希望触发什么情绪？

步骤-1c：论文风格判定
├── 用户是否选择论文风格？
│   ├── 是 → 启用 S1/S2/S3 检查
│   └── 否 → 跳过风格适配层

步骤-1d：三层基因评分
├── 传播基因 T1-T4 评分
├── 内容基因 C1-C6 评分
└── 论文风格基因 S1-S3 评分（若启用）

步骤-1e：评级与决策
├── S级 → 直接开写
├── A级 → 开写，补强弱项
├── B级 → 基因补强后再写
└── C级 → 建议换话题

步骤-1f：选题验证（可选，推荐执行）
├── 事前预测：预估打开率 / 转发率 / 完读率
├── 小范围测试：朋友圈 / 小群投放
├── 数据回测：与历史爆款对比基因组合
└── 沉淀：将结果写入选题知识库
```

## 基因评分表

### 传播基因评分（T1-T4）

| 基因 | 0分 | 1分 | 2分 |
|------|-----|-----|-----|
| T1 与我有关 | 读者无法判断相关性 | 与部分读者相关 | 目标读者一眼就知道"与我有关" |
| T2 情绪杠杆 | 无明显情绪 | 有情绪但偏弱 | 能触发强烈转发冲动 |
| T3 社交货币 | 转发不能表达身份 | 有一定表达价值 | 转发能明确表达立场/品味/身份 |
| T4 信息缺口 | 信息已众所周知 | 有少量新信息 | 能制造显著"原来如此"感 |

### 内容基因评分（C1-C6）

| 基因 | 0分 | 1分 | 2分 |
|------|-----|-----|-----|
| C1 钱袋子 | 与个人财产无关 | 间接影响个人财产 | 直接影响，读者能算出省/亏多少 |
| C2 制度变迁 | 无制度变迁 | 有变迁但跨度<5年 | 跨度≥10年，有清晰演化脉络 |
| C3 权属之争 | 无争议 | 有争议但不够鲜明 | 存在明确对立立场，读者会站队 |
| C4 空白→填补 | 无里程碑 | 有里程碑但感知弱 | "从无到有"的突破，读者有"终于有了"的感觉 |
| C5 时效性 | 无新规锚点 | 有新规但>6个月 | 6个月内有新规/新法落地 |
| C6 逻辑拆解 | 无法拆解 | 可拆解但不复杂 | 复杂制度可拆成3-5步讲清楚 |

### 论文风格适配评分（S1-S3）

| 基因 | 0分 | 1分 | 2分 |
|------|-----|-----|-----|
| S1 问题意识 | 没有问题焦点 | 问题较宽泛 | 问题具体且值得回答 |
| S2 证据链 | 无可溯源证据 | 有少量二手证据 | 有 2-3 个可溯源核心证据 |
| S3 反直觉空间 | 完全符合常识 | 有一定新意 | 结论与常识有张力，值得论证 |

### 评级标准

```
传播层满分 8 分，内容层满分 12 分，风格适配层满分 6 分。

综合评级：
├── S级：传播层 ≥6 且 内容层 ≥9 且 风格适配层 ≥4（若启用） → 直接开写
├── A级：传播层 ≥4 且 内容层 ≥7 且 风格适配层 ≥3（若启用） → 开写，补强弱项
├── B级：传播层 ≥2 且 内容层 ≥4 → 需要基因补强后再写
└── C级：低于 B 级 → 建议换话题
```

> **简化场景**：若用户未指定论文风格，可只用传播层 + 内容层（满分 20 分），S/A/B/C 阈值相应调整为 15/12/8。

## 基因补强策略

### 传播层补强

| 缺失基因 | 补强动作 |
|---------|---------|
| 缺 T1 | 在标题/引言中明确点出目标读者身份或痛点 |
| 缺 T2 | 找到话题中的焦虑点、争议点或希望点 |
| 缺 T3 | 设计能让读者表达立场或品味的观点 |
| 缺 T4 | 挖掘读者"知道一点但不知道全部"的信息差 |

### 内容层补强

| 缺失基因 | 补强动作 |
|---------|---------|
| 缺 C1 | 找到"这事关你多少钱"的切入点 |
| 缺 C2 | 拉长时间线，找制度演化脉络 |
| 缺 C3 | 寻找对立立场，制造"归谁"的争议 |
| 缺 C4 | 寻找"从无到有"的里程碑节点 |
| 缺 C5 | 绑定最近的政策/法规时间点 |
| 缺 C6 | 设计"3步看懂""5分钟理清"的拆解路径 |

### 论文风格适配层补强

| 缺失基因 | 补强动作 |
|---------|---------|
| 缺 S1 | 把宽泛话题收窄为一个具体的研究问题 |
| 缺 S2 | 补充学术论文、权威数据、原始法条、典型案例 |
| 缺 S3 | 寻找主流观点的反例或反常数据 |

## 话题生成策略

当用户没有明确话题时，按以下策略生成候选：

### 策略1：领域扫描法

```
1. 确定用户关注的领域（如医疗、教育、养老、数据、房产、税收、AI、认知科学）
2. 搜索该领域最近6个月的政策/法规/学术/市场动态
3. 对每个变化点，用三层基因快速评分
4. 选出得分最高的2-3个
```

### 策略2：基因反推法

```
1. 从最强的传播基因出发（通常是 T1 + T4 组合）
2. 问：目标读者最近最想知道但还不知道什么？
3. 对找到的话题，检查内容基因和风格适配基因
4. 不满足的尝试补强
```

### 策略3：历史爆款延伸法

```
1. 回顾用户历史爆款文章的基因组合
2. 寻找同基因组合在不同领域的映射
3. 验证新领域是否有足够的素材
4. 推荐同基因跨域话题
```

### 策略4：情绪扫描法（新增）

```
1. 确定目标读者圈层
2. 扫描该圈层最近 30 天的集体情绪（焦虑 / 愤怒 / 认同 / 希望）
3. 找到情绪背后的具体事件或制度变化
4. 用内容基因和风格适配基因筛选可写成文章的话题
```

## 论文风格选题适配

若用户明确选择论文风格，在选题阶段执行以下检查：

```
论文风格选题检查清单：
├── [ ] 能提出一个具体、可回答的问题
├── [ ] 能找到 2-3 个可溯源的核心证据
├── [ ] 结论与常识有张力，值得论证
├── [ ] 能按"背景 → 论点 → 证据 → 反驳 → 结论"五段式展开
└── [ ] 标题能同时满足专业性与传播性
```

**论文风格选题的判断标准**：

| 适合论文风格 | 不适合论文风格 |
|---|---|
| 有明确学术/行业问题 | 只有情绪，没有问题 |
| 有可溯源的证据链 | 全是个人观点，无材料支撑 |
| 结论有反直觉空间 | 结论显而易见，无需论证 |
| 能经得起"为什么"三层追问 | 一问就到底，没有展开深度 |

> **与输出阶段的衔接**：选题阶段只判断"适不适合"，形式规范（五段式、去口语化、参考文献 ≤5 条）保留在 `writing-style-guide.md` 中处理。

## 选题→标题转化

### 标题公式库

| 类型 | 基因组合 | 标题公式 | 示例 |
|-----|---------|---------|------|
| 政策财经 | C1+C5+C2 | [主题]全解：从[旧状态]到[新规年份]新规 | 土地增值税全解：从税率计算到2026新规 |
| 权属争议 | C3+C2+C4 | [主题]到底归谁？从[空白期]到[里程碑]，[时间]权属之争全梳理 | 小区地下车位到底归谁？从法律空白到民法典，二十年权属之争全梳理 |
| 制度拆解 | C2+C6 | [大主题]与[子主题]：[N]大制度的[时间]纠缠与未来破局 | 中国土地、税收与城镇化：三大制度的百年纠缠与未来破局 |
| 里程碑梳理 | C4+C6 | 全国首部[级别][主题]法规逻辑梳理 | 全国首部省级12345热线法规逻辑梳理 |
| 新规影响 | C1+C3+C5 | [主题]归谁？[年份]新规下，[具体影响] | 小区公共收益归谁？2026严查风暴下，七类收入终于算清了 |
| **论文风格** | **S1+S3+S2** | **[反直觉结论]：[常规认知]错在哪？** | **我们高估了学区房：教育回报率的十年真相** |
| **问题意识** | **S1+T4** | **[领域]有一个被忽视的问题** | **中国医保改革中，一个被忽视的道德风险** |
| **反直觉** | **S3+T4** | **为什么 [看起来对的观点] 其实是错的** | **为什么"多刷题"正在毁掉学生的创造力** |
| **情绪共鸣** | **T2+T1** | **[群体]的 [痛点]，没人敢说** | **35 岁程序员的困境：不是能力，是系统** |
| **社交货币** | **T3+T4** | **真正懂 [领域] 的人，都 [怎么做]** | **真正懂投资的人，都不看 K 线** |
| **未来推演** | **T4+C5** | **[趋势] 十年后，[什么] 会消失** | **通用人工智能十年后，这五种职业会先消失** |
| **信息缺口** | **T4+C6** | **[领域]的 [现象]，90% 的人理解错了** | **个人所得税的 90% 的人理解错了抵扣逻辑** |

### 标题禁忌

- ❌ 不用"震惊""重磅""赶紧看"等标题党词汇
- ❌ 不用问号结尾超过1个
- ❌ 不用纯情绪化表达，必须有信息量
- ✅ 标题本身就是一个完整的信息单元
- ✅ 读者看标题就能判断"这篇文章跟我有没有关系"

## 选题验证与迭代闭环

选题确定后，进入验证与迭代流程：

```
步骤-1f：选题验证闭环
├── 事前预测
│   ├── 预估打开率（基于 T1 + T2 + T4）
│   ├── 预估转发率（基于 T2 + T3）
│   └── 预估完读率（基于 C6 + S1）
├── 小范围测试
│   ├── 朋友圈/小群投放标题+摘要
│   ├── 收集 3-5 条真实反馈
│   └── 判断是否需要调整角度
├── 数据回测
│   ├── 与历史爆款对比基因组合
│   ├── 找出同基因组合的高/低表现案例
│   └── 更新对当前选题的预期
└── 沉淀
    ├── 将选题、基因组合、实际表现写入 topic-knowledge-base.md
    └── 为下一轮选题提供历史爆款延伸素材
```

**验证标准**：

| 指标 | 测试方法 | 通过标准 |
|------|---------|---------|
| 标题吸引力 | 小范围投票 | 3 人以上愿意点击 |
| 话题争议性 | 观察讨论 | 能引发不同观点 |
| 证据可获取性 | 预检索 | 30 分钟内找到 2-3 个核心证据 |
| 论文可展开性 | 写 200 字摘要 | 能自然引出 3-4 个分论点 |

## 选题与内容输出的衔接

选题确定后，三层基因不仅指导选题，还指导内容结构：

| 基因 | 对应内容章节 |
|------|------------|
| T1 与我有关 | 引言第一句：直接点出读者身份或痛点 |
| T2 情绪杠杆 | 引言中段：用具体场景放大情绪共鸣 |
| T3 社交货币 | 结语：给出读者愿意转发的核心观点 |
| T4 信息缺口 | 引言结尾：预告读者将获得的认知增量 |
| C1 钱袋子 | 引言：用具体数字建立"这事关你的钱" |
| C2 制度变迁 | 核心章节：时间线叙事，旧制度→新制度 |
| C3 权属之争 | 核心章节：对立立场分析，"到底归谁" |
| C4 空白→填补 | 核心章节：里程碑节点，"从无到有" |
| C5 时效性 | 引言+结语：新规要点+行动号召 |
| C6 逻辑拆解 | 全文结构：用3-5步拆解复杂制度 |
| S1 问题意识 | 引言：明确提出文章要回答的问题 |
| S2 证据链 | 核心章节：每个论点配备可溯源证据 |
| S3 反直觉空间 | 核心章节：先立常识，再破常识，再立新论 |

这个映射关系确保：**选题时的基因评分 = 内容结构的基因覆盖**，选题和写作形成闭环。
