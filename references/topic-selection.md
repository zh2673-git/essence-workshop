# 选题基因规范（Topic Selection Gene System）

> 基于6个爆款认知基因的选题诊断与生成系统。
> 适用于公众号管线和视频号管线的选题环节。

---

## 核心公式

```
爆款选题 = 驱动层（让人点进来） × 满足层（让人读完转发）

驱动层 = G1(钱袋子) + G5(时效性) + G3(权属之争)
满足层 = G2(制度变迁) + G4(空白→填补) + G6(逻辑拆解)
```

---

## 6个爆款基因定义

| 基因 | 一句话 | 触发机制 | 检验标准 |
|------|--------|---------|---------|
| **G1 钱袋子** | 选题直击读者钱包 | 损失厌恶 > 获得偏好 | 读者会觉得"这事关我的钱" |
| **G2 制度变迁** | 从旧制度到新制度的演化叙事 | 长时间跨度制造戏剧张力 | 能画出10年以上的时间线 |
| **G3 权属之争** | "到底归谁"的归属争议 | 争议性天然引发站队和讨论 | 存在至少两方对立立场 |
| **G4 空白→填补** | 从无法可依到有法可依的进程 | "原来现在有法了"的认知增量 | 能找到"从无到有"的里程碑 |
| **G5 时效性** | 绑定新规/新法时间节点 | FOMO（错失恐惧）驱动点击 | 有明确的新政策/新法规锚点 |
| **G6 逻辑拆解** | 把复杂制度拆成可理解的逻辑链 | 降低认知门槛，获得掌控感 | 能用3-5步把复杂制度讲清楚 |

## 选题诊断流程

### 触发条件

- 用户说「写公众号」但没有给具体话题
- 用户说「帮我选题」「不知道写什么」
- 用户给了一个模糊方向（如"想写医保""想写教育"）

### 诊断步骤

```
步骤-1a：判断用户是否有明确话题
├── 有明确话题 → 跳到步骤-1c（基因评分）
└── 没有明确话题 → 步骤-1b（话题生成）

步骤-1b：话题生成
├── 追问1-2轮，定位用户关注的领域
├── 搜索该领域最新政策/法规动态
├── 用6基因筛选候选话题（见下方）
└── 推荐2-3个候选话题

步骤-1c：基因评分
├── 对候选话题逐个打6基因评分（0-2分）
├── 计算总分（满分12分）
├── S级(10-12) / A级(7-9) / B级(4-6) / C级(0-3)
└── 低于B级的话题建议更换或改造

步骤-1d：基因补强（可选）
├── 如果话题缺少某些基因，尝试补强
├── 缺G1 → 找到"这事关你多少钱"的切入点
├── 缺G2 → 拉长时间线，找制度演化脉络
├── 缺G3 → 寻找对立立场，制造"归谁"的争议
├── 缺G4 → 寻找"从无到有"的里程碑节点
├── 缺G5 → 绑定最近的政策/法规时间点
└── 缺G6 → 设计"3步看懂""5分钟理清"的拆解路径
```

---

## 基因评分表

对每个候选话题，按以下标准打分：

| 基因 | 0分 | 1分 | 2分 |
|------|-----|-----|-----|
| G1 钱袋子 | 与个人财产无关 | 间接影响个人财产 | 直接影响，读者能算出省/亏多少 |
| G2 制度变迁 | 无制度变迁 | 有变迁但跨度<5年 | 跨度≥10年，有清晰演化脉络 |
| G3 权属之争 | 无争议 | 有争议但不够鲜明 | 存在明确对立立场，读者会站队 |
| G4 空白→填补 | 无里程碑 | 有里程碑但感知弱 | "从无到有"的突破，读者有"终于有了"的感觉 |
| G5 时效性 | 无新规锚点 | 有新规但>6个月 | 6个月内有新规/新法落地 |
| G6 逻辑拆解 | 无法拆解 | 可拆解但不复杂 | 复杂制度可拆成3-5步讲清楚 |

**评级标准**：

| 等级 | 总分 | 建议 |
|------|------|------|
| **S级** | 10-12 | 直接开写，爆款概率极高 |
| **A级** | 7-9 | 开写，但注意补强弱项基因 |
| **B级** | 4-6 | 需要基因补强后再写 |
| **C级** | 0-3 | 建议换话题 |

## 话题生成策略

当用户没有明确话题时，按以下策略生成候选：

### 策略1：领域扫描法

```
1. 确定用户关注的领域（如医疗、教育、养老、数据、房产、税收）
2. 搜索该领域最近6个月的政策/法规变化
3. 对每个变化点，用6基因快速评分
4. 选出得分最高的2-3个
```

### 策略2：基因反推法

```
1. 从最强的基因出发（通常是G1+G5组合）
2. 问：最近6个月，什么新规直接影响普通人的钱袋子？
3. 对找到的话题，检查其他4个基因是否满足
4. 不满足的尝试补强
```

### 策略3：历史爆款延伸法

```
1. 回顾用户历史爆款文章的基因组合
2. 寻找同基因组合在不同领域的映射
3. 验证新领域是否有足够的政策素材
4. 推荐同基因跨域话题
```

## 选题→标题转化

选定话题后，按基因组合生成标题：

### 标题公式

| 基因组合 | 标题公式 | 示例 |
|---------|---------|------|
| G1+G5+G2 | [主题]全解：从[旧状态]到[新规年份]新规 | 土地增值税全解：从税率计算到2026新规 |
| G3+G2+G4 | [主题]到底归谁？从[空白期]到[里程碑]，[时间]权属之争全梳理 | 小区地下车位到底归谁？从法律空白到民法典，二十年权属之争全梳理 |
| G2+G6 | [大主题]与[子主题]：[N]大制度的[时间]纠缠与未来破局 | 中国土地、税收与城镇化：三大制度的百年纠缠与未来破局 |
| G4+G6 | 全国首部[级别][主题]法规逻辑梳理 | 全国首部省级12345热线法规逻辑梳理 |
| G1+G3+G5 | [主题]归谁？[年份]新规下，[具体影响] | 小区公共收益归谁？2026严查风暴下，七类收入终于算清了 |

### 标题禁忌

- ❌ 不用"震惊""重磅""赶紧看"等标题党词汇
- ❌ 不用问号结尾超过1个
- ❌ 不用纯情绪化表达，必须有信息量
- ✅ 标题本身就是一个完整的信息单元
- ✅ 读者看标题就能判断"这篇文章跟我有没有关系"

## 选题与内容输出的衔接

选题确定后，6基因不仅指导选题，还指导内容结构：

| 基因 | 对应内容章节 |
|------|------------|
| G1 钱袋子 | 引言：用具体数字建立"这事关你的钱" |
| G2 制度变迁 | 核心章节：时间线叙事，旧制度→新制度 |
| G3 权属之争 | 核心章节：对立立场分析，"到底归谁" |
| G4 空白→填补 | 核心章节：里程碑节点，"从无到有" |
| G5 时效性 | 引言+结语：新规要点+行动号召 |
| G6 逻辑拆解 | 全文结构：用3-5步拆解复杂制度 |

这个映射关系确保：**选题时的基因评分 = 内容结构的基因覆盖**，选题和写作形成闭环。
