# 元素层规范（Element Specification）

> 元素层是三层交付架构的底层——所有管线的原子资产来源。元素是自包含、可寻址、可复用、跨管线共享的。

---

## 元素分类

### 1. 文本元素（Text）

| 子类型 | 格式 | 说明 |
|--------|------|------|
| 标题 | Markdown H1-H6 | 文章/幻灯片标题 |
| 段落 | Markdown段落 | 正文内容 |
| 列表 | Markdown列表 | 有序/无序要点 |
| 表格 | Markdown表格 | 数据对比 |
| 代码块 | Markdown代码块 | 带语言标注的代码 |
| 引用 | Markdown引用 | 引述内容 |

**存放**：`output/elements/text/*.md`

**约束**：
- ⚠️ 源格式必须是 Markdown，不使用任何扩展语法
- ⚠️ H1 仅用于文章标题，正文从 H2 开始
- ⚠️ 代码块必须标注语言

### 2. 图形元素（Graphics）

| 子类型 | 格式 | 说明 |
|--------|------|------|
| SVG图谱 | .svg | 知识图谱、思维导图 |
| SVG流程图 | .svg | 流程、步骤、时序 |
| SVG架构图 | .svg | 系统架构、模块关系 |
| SVG对比图 | .svg | 多栏对比、四宫格 |
| SVG示意图 | .svg | 概念示意、模型图 |
| SVG图表 | .svg | 数据可视化 |

**存放**：`output/elements/graphics/*.svg`

**约束**：
- ⚠️ 所有图形元素首选 SVG 格式
- ⚠️ SVG 必须设置 viewBox 属性
- ⚠️ SVG 必须包含 xmlns 命名空间
- ⚠️ 文字使用 `<text>` 标签，不使用 `<foreignObject>`
- ⚠️ 字体使用系统字体（sans-serif），不依赖外部字体
- ⚠️ 颜色使用 CSS 变量或品牌色，便于管线覆写

**SVG 生成决策树**：

```
内容是什么？
├── 结构/关系/对比 → SVG静态图
│   ├── 流程/步骤 → SVG流程图
│   ├── 系统架构 → SVG架构图
│   ├── 多维对比 → SVG对比图
│   └── 概念模型 → SVG示意图
├── 变化/过程/循环 → SVG动画（见动画元素）
├── 数据/统计 → SVG图表
└── 知识体系 → SVG图谱
```

### 3. 动画元素（Animation）

| 子类型 | 格式 | 说明 |
|--------|------|------|
| SVG动画 | .svg（SMIL） | 简单动画，跨管线兼容性最好 |
| CSS动画 | .css | 配合SVG/HTML使用 |
| Canvas帧序列 | .js | 复杂动画，视频号管线首选 |

**存放**：`output/elements/animations/`

**约束**：
- ⚠️ 简单动画优先使用 SVG SMIL（兼容性最好）
- ⚠️ 复杂动画使用 Canvas（仅视频号和HTML交互管线支持）
- ⚠️ 每个动画元素必须有静态降级方案（关键帧截图）

**动画技术选择决策树**：

```
动画复杂度？
├── 简单（淡入/移动/缩放）→ SVG SMIL
├── 中等（多步/路径动画）→ CSS animation + SVG
└── 复杂（粒子/物理/大量帧）→ Canvas
```

### 4. 音频元素（Audio）

| 子类型 | 格式 | 说明 |
|--------|------|------|
| TTS旁白 | .mp3 | Edge TTS 生成的语音 |
| BGM | .mp3 | 背景音乐 |
| 音效 | .mp3 | 转场/提示音效 |

**存放**：`output/elements/audio/`

**约束**：
- ⚠️ 音频格式统一使用 MP3
- ⚠️ TTS 旁白每段对应一个镜头/章节
- ⚠️ BGM 音量必须低于旁白（-10dB 以下）

### 5. 交互元素（Interaction）

| 子类型 | 格式 | 说明 |
|--------|------|------|
| 拖拽组件 | .html + .js | 可拖拽排序/匹配 |
| 折叠面板 | .html + .js | 渐进式展开内容 |
| 路由匹配 | .html + .js | 场景路由交互 |
| 标准模块 | modules/ | 预制交互组件 |

**存放**：`output/elements/interactions/`

**约束**：
- ⚠️ 交互元素仅 HTML交互管线完整支持
- ⚠️ 其他管线消费交互元素时必须降级为静态截图
- ⚠️ 交互元素必须有无 JS 降级方案

### 6. 数据元素（Data）

| 子类型 | 格式 | 说明 |
|--------|------|------|
| 结构数据 | .json | 幻灯片顺序、章节结构 |
| 配置参数 | .json | 品牌色、字体、间距 |
| 状态对象 | .json | 交互状态、进度 |

**存放**：`output/elements/data/`

**约束**：
- ⚠️ 数据元素是管线间的协调器
- ⚠️ 结构数据驱动所有管线的章节拆分
- ⚠️ 配置参数统一品牌风格

---

## 元素生命周期

```
1. 生成：场景A/B/C/D产出 → 风格改写 → 拆解为元素
2. 存储：按类型存入 output/elements/ 对应子目录
3. 读取：各管线从元素层读取所需元素
4. 转换：管线按平台约束转换元素格式
5. 交付：转换后的元素组装为平台成品
```

## 元素复用规则

- 同一元素可被多条管线消费
- 元素生成一次，管线切换时无需重新生成
- 修改元素后，所有消费该元素的管线输出都需要重新生成
- 元素版本通过 Git 管理

## 质量自检

| 产出类型 | 检查项 | 标准 | 异常处理 |
|---------|--------|------|---------|
| SVG | viewBox | 非空 | 缺失则补充 |
| SVG | xmlns | 存在 | 缺失则添加 |
| SVG | 文字溢出 | 文字在框内 | 溢出则调整框高度 |
| SVG→PNG | 字体 | 非衬线体 | 出现衬线体说明未用Playwright |
| SVG→PNG | 清晰度 | 2x DPI | 模糊则检查 device_scale_factor |
| GIF | 文件大小 | ≥100KB且≤3MB | <100KB说明动画未生效 |
| MP3 | 采样率 | 22050Hz+ | 过低则重新生成 |
| JSON | 语法 | 合法JSON | 语法错误则修复 |
