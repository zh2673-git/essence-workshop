# 蒸馏Skill模板

> 本模板用于场景B（人物蒸馏）的Phase 3构建阶段。
> 完整模板详见 [references/skill-template.md](../references/skill-template.md)

## 使用方式

1. 读取 `references/skill-template.md` 获取完整模板
2. 将Phase 2三阶蒸馏结果填入模板对应位置
3. 确保遵循坡度理解呈现（30秒→2分钟→按需深入）
4. 确保路由机制可执行
5. 确保根因追溯体现在Skill中

## 填充检查清单

- [ ] frontmatter 的 name 和 description 已正确填写
- [ ] 坡度1（30秒核心）：一句话定义 + 身份卡 + 心智模型一览
- [ ] 坡度2（2分钟框架）：模型关系图 + 场景路由表 + 根因追溯摘要
- [ ] 坡度3（按需深入）：角色扮演规则 + 详细模型 + 决策启发式 + 场景路由 + 表达DNA
- [ ] 价值观与反模式
- [ ] **反例与黑名单**（必须包含"不要做什么"清单，详见下方）
- [ ] 智识谱系
- [ ] 人物时间线
- [ ] Few-shot示例（Phase 4.5生成后写入）
- [ ] 诚实边界
- [ ] 调研来源附录

## 反例与黑名单（必填章节）

> **设计依据**：借鉴 Darwin Skill dim9 + SkillLens 实证——有"不要做什么"清单的Skill质量显著更高。
> 详见 [references/skill-antipatterns.md](../references/skill-antipatterns.md)

在Skill模板的"价值观与反模式"章节之后，必须增加"反例与黑名单"章节，格式如下：

```markdown
### 反例与黑名单

| # | 反模式 | 为什么不要做 | 替代做法 |
|---|--------|-------------|---------|
| 1 | [此人物绝对不会做的事] | [为什么这违背其思维] | [此人会怎么做] |
| 2 | [此人物绝对不会做的事] | [为什么这违背其思维] | [此人会怎么做] |
| ... | | | |
```

**填充规则**：
- 至少列出5条反模式
- 反模式必须具体可执行（不是"不要做坏事"这种空话）
- 每条反模式必须对应一个此人的核心心智模型或价值观
- 反模式是"此人绝对不会做的事"，不是"此人偶尔会做的事"
