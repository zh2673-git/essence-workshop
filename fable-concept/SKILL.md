# 概念寓言化 · Skill

## 核心方法

坡度理解三阶段 × 寓言叙事：先用寓言建立联系（阶段2），再点出概念（阶段1收束），最后详细解释并迁移（阶段3）。

```
输入概念 → 概念脱水（提取本质）→ 寓言生成（阶段2建立联系）→ 点破概念（阶段1收束）→ 正式解释+迁移问题（阶段3）
```

## 触发条件

- 「用寓言讲XX」「XX寓言化」「给XX编个故事」
- 「用故事解释XX」「坡度寓言XX」「fable XX」
- 其他场景（A/A2/B/E/E2）需要"建立联系"载体时调用本Skill生成寓言元素

## 输出

一则寓言故事（300-800字）+ 概念点破句 + 正式解释 + 2个迁移问题。可作为元素层资产被内容输出管线复用。
