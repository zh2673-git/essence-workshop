# 概念寓言化 · Skill

## 核心方法

三阶合一 × 寓言叙事：先用本质定义点出本质（阶段1），再用寓言建立因果并触发反常识根因（阶段2），最后展开迁移并视角升级（阶段3）。寓言的内核由本质定义锚定，反差/反转对应反常识根因，寓意收尾对应视角升级。

```
输入概念 → 本质定义先行（阶段1点出本质·本质检验·一句话点破）→ 寓言生成（阶段2建立因果·反常识根因）→ 迁移+视角升级（阶段3）
```

## 触发条件

- 「用寓言讲XX」「XX寓言化」「给XX编个故事」
- 「用故事解释XX」「坡度寓言XX」「fable XX」
- 其他场景（A/A2/B/E/E2）需要"建立因果"载体时调用本Skill生成寓言元素

## 输出

本质定义 + 一则寓言故事（300-800字）+ 概念点破句 + 寓言→概念映射表（回溯到本质）+ 2个迁移问题 + 视角升级句。可作为元素层资产被内容输出管线复用。
