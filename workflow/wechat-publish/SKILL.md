# 微信公众号适配器

> 将通用HTML/Markdown适配为公众号受限HTML。配色/风格/配图内容由上层指定，本适配器只做平台约束执行、格式转换与素材上传。

## 输入与输出

| 项 | 说明 |
|----|------|
| 输入 | Markdown/HTML 格式产物 + 配图素材（SVG/PNG/GIF/封面图，来自 format） |
| 输出 | 公众号受限HTML（推送草稿箱） |

## 平台约束（关键差异）

| 约束项 | 说明 |
|--------|------|
| CSS/JS | ❌ 不支持外部CSS/JS → 样式必须内联 |
| 图片 | 必须上传素材库后用素材URL引用 |
| 总字符数 | ≤20000（超出走智能样式去重） |
| 容器 | 禁止 `:::block` 容器语法 |
| 标题 | 正文不用 `# H1`（H1由标题位占用） |
| 元信息 | 文章不得提及蒸馏方法论/本质工坊工作流程 |

## 主题系统（配色由上层指定）

配色由上层通过 `--brand-spec` 参数指定。未指定时使用默认配色：
- 浅色背景 `#FFFFFF` + 深色文字 `#333333` + 强调色 `#D97757`（陶土色）
- 本适配器负责将指定配色以内联样式应用到公众号HTML

## 配图约束（风格由上层指定）

风格由上层通过 `--style` 参数指定，本适配器按风格约束执行配图数量与字数检查。配图内容（SVG/PNG/GIF）由 format 生成，本适配器只做平台格式检查与上传。

- 按风格分流：论文7张 / 专栏5张 / 故事5张 / 教程4张 / 观点3张 / 连载3-5张
- GIF 不可省略且 ≥100KB；GIF 必须与所有 PNG 内容主题都不雷同
- 连续无图文字 ≤1500字

### 封面图（平台必选约束）

- 1240×770 wide 画幅（公众号平台硬性要求）
- 必须与正文配图完全不同
- 封面图内容由上层提供，本适配器负责尺寸约束检查与上传
- 推送参数 `--cover`

### GIF平台约束

- GIF 格式要求：≥100KB 且 ≤3MB（公众号平台限制）
- GIF 必须与所有 PNG 内容主题不雷同
- GIF 内容生成由 format 处理，本适配器负责格式检查与上传

## 字数约束（风格由上层指定）

风格由上层通过 `--style` 参数指定，本适配器按风格执行字数检查：

| 风格 | 字数范围 |
|------|----------|
| 论文 | 7000-8000 |
| 专栏 | 3000-5000 |
| 故事 | 2500-4500 |
| 教程 | 1500-3000 |
| 观点 | 1200-2500 |
| 连载 | 2000-4000 |

## 适配流程

```
1. 读取格式产物（Markdown/HTML + SVG/PNG配图 + 封面图，均来自 format）
2. SVG → PNG（2x DPI，公众号不渲染SVG）
3. 样式内联化（Markdown/HTML → 公众号受限HTML）
4. 配图插入正文
5. 图片上传素材库，替换为素材URL
6. 字数检查（按上层指定的风格分流）
7. 封面图尺寸检查（1240×770）
8. 推送草稿箱（必须带 --cover）
```

## 失败模式

| 失败 | 处理 |
|------|------|
| 封面图缺失 | 推送报 `[40007]invalid media_id` → 必须先由上层提供封面图 |
| GIF <100KB | 动画未生效 → 重新录制或增加帧数 |
| 总字符 >20000 | 智能样式去重 |
| 参考文献 >5条 | 移除并替换为行内备注 |

## 质量自检

- [ ] 封面图已就位（1240×770，与正文配图完全不同，由上层提供）
- [ ] GIF ≥100KB 且 ≤3MB，且与所有 PNG 主题不雷同
- [ ] 样式全部内联，无外部CSS/JS引用
- [ ] 字数符合风格分流区间
- [ ] 图片已上传素材库并以素材URL引用
- [ ] 正文未出现 `# H1`、`:::block` 容器
- [ ] 文章未提及蒸馏方法论/本质工坊工作流程
- [ ] 参考文献 ≤5条
- [ ] 推送命令带 `--cover` 参数

## 脚本

```
wechat/
└── scripts/
    ├── client.py            # 素材库/草稿箱客户端
    ├── converter.py         # Markdown/HTML → 公众号受限HTML
    └── publish.py           # 推送草稿箱
```

命令示例：

```
python -m scripts.cli publish article.md --cover output/wechat/images/cover.png --author "公众号名"
```
