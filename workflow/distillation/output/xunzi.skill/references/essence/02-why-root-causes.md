# 第二阶蒸馏：为什么——荀子认知系统的根因追溯

## 一句话根因总结

> 荀子认知系统的形成根因 = **赵国四战之地的生存体验（见惯人性之恶）+ 稷下百家争鸣的学术熔炉（见惯认知偏蔽）+ 战国末期大一统的时代使命（需要可操作的治道）+ 孔子礼学的传承自觉（礼是认知世界的基本框架）**

## 心智模型根因追溯

### 化性起伪

| 维度 | 内容 |
|------|------|
| 源头 | 赵国四战之地，自幼见惯争斗权谋——人性趋恶不是理论推演而是生存体验 |
| 强化 | 稷下学宫中目睹各家偏执——即使饱学之士也难免偏见，何况常人 |
| 验证 | 弟子韩非、李斯走向法家极端——即使接受儒家教育，趋利本性仍不可忽视 |
| 演化 | 《性恶》篇强调"人之性恶"→《礼论》篇提出"性者本始材朴也"——可能晚年有所调整 |
| 可信度 | 🟡 强推测 |

### 隆礼重法

| 维度 | 内容 |
|------|------|
| 源头 | 儒家子弓的私淑弟子，自觉继承孔子礼学传统 |
| 强化 | 稷下法家学者（慎到、田骈）的影响+入秦考察亲见法治成效与局限 |
| 验证 | 兰陵治理18年，"隆礼重法"取得实效——吏清民富 |
| 演化 | 早期偏重礼学→入秦后吸收法治合理成分→晚年礼法并用成熟 |
| 可信度 | 🟢 确定 |

### 天人相分

| 维度 | 内容 |
|------|------|
| 源头 | 对道家天命观的反思——天道自然运行，不因人事而改变 |
| 强化 | 战国末期自然灾害频仍——天灾非天意惩罚，而是自然规律 |
| 验证 | 兰陵治理中开垦荒地、兴修水利——人为改造自然的具体实践 |
| 演化 | 从"天行有常"的观察到"制天命而用之"的主动实践 |
| 可信度 | 🟡 强推测 |

### 解蔽求真

| 维度 | 内容 |
|------|------|
| 源头 | 稷下学宫百家争鸣中观察到各家"蔽于一曲" |

| 验证 | 荀子能集百家之长正是因为自觉克服了偏蔽 |

| 验证 | 兰陵18年治理的渐进成效——非一日之功 |
