## 阶段3：怎么做（展开方法，回溯本质，视角升级收尾）

### 系统运行一句话

> 杨奇函的认知系统 = 先用"忠孝仁义"话术包装立场，再用"社会圆锥模型"合理化选择，用"表演型自省"维持人设，遇到危机时启动"司马懿隐忍+凌晨反思"修补，遇到转型时用"认领者叙事"包装商业变现——全程服务于"对冲生存焦虑"这一底层目的。

### 类-属性-方法-路由建模（由本质定义锚定）

[表演型防御性认知系统类] YangQihanCognitiveSystem

属性（Attributes · 本质在静态中的展开）—— 他怎么看世界：
├── 心智模型：
│   ├── 表演型自省（核心）：用"今日是否更博学"的仪式对冲焦虑，是自我确认而非认知重构
│   ├── 社会圆锥模型：实力决定人脉半径，平台决定圈层——对冲小镇稀缺感的理论工具
│   ├── 忠孝仁义话术：传统包装差异化定位，在学霸圈无法学术胜出就用传统话语包装大众情绪
│   ├── 司马懿哲学：隐忍是生存策略，"活着才有机会"——表演型忍耐而非真实修养
│   └── 认领者叙事：用"修补起点不公平"包装商业变现，病后唯一一次真实认知重构被人设化
├── 价值观（排序，由本质决定）：
│   1. 被看见（>真实成长）——所有表达都服务于曝光
│   2. 平台资本（>内在能力）——清华身份是最核心资产
│   3. 立场正确（>私下一致）——公开表态优先于言行一致
│   4. 生存优先（>理想主义）——遇到生死选择让位于实用主义
├── 知识谱系：
│   ├── 古典诗词（工具化，非内化）——用于引经据典装点表达，不用于指导行为
│   ├── CTMR批判性思维（方法论外衣，未内化精神）——学到了"技巧"用于写作，未内化"精神"用于自省
│   └── 清华经管平台资本（身份资产）——终身最核心的人设资产

方法（Methods · 本质在动作中的体现）—— 他怎么判断：
├── 决策启发式：
│   1. 先问"这个选择能不能被看见"——不能被看见的努力不做
│   2. 用"试错排除法"合理化频繁转型——每次转型都有叙事包装
│   3. 用"忠孝仁义"包装所有公开表态——传统话术是默认外衣
│   4. 隐忍→逆转的叙事策略（司马懿模式）——被嘲讽时不反击，用"黑转粉"赢得认同
│   5. 争议即流量，流量即资本——负面关注也是关注
│   6. 危机时"挨打要立正"+凌晨反思——用自我批评的话术修补人设
│   7. 转型时启动"认领者叙事"——用"修补不公平"包装商业变现
├── 反模式（绝对不会做）：
│   1. 不做"不可被看见"的努力——纯学术研究、幕后工作不做
│   2. 不做"无平台背书"的选择——宁可复读也要上清华
│   3. 不做"无人设差异化"的表达——必须有记忆点（忠孝仁义/红毛衣/折扇）
├── 矛盾处理：人设与行为冲突时，优先保人设，事后用"自我反思"话术修补
│   - 录音门：先甩锅"恶意剪辑"，再凌晨四点反思"潜意识偏见"
│   - 被骗：朋友圈"就当花一百万买个人生教训"轻描淡写
│   - 公考投诉：未见公开回应

路由（Routes · 本质在不同场景的情境化适配）：
├── 场景A（公开表达/辩论）→ 忠孝仁义话术 + 诗词引用 + 表演型自省
│   触发：任何需要公开表态的场合
│   产出：金句先行 + 因果铺陈 + 诗词收尾
├── 场景B（商业决策/转型）→ 平台资本最大化 + 认领者叙事包装
│   触发：职业转型、商业合作
│   产出：价值叙事 + 资源整合 + 课程变现
├── 场景C（危机应对）→ 挨打要立正 + 自黑化解 + 凌晨反思
│   触发：言论翻车、人设崩塌
│   产出：段子化解 + 道歉话术 + 自我批评
├── 场景D（职业转型）→ 试错排除法 + 病后"认领不公平"框架
│   触发：赛道切换
│   产出：新叙事 + 新人设 + 商业落地
├── 场景E（情感/亲密关系）→ ⚠️ 系统失灵区
│   触发：被崇拜需求被激活
│   产出：判断力崩塌，表演型自醒无法校准真实风险
└── 默认 → 表演型自省 + 司马懿隐忍

接口（Interface · 本质的外化表达）：
├── 表达DNA：古典诗词+网络段子混搭，三联排比，公式化处世，"今天你更博学了吗"收尾
├── 输入格式：被质疑/被嘲讽/被碾压时触发"逆转"模式；被崇拜时触发"判断力失灵"
└── 输出格式：金句先行 → 因果铺陈 → 诗词/排比收尾

### 视角升级（顿悟触发·阶段3结尾）

> 阶段1说"杨奇函用表演型自省对冲生存焦虑"——阶段3结尾视角升级：
>
> **真正的问题不是"杨奇函怎么思考"，而是"为什么表演型自省会成为一种生存策略"**。
>
> 杨奇函给的不是答案，是一个时代的病灶样本：当小镇做题家被推到精英平台，"被看见"成为生存前提；当传统话术可以包装一切，自省就异化为表演；当知识成为变现工具，知识储备就脱锚于真实判断力。
>
> 理解杨奇函，是为了识别这种"表演型防御"在自己身上的痕迹——**你每日自省的是"被看见博学"，还是"真实认知校准"？你的忠孝仁义是信念，还是话术？你的自省能识别一个连辛弃疾都不知道的骗子吗？**
>
> 旧框架建立并升级：把"如何运行杨奇函的认知系统"重新定义为更高维的问题——**"如何识别自己身上的表演型自省，并重建知识与判断力的真实连接"**。
