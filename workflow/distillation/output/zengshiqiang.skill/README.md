# 曾仕强 · 认知操作系统

> **用易经的"持经达变"把任何二元对立都"二看成三"为"合理"的防御性认知系统**——其驱动力不是博学，而是39岁危机的痛感记忆（"单一思维会害死自己"）。

## 核心心智模型

| 模型 | 一句话 | 适用场景 |
|------|--------|---------|
| 持经达变 | 经（道）不变，权（方法）可变——以"合理"为判准 | 任何二元张力的化解（管理/易经/人际/生死） |
| 二看成三 | 二选一→二合一为合理——把对立转化为流动 | 任何二元对立场景（中西/对错/好坏/法治人治） |
| 修己安人 | 修己是因，安人是果——管理是修己安人的历程 | 组织/家庭管理 |
| 情理法 | 法是过去，情是未来，理是现在——依理应变 | 中国情境决策 |
| 圆通非圆滑 | 圆通=技巧+伦理，圆滑=技巧-伦理——以"持经达变"为判准 | 中国式沟通 |

## 一句话点破

> **用一套易经的"持经达变"，把人类单一思维的偏差变成可被对抗的系统性缺陷**——本质不是国学大师，是痛感驱动的认知防御系统设计者。

## 反常识根因

> 表面看曾仕强博学（管理+易经+儒家+道家+历史），所以能讲"持经达变""二看成三"——博学是因，多元是果。
>
> **本质看**：曾仕强39岁前"满脑子都是西方的东西"，单一思维（西化）差点要了他的命（5种病+人际危机）。这次危机让他形成**对"单一思维会害死自己"的痛感记忆**——这才是他构建"持经达变"防御系统的真正驱动力。博学不是因，是"对自身认知偏差的恐惧"驱动的防御性构建。

## 视角升级

> 真正的问题不是"曾仕强怎么思考"，而是**"如何为自己构建一套对抗单一思维偏差的认知免疫系统"**——曾仕强给的不是答案，是一套可被任何人复用的"二看成三"认知防御设计模板。其价值不在对错，而在提醒：**单一思维（无论中西、无论科学与玄学、无论法治与人治）都会要命**。

## 诚实边界

此 Skill 的局限：
- 学界接受度低（"文化布道者"而非严格学者）
- 理论实证性不足（修己安人、中庸等概念缺乏操作定义）
- 案例陈旧（多为胡雪岩、台塑等传统案例）
- 反科学立场张力（"太相信科学是人类的悲哀"被批评）
- 死后伪造语录严重（2019年后大量AI合成"预言""催财术"）

不应替代：现代科技企业管理决策、学术研究、法治化协作场景、紧急决策、专业医疗决策。
