# 老子 · 他者视角与批评

## 正面评价

1. **孔子**："其犹龙邪"——变化莫测，深不可测
2. **黑格尔**：承认老子拥有"玄妙的哲学"，但将"道"简化为"纯粹无"（误读）
3. **海德格尔**：晚年与萧师毅合译《道德经》，试图以"道"突破存在论局限
4. **尼采**："《道德经》像一个永不枯竭的井泉，满载宝藏"
5. **联合国教科文组织**：《道德经》外译数量仅次于《圣经》，全球发行量超5亿册

## 主要批评

### 1. 反智主义与愚民

- 李零（北京大学）："它也提倡复古，它也崇拜圣人，它也主张愚民。愚民的伎俩更狡猾，也更高明。"
- 第65章"古之善为道者，非以明民，将以愚之"——直白的愚民主张
- 第3章"常使民无知无欲"——无论如何解释，"无知"二字难以消解

### 2. 消极避世

- 王夫之在《老子衍》中系统批判老子的消极退让
- "无为"被误解为"什么都不做，躺平就能成功"
- 实际上河上公注："无为者，循道而行，不妄为也"——非消极不作为

### 3. 过度夸大柔弱

- "兵强则灭，木强则折"——但历史上弱者被灭的例子远多于强者
- 过于强调"反者道之动"的必然性，忽视了转化的条件性

### 4. 理想化上古

- "小国寡民"的理想社会被批评为乌托邦式的倒退幻想
- "绝圣弃智"的复古主张与文明发展相悖

### 5. 文本争议

- 郭店楚简本与传世本差异显著：简本无激烈反儒内容
- "绝圣弃智"在简本中作"绝智弃辩"——早期版本可能更温和
- "大器晚成"在帛书本作"大器免成"——颠覆传统解读

## 学术争议焦点

1. "道"是实存还是境界？陈鼓应主张实存性，牟宗三主张主观境界
2. "无为"是消极还是积极？河上公注为"不妄为"，王弼注为"顺自然"
3. 老子其人是否真实存在？顾颉刚"层累说"质疑，考古发现支持春秋末年成书
