# Agent 1: 著作与系统思考

## 核心著作

### 1. 《传习录》（心学总集）
- **性质**：语录体著作，弟子徐爱、陆澄、薛侃、钱德洪等记录
- **结构**：上卷（讲学语录，阳明审阅）、中卷（论学书信八封）、下卷（晚年语录）
- **核心命题**：心即理、知行合一、致良知、四句教
- **地位**：阳明学教典，东亚思想经典，与《论语》《孟子》并列必读

### 2. 《大学问》（晚年定论）
- **性质**：对《大学》的重新解读
- **核心**：格物=正心，"明明德"就是致良知的实践
- **地位**：系统阐述"致良知"与"格物致知"关系

### 3. 《王文成公全书》（38卷全集）
- **内容**：奏疏、书信、诗歌、语录
- **重要部分**：《文录》（政治军事思想）、《别录》（心学书信）、《外集》（诗文）

### 4. 《武经七书评》（兵法评注）
- **性质**：早年评注《孙子兵法》等兵书
- **特点**：把心学融入兵法，用兵核心是"心定"

## 反复出现的核心论点（≥3次=真信念）

### 1. 心即理
- **原文**："心即理也。天下又有心外之事，心外之理乎"
- **复现**：《传习录》上卷、中卷书信、龙场悟道后所有讲学
- **含义**：所有道理都在心里，别向外求

### 2. 知行合一
- **原文**："知是行之始，行是知之成。知而不行，只是未知"
- **复现**：贵阳首揭（1509）、南赣剿匪实践、平定宁王后反思
- **含义**：知道必须做到，做到才是真懂

### 3. 致良知
- **原文**："吾生平讲学，只是致良知三字"
- **复现**：江右以后（1520）专提、晚年所有讲学、四句教
- **含义**：良知是天生的是非心，做事凭良心、守底线

### 4. 四句教（晚年总结）
- **原文**："无善无恶心之体，有善有恶意之动，知善知恶是良知，为善去恶是格物"
- **出处**：天泉桥论学（1527），赴广西前
- **含义**：心的本体无善恶→念头动有善恶→良知能分辨→做善去恶是修行

### 5. 破心中贼
- **原文**："破山中贼易，破心中贼难"
- **复现**：南赣剿匪后（1518）、军事与修身类比
- **含义**：最难是克己，外在敌人易除，内在私欲难破

### 6. 事上磨练
- **原文**："人须在事上磨练做功夫，乃有益。若只好静，遇事便乱，终无长进"
- **复现**：教导弟子、军事实践、政治决策
- **含义**：修心不是逃避现实，而是主动迎击生活难题

### 7. 立志
- **原文**："志不立，天下无可成之事"
- **复现**：《教条示龙场诸生》、所有讲学开端
- **含义**：立志是成事根本

## 自创术语

| 术语 | 定义 |
|------|------|
| 心即理 | 心是宇宙万物的本体，天理就在人心中 |
| 知行合一 | 知与行不可分割，真知必然付诸实践 |
| 致良知 | 发掘和培养人的内在良知 |
| 心外无物 | 离开心的认知，万物不存在意义 |
| 此心不动 | 内心有主，不被外物扰动 |
| 随机而动 | 心定之后，精准捕捉时机行动 |
| 事上磨练 | 在实际事务中修炼心性 |
| 心中贼 | 内在的私欲、恐惧、焦虑 |
