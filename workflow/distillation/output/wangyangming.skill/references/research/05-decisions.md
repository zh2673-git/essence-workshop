# Agent 5: 决策记录与行动

## 重大决策记录

### 1. 抗疏救戴铣（1506，35岁）
- **背景**：宦官刘瑾专权，御史戴铣等上疏请诛刘瑾，被逮下诏狱
- **决策**：挺身而出，抗疏力救
- **结果**：被廷杖四十，贬为贵州龙场驿驿丞
- **决策逻辑**：遵从本心，明知会触怒权贵仍坚持正义
- **事后反思**：无后悔，龙场期间反而悟道
- **言行一致**：✅ 与"致良知"高度一致

### 2. 龙场悟道（1508，37岁）
- **背景**：贬谪龙场，环境恶劣，随从病倒，面临生死考验
- **决策**：日夜端居澄默，思考"圣人处此，当有何道"
- **结果**：大彻大悟"圣人之道，吾性自足，向之求理于事物者误也"
- **决策逻辑**：在极端困境中向内求索，而非向外抱怨
- **言行一致**：✅ 与"心即理"高度一致

### 3. 南赣剿匪（1516-1518，45-47岁）
- **背景**：南赣地区匪患数十年，官府屡剿屡败
- **决策**：
  - 先清内奸，推行"十家牌法"切断匪徒情报网
  - 声东击西，放弃最近的土匪，突袭最远防备最松懈的詹师富部
  - 首创"教卒之制"，精选骨干高强度训练
- **结果**：半年平定数十年匪患
- **决策逻辑**：先治本（切断情报）再治标（军事打击），知行合一
- **言行一致**：✅ 与"知行合一""事上磨练"高度一致

### 4. 平定宁王之乱（1519，48岁）
- **背景**：宁王朱宸濠十万大军叛乱，王阳明手中无正规军
- **决策**：
  - 虚张声势：散布朝廷十六万大军南下的假情报
  - 避实击虚：不救安庆，直捣南昌老巢
  - 围点打援：在鄱阳湖设伏，火攻破敌
  - 功成身退：将宁王交付张永，称病归隐
- **结果**：35天平定叛乱，生擒宁王
- **决策逻辑**："此心不动，随机而动"——心定才能看清全局，顺势而为
- **言行一致**：✅ 与"此心不动""致良知"高度一致

### 5. 广西平叛（1527-1528，56-57岁）
- **背景**：广西断藤峡、八寨地区长期扰攘
- **决策**：
  - 优先招抚受压迫的本地武装
  - "以贼攻贼"策略，利用其熟悉地形优势
  - 承诺战后妥善安置
- **结果**：以极小代价平定困扰明廷数十年的边患
- **决策逻辑**：攻心为上，不战而屈人之兵
- **言行一致**：✅ 与"致良知""破心中贼"高度一致

### 6. 天泉桥传四句教（1527，56岁）
- **背景**：赴广西前，弟子钱德洪、王畿对学说理解有分歧
- **决策**：传"四句教"作为最终定论
- **结果**：成为阳明学最终总结，但也引发后世争议
- **决策逻辑**：在生前统一学说，防止曲解
- **言行一致**：⚠️ 四句教本身存在内在张力（无善无恶 vs 知善知恶）

## 言行一致/不一致分析

### 高度一致
- **心即理**：一生向内求索，不随波逐流
- **知行合一**：所有军事、政治决策都是心学的实践
- **致良知**：抗疏救戴铣、平定叛乱都以"为民"为出发点
- **事上磨练**：在极端困境中修炼心性，而非逃避

### 潜在张力
- **四句教**：无善无恶的心体 vs 知善知恶的良知——内在逻辑张力
- **权术使用**：平叛中使用假情报、离间计——与"心体光明"的张力
- **功成身退**：平定宁王后称病归隐——是"致良知"还是政治自保？
