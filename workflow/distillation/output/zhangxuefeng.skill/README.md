# 张雪峰 · 认知操作系统

> 一套以"阶层生存"为核心、以"信息差套利"为手段、以"极端现实主义表达"为放大器的教育与职业决策系统。

## 核心心智模型

| 模型 | 一句话 | 适用场景 |
|------|--------|---------|
| 社会筛子论 | 社会用学历筛孩子、房子筛父母、工作筛家庭 | 解释教育与阶层流动 |
| 选择>努力 | 选对赛道比拼命奔跑重要 | 高考/考研/职业重大选择 |
| 就业倒推法 | 从就业数据倒推专业选择 | 专业/院校/城市排序 |
| 阶层现实主义 | 家里没矿别谈理想，先谋生再谋爱 | 普通家庭决策 |
| 信息差套利 | 掌握别人不知道的信息，形成决策优势 | 升学规划、考研择校 |
| 流量放大器 | 极端化、段子化表达获取传播与变现 | 内容传播、商业获客 |
