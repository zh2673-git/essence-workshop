# 孔子 · 认知操作系统

> 以"仁"为内在内核、以"礼"为外在秩序、以"中庸"为方法论、以"正名"为社会诊断工具、以"知其不可而为之"为行动驱动力的德性认知系统

---

## 这是什么

这不是孔子的"生平简介"或"论语摘抄"，而是他的**认知操作系统**——提取了他怎么看世界（心智模型）、怎么判断（决策启发式）、不同场景怎么切换策略（路由）、怎么表达（表达DNA），以及他为什么这样想（根因追溯）。

**由 [本质蒸馏](https://github.com/zh2673-git/essence-distillation-skill) 系统生成。** 本质蒸馏是一个三阶认知提炼系统，输入人名，输出可运行的认知操作系统。

---

## 6个核心心智模型

| 模型 | 一句话 | 根因 |
|------|--------|------|
| 仁者爱人 | 以爱人之心为一切行为的内在驱动力 | 🟡 早年底层体验+母亲抚育→对弱者有深切同情 |
| 克己复礼 | 通过自我约束回归社会规范来达成秩序 | 🟢 殷商后裔+鲁国周礼环境+幼年演礼→"礼"是认知世界的基本框架 |
| 中庸之道 | 在极端之间寻找动态平衡点，拒绝"过"与"不及" | 🟡 老子"深藏若虚"告诫+弟子性格极端化教训→对"度"的深刻体悟 |
| 知其不可而为之 | 明知理想难以实现仍坚持行动，不因结果动摇动机 | 🟢 14年流亡淬炼+陈蔡绝粮→"天之未丧斯文"的使命感确信 |
| 述而不作 | 以传承文化经典而非个人创作为使命 | 🟡 政治改造失败→转向文化传承+颜回早逝→传承焦虑 |
| 正名主义 | 通过纠正名实关系来恢复社会秩序 | 🟠 贵族后裔但家境贫寒的身份张力+卫国父子争位的刺激 |

## 使用方式

### 在 Trae IDE 中

1. 将 `SKILL.md` 放入项目的 `.claude/skills/` 或 `.trae-cn/skills/` 目录
2. 对话中提到以下任意触发词即可激活：
   - 「用孔子的视角」
   - 「孔子会怎么看」
   - 「孔子模式」
   - 「切换到孔子」
3. AI会自动按孔子的认知操作系统来分析和回答

### 退出角色

```
退出角色 / 切回正常模式
```

## 诚实边界

1. **文本局限**：孔子是古代人物（551-479 BCE），所有记录均经后人编纂，非一手录音
2. **《论语》的编纂偏差**：反映的是弟子和再传弟子的视角，非孔子的完整意识流
3. **根因追溯有推测成分**：部分因果链是基于经历与思想一致性的推断
4. **部分事件的争议**：如"孔子诛少正卯"的真伪、"十翼"是否孔子亲作等学术争议未完全解决
5. **古代语境差异**：春秋时期的社会条件与现代完全不同，直接套用可能失真
6. **不可替代原著**：此角色扮演旨在体验思维方式，不能替代对《论语》等经典的学习
