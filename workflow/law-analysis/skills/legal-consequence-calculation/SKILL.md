# 法律后果量化计算 · 辅助原子技能

## 法律声明

> 本技能提供的量化计算结果仅供参考，**不构成法律意见**。实际赔偿、刑期、处罚数额受证据、地区标准、裁量情节等多重因素影响，最终应由有权机关或执业法律专业人员确定。

---

## 一、核心概念

### 1.1 四层次推导模型

```
基础后果
  ↓
调整因素（法定/酌定）
  ↓
竞合/聚合处理
  ↓
量化计算结果（区间）
```

### 1.2 民事赔偿主要项目

| 项目 | 计算依据 |
|------|----------|
| 医疗费 | 实际发生，凭票据 |
| 误工费 | 误工时间 × 收入 |
| 护理费 | 护理期限 × 护理人员收入 |
| 交通费 | 实际发生合理费用 |
| 残疾赔偿金 | 受诉法院所在地上一年度人均收入 × 伤残系数 × 赔偿年限 |
| 死亡赔偿金 | 受诉法院所在地上一年度人均收入 × 赔偿年限 |
| 精神损害抚慰金 | 由法院酌定 |

### 1.3 刑事量刑基本步骤

| 步骤 | 内容 |
|------|------|
| 确定量刑起点 | 根据基本犯罪构成事实 |
| 确定基准刑 | 考虑犯罪数额、后果、手段等 |
| 调节情节 | 自首、立功、累犯、赔偿谅解等 |
| 确定宣告刑 | 综合调节后刑期 |

### 1.4 行政处罚裁量

| 情节 | 效果 |
|------|------|
| 从轻处罚 | 在法定幅度内较低处罚 |
| 减轻处罚 | 低于法定幅度下限 |
| 从重处罚 | 在法定幅度内较高处罚 |

---

## 二、方法要点

- **确定基础后果**：根据适用规范确定基础法律后果类型
- **识别调整因素**：列出法定与酌定调整因素
- **处理竞合与聚合**：判断是否存在责任竞合或聚合
- **量化计算**：按公式逐项计算
- **输出区间**：考虑不确定因素，给出结果区间而非单一数值

---

## 三、输入/输出描述

### 输入

- `fallback-application` 或 `subsumption` 输出的法律后果类型；
- 相关事实数据（损害金额、伤残等级、犯罪数额等）。

### 输出

- 基础后果类型、调整因素（影响方向/幅度）、计算过程（项目/公式/金额或刑期）、竞合/聚合处理、计算结果（区间/最可能值/置信度）、风险提示。
- 输出格式由 format/markdown 处理，本技能不规定具体文档模板。

---

## 四、示例

**输入**：人身损害，医疗费 2 万元，误工 3 个月，月收入 8000 元，护理 1 个月，护理人员月收入 6000 元。

**输出**：

```markdown
### 三、计算过程

| 项目 | 公式/依据 | 金额 |
|------|-----------|------|
| 医疗费 | 实际发生 | 20,000 元 |
| 误工费 | 3个月 × 8,000元/月 | 24,000 元 |
| 护理费 | 1个月 × 6,000元/月 | 6,000 元 |

### 五、计算结果
- **区间**：50,000 — 55,000 元（不含精神损害抚慰金）
- **最可能值**：50,000 元
- **置信度**：中
```

## 六、失败模式与降级

| 失败场景 | 处理动作 |
|----------|----------|
| 缺少必要数据 | 列出缺失数据，给出假设条件下的区间 |
| 地区标准不明 | 标注"需按当地标准调整" |
| 情节复杂无法量化 | 给出定性分析与大致区间 |
| 涉及自由裁量过大 | 降低置信度，建议咨询专业人士 |
