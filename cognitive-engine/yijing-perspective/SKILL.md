# 易经视角 · Skill

## 核心方法（双引擎）

> 三阶合一框架定义详见 [../general-logic/SKILL.md](../general-logic/SKILL.md)。本节描述易经视角如何实现三阶段。

三阶合一 × 易经象思维：

```
阶段1（是什么·定象）—— 顿悟触发①
  取象 = 原型式本质定义
  内卦（下卦）= 事物的内在本质/根基
  外卦（上卦）= 事物的外在表现/环境
  组合 → 64卦之一 → 本质检验 → 一句话点破（用卦辞/大象传）
       ↓
阶段2（为什么·属性拆解）—— 顿悟触发②
  内外卦组合逻辑 = 因果链
  卦辞+彖传+大象传 = 属性展开
  卦象组合的反直觉智慧 = 反常识根因
       ↓
阶段3（怎么做·象的变化）—— 顿悟触发③
  动爻变卦 = 具体行动路径
  互卦 = 隐藏趋势
  错卦 = 反向视角
  综卦 = 换位视角
  视角升级 = 用"象的流动"重新定义问题
```

## 双引擎机制

```
引擎A（分析式）—— 通过属性分析推导本质
引擎B（原型式）—— 易经新增：通过取象匹配64种原型模式

协同模式：
  并行 → 两引擎同时运行，结果相互验证
  串行 → 先用引擎B快速定位原型，再用引擎A深入分析
  降级 → 当引擎B取象不确定时，回退到引擎A
```
