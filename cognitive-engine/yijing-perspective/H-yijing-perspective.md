# 场景H：易经视角（象思维认知框架）

> **双引擎锚点**：本场景的核心——将"原型式本质定义"（取象）作为与"分析式本质定义"并列的本质定义双引擎。详见 [essence-mapping.md](references/essence-mapping.md)。

> **核心思想**：世间万物均可成象，象由天地水火雷风山泽八种基本象组合而成（内外卦），共64种原型模式。通过定象→属性拆解→象的变化，回答"是什么、为什么、怎么做"。

---

## 执行流程（三阶合一×易经）

```
第1步：阶段1（是什么·定象）—— 顿悟触发①
    ├── 观察事物/现象/行为/问题的特征
    ├── 取象（匹配8种基本象：天地水火雷风山泽）
    │   ├── 内卦（下卦）= 事物的内在本质/根基/出发点
    │   │   └── 查 [8trigrams.md](references/8trigrams.md) 取象判定特征
    │   ├── 外卦（上卦）= 事物的外在表现/环境/目标
    │   │   └── 查 [8trigrams.md](references/8trigrams.md) 取象判定特征
    │   └── 取象决策树详见 [xiang-method.md](references/xiang-method.md)
    ├── 组合成64卦之一（定象）
    │   └── 查 [64hexagrams.md](references/64hexagrams.md) 获取完整卦象数据
    ├── 本质检验：此卦能否解释该事物的所有关键特征？
    │   └── 能→保留；不能→重新取象
    └── 一句话点破（顿悟触发①）
        └── 用卦辞/大象传点破本质（如"天行健，君子以自强不息"）
           ↓
第2步：阶段2（为什么·属性拆解）—— 顿悟触发②
    ├── 内外卦组合逻辑（为什么这样组合？）
    │   ├── 内卦属性 → 事物的根（是什么支撑了它）
    │   ├── 外卦属性 → 事物的境（它处于什么环境）
    │   └── 根×境 = 此事物的独特属性
    ├── 卦辞+彖传解读（因果链）
    │   └── 卦辞点明吉凶趋势，彖传解释原因
    ├── 大象传→行动启示
    │   └── "君子以..."是卦象给人的行动指南
    ├── 反常识根因（顿悟触发②）
    │   └── 卦象组合中往往藏着反直觉的智慧
    │       示例：泰卦（地天泰）——地在天上看似反常，却是通达
    │             否卦（天地否）——天在地上看似正常，却是不通
    └── 依据支撑（引用卦辞/彖传/象传原文）
           ↓
第3步：阶段3（怎么做·象的变化）—— 顿悟触发③
    ├── 确定当前爻位
    │   └── 事物当前处于6爻中的哪一爻？（初/二/三/四/五/上）
    │       ├── 初爻：事物的开始/基层/潜在因素
    │       ├── 二爻：事物的初步发展/中层/内在支撑
    │       ├── 三爻：事物的过渡阶段/中层/不稳定点
    │       ├── 四爻：事物的发展期/高层/外在表现
    │       ├── 五爻：事物的高峰/领导位/最佳状态
    │       └── 上爻：事物的终结/过度/物极必反
    ├── 动爻变卦
    │   ├── 该爻阳变阴/阴变阳 → 得到变卦
    │   ├── 变卦 = 事物的发展趋势
    │   └── 结合本卦爻辞解读当前状态，变卦解读未来走向
    ├── 互卦分析（隐藏趋势）
    │   └── 二三四爻组成下互卦，三四五爻组成上互卦
    │       → 揭示事物内部蕴含的隐藏方向
    ├── 错卦分析（反向视角）
    │   └── 阴阳全变 → 从完全相反的角度看同一个问题
    ├── 综卦分析（换位视角）
    │   └── 上下颠倒 → 站在对方立场看问题
    ├── 行动建议（由象的变化推导）
    │   └── 每个建议必须能回溯到本卦或变卦的本质
    └── 视角升级（顿悟触发③）
        └── 用"象的流动"重新定义问题
           ↓
第4步：双引擎验证（可选）
    ├── 引擎A（分析式）本质定义：如有，填写分析式结论
    ├── 引擎B（原型式）本质定义：即定象结果的一句话点破
    └── 一致性检查：两个引擎是否指向同一本质？
        ├── 一致 → 增强信心，可选用更简洁的表达
        └── 不一致 → 分析差异原因，可能是视角互补而非矛盾
```

## 取象操作指南

### 取象决策树（详见 xiang-method.md）

```
我要分析的事物...

内在根基最像什么？
├── 创造/刚健/主导 → ☰ 乾（天·健）
├── 承载/柔顺/支持 → ☷ 坤（地·顺）
├── 触发/变革/启动 → ☳ 震（雷·动）
├── 渗透/传播/渐进 → ☴ 巽（风·入）
├── 危险/深沉/流动 → ☵ 坎（水·陷）
├── 光明/依附/文明 → ☲ 离（火·丽）
├── 静止/边界/休止 → ☶ 艮（山·止）
└── 交流/喜悦/沟通 → ☱ 兑（泽·悦）
→ 定内卦（下卦）

外在环境最像什么？
（同上8选1）
→ 定外卦（上卦）

内卦+外卦组合 → 查64卦表 → 定象
→ 本质检验：此卦能否解释该事物的所有关键特征？
  ├── 能 → 保留取象
  └── 不能 → 重新判定内外卦
```

### 取象的三大维度

1. **时间维度**：事物处于什么阶段（对应6爻的位置）
2. **空间维度**：事物处于什么位置（内=根基，外=环境）
3. **关系维度**：事物与他者的关系（比/应/承/乘）

## 变卦操作指南

### 动爻变卦（详见 hexagram-changes.md）

```
1. 确定当前卦象（本卦）
2. 识别事物当前处于哪一爻的阶段
3. 该爻阳变阴/阴变阳 → 得到变卦
4. 本卦爻辞 → 解读当前状态
5. 变卦卦辞 → 解读未来走向
```

### 互卦取法

```
本卦：  上爻
        五爻 ─┐
        四爻 ─┤ 上互卦（取四三五爻）
        三爻 ─┤ 下互卦（取二三四爻）
        二爻 ─┘
        初爻

下互卦 + 上互卦 = 互卦（揭示隐藏趋势）
```

### 错卦与综卦

- **错卦**：本卦每爻阴阳互换（反向视角）
- **综卦**：本卦上下颠倒（换位视角）
- 注意：8个卦的综卦是自己（乾、坤、坎、离、颐、大过、中孚、小过）

## 失败模式编码

| 步骤 | 失败场景 | 处理动作 |
|------|---------|---------|
| 阶段1 定象 | 取象无法通过本质检验 | 重新判定内外卦，或考虑事物可能处于卦变中间态 |
| 阶段1 定象 | 内外卦判定不清晰 | 使用排除法，逐一对照8trigrams.md的取象判定特征 |
| 阶段1 定象 | 事物过于复杂，单一卦象无法覆盖 | 拆分为子问题，分别取象 |
| 阶段2 属性拆解 | 卦辞彖传解读与事物特征脱节 | 标注"解读缺口"，不强求对应 |
| 阶段2 属性拆解 | 找不到反常识根因 | 检查内外卦组合是否有反直觉之处（如泰否、既济未济） |
| 阶段3 象的变化 | 动爻位置难以确定 | 结合时间维度和事物发展阶段综合判断 |
| 阶段3 象的变化 | 变卦过多导致分析混乱 | 一次分析最多用2-3种变卦方法（动爻+互卦，或错卦+综卦） |
| 双引擎验证 | 两个引擎结论不一致 | 保留差异，分析是否为视角互补，不强求统一 |

## 检查点

- 🔴 **CHECKPOINT · 定象完成后**：确认卦象已通过本质检验、一句话点破可独立传播
- 🔴 **CHECKPOINT · 属性拆解完成后**：确认内外卦组合逻辑清晰、反常识根因已找出
- 🟡 **AUTO-CHECK · 象的变化完成后**：输出三阶分析结果摘要（含3个顿悟触发点），自动生成文档
- 🟡 **AUTO-CHECK · 双引擎验证（可选）**：如果同时使用分析式本质定义，进行一致性检查

## Few-shot 示例

### 示例1：用易经看"创业团队"

**定象**：
- 内卦（根基）：团队有激情和创新动力 → ☳ 震（雷·动）
- 外卦（环境）：市场竞争激烈，充满不确定性 → ☵ 坎（水·陷）
- 组合：震下坎上 = 第3卦**屯**（动而遇险）

**一句话点破**：初生的本质是"动而遇险"——生机与困难并存，宜建侯而不宁。

**属性拆解**：
- 内卦震（动）= 团队的创业激情和变革动力
- 外卦坎（险）= 市场的危险和不确定性
- 动×险 = 创业的本质：在危险中前行，动而遇险但生机勃勃
- 反常识根因：屯卦虽险，但"元亨利贞"——初生之难中蕴含最大的生机

**象的变化**：
- 当前爻位：初九"磐桓"——创业初期徘徊不前
- 变卦：初九阳变阴 → 第8卦**比**（亲比）→ 创业需要找到正确的合作伙伴
- 互卦：二三四爻为坤（顺），三四五爻为艮（止）→ 隐藏趋势是"顺而止"，即顺应形势、适时止步
- 视角升级：创业不是"克服困难"，而是"在困难中寻找生机"——屯卦的智慧是"利建侯"，即找到对的人比对的事更重要

---

### 示例2：用易经看"焦虑"

**定象**：
- 内卦（根基）：内心不安、流动不居 → ☵ 坎（水·陷）
- 外卦（环境）：外界压力散漫、无孔不入 → ☴ 巽（风·入）
- 组合：坎下巽上 = 第59卦**涣**（风行水上，涣散）

**一句话点破**：焦虑的本质是"风行水上"——心神如水般散乱，压力如风般无孔不入。

**属性拆解**：
- 内卦坎（陷）= 内心的深沉不安
- 外卦巽（入）= 外界压力的渗透扩散
- 陷×入 = 焦虑：内心有陷，外界有入，内外夹击
- 反常识根因：涣卦"亨"——涣散中反而有通达之机，"王假有庙"提示回归精神中心可化解涣散

**象的变化**：
- 当前爻位：六三"涣其躬"——焦虑已经影响到身体
- 变卦：六三阴变阳 → 第57卦**巽**（随风）→ 化解方向是"申命行事"，即建立秩序、重复正念
- 视角升级：焦虑不是要"消灭"，而是要"涣散"——把聚集的负能量散开，回归中心（王假有庙）
