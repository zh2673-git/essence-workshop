# 寓言与三阶合一的关系

> **方法论引用**：三阶合一框架定义详见 [../../general-logic/references/methodology.md](../../general-logic/references/methodology.md)。本文件只描述寓言视角与三阶合一的关系。

> **核心定位**：概念寓言化是三阶合一"阶段2·建立因果"的一种高具象载体。寓言横跨三阶（思考层）——开头呼应本质（阶段1）→中段建立反差（阶段2反常识根因）→结尾视角升级（阶段3）。

---

## 一、三阶合一在寓言中的体现

```
阶段1：点出本质      ← 本质定义先行 + 本质检验 + 一句话点破（寓言开头的隐性本质呈现）
阶段2：建立因果      ← 【寓言在此】用叙事把概念嫁接到熟悉情境 + 反常识根因（寓言中段的反差/反转）
阶段3：展开方法·视角升级 ← 映射表（回溯到本质）+ 迁移问题 + 视角升级（寓言寓意收尾）
```

寓言不是只在阶段2工作，而是横跨三阶（思考层对应关系，不是产物分节）：
- **寓言开头**：用故事场景隐性呈现阶段1的本质定义（不是直接说出本质定义，而是让本质在情境中显现）
- **寓言中段**：通过反差/反转触发阶段2的反常识根因，制造认知冲突
- **寓言结尾**：通过寓意收尾实现阶段3的视角升级，重新定义问题

---

## 二、寓言 vs 其他"建立因果"载体

| 载体 | 具象度 | 适用受众 | 认知负荷 | 适用场景 |
|------|--------|---------|---------|---------|
| 逻辑类比表格 | 中 | 同行/专业人士 | 低 | 知识探索、项目解析 |
| 生活案例 | 高 | 外行/初学者 | 中 | 内容输出、K12拆解 |
| **寓言故事** | **最高** | **外行/儿童/全受众** | **中** | **全场景通用** |
| 图示/动画 | 高 | 视觉学习者 | 低 | K12拆解、HTML交互 |

**判断规则（认知层）**：
- 受众是同行 → 用逻辑类比表格即可，不必寓言化
- 受众是外行/儿童 → 优先用寓言
- 受众混合 → 寓言引入 + 类比表格补充
