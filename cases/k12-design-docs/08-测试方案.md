# 知本（KnowledgeRoot）- 测试方案

## 测试目标

验证知本Skill的核心能力：**按学科自动选择范式族插件，按学段适配表述，生成七维拆解内容，输出交互式HTML页面**。

## 测试用例

### 用例1：数学 - 负数（形式科学族）

| 项目 | 值 |
|------|-----|
| 知识点 | 负数 |
| 学科 | 数学 |
| 范式族 | 形式科学族 |
| 学段 | 小学5年级 |
| 测试重点 | 逻辑困境设计（只有正数时3-5算不了）、生活锚点（温度/欠钱）、单位换算微框架不触发 |
| 预期特征 | 定义锚点有生活物品；矛盾是逻辑困境；验证题不考计算考理解 |

### 用例2：语文 - 以乐景衬哀情（人文学科族）

| 项目 | 值 |
|------|-----|
| 知识点 | 以乐景衬哀情 |
| 学科 | 语文 |
| 范式族 | 人文学科族 |
| 学段 | 初中8年级 |
| 测试重点 | 4W时间锚框架触发（安史之乱）、情感体验锚定、仿写迁移应用 |
| 预期特征 | 定义锚点有情感共鸣；溯源有4W时间锚；矛盾是表达困境；验证题考感受辨析 |

### 用例3：物理 - 惯性（形式科学族+学科微调）

| 项目 | 值 |
|------|-----|
| 知识点 | 惯性 |
| 学科 | 物理 |
| 范式族 | 形式科学族（+物理学科微调） |
| 学段 | 初中8年级 |
| 测试重点 | 物理学科微调是否生效（直觉与实验冲突、可观察物理现象、实验设计验证题） |
| 预期特征 | 矛盾是直觉与实验冲突；应用有可观察现象；验证有实验设计题 |

### 用例4：英语 - 一般现在时（人文学科族+学科微调）

| 项目 | 值 |
|------|-----|
| 知识点 | 一般现在时 |
| 学科 | 英语 |
| 范式族 | 人文学科族（+英语学科微调） |
| 学段 | 初中7年级 |
| 测试重点 | 英语学科微调是否生效（母语思维对比、交际场景、角色扮演） |
| 预期特征 | 矛盾是母语干扰；应用有角色扮演；验证有交际选择题；所有例子含中文翻译 |

## 测试检查清单

每个测试用例生成后，按以下清单检查：

### A. 框架完整性

- [ ] 7个维度全部生成
- [ ] 认知组（定义/溯源/矛盾）3个维度内容充实
- [ ] 应用组（应用/延伸/网络）3个维度内容充实
- [ ] 验证组（验证理解）有3题以上

### B. 范式族适配

- [ ] 形式科学族：定义有生活锚点，矛盾是逻辑困境
- [ ] 人文学科族：定义有情感锚点，溯源有4W时间锚
- [ ] 生命科学族：定义有可观察现象，矛盾是表象vs本质

### C. 学科微调

- [ ] 数学：溯源讲逻辑链条
- [ ] 物理：矛盾是直觉vs实验，验证有实验设计
- [ ] 化学：实验内容有安全提示
- [ ] 语文：溯源有作者处境+历史背景
- [ ] 历史：溯源有4W完整展开，验证有因果推理
- [ ] 英语：矛盾是母语干扰，例子含中文翻译

### D. 学段适配

- [ ] 低年级：代际比喻、生活场景锚点
- [ ] 高年级：年份+历史坐标、概念直接面对

### E. 质量底线

- [ ] 无AI套话（"探索""旅程""赋能"等）
- [ ] 无空洞表述（每个观点有具体支撑）
- [ ] 故事有细节（人物+困境+突破）
- [ ] 验证题不是纯计算/纯背诵

### F. HTML呈现

- [ ] 深色背景+高对比文字
- [ ] 强调色与学科对应
- [ ] 三组折叠面板可交互
- [ ] 知识地图可显示
- [ ] 验证题目可交互（选择+反馈）
