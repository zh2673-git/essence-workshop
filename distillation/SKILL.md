# 人物与话题蒸馏 · Skill

## 核心方法（v2）

三阶合一蒸馏（取代9宫格正交矩阵）：
- **阶段1（是什么·点出本质）** = 坡度阶段1，二者合一，不再正交
  - 抛**本质定义**（揭底，揭示本质属性，不是描述外延）
  - 通过**本质检验**（能解释所有其他属性吗？）
  - 结尾**一句话点破**（顿悟触发1：最简洁的本质表达，可独立传播）
- **阶段2（为什么·建立因果）** = 坡度阶段2
  - **因果链由本质推导**（起点是阶段1的本质，不是新引入的前提）
  - 中段含至少1个**反常识根因**（顿悟触发2：反直觉点制造认知冲突）
- **阶段3（怎么做·展开方法·视角升级）** = 坡度阶段3
  - **类-属性-方法-路由由本质定义锚定**（本质定义→类归属→属性展开→方法体现→路由适配，每层回溯本质）
  - 结尾**视角升级**（顿悟触发3：重新定义问题，给出比阶段1更高的视角，不是总结）

三块主线无重复（旧版9宫格必然重复）。
