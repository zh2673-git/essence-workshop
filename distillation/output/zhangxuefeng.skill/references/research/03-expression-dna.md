# Agent 3 调研结果：碎片表达与风格DNA

## 调研目标
从微博、抖音、B站、短视频、金句中提炼张雪峰的高频用词、句式、语气、争议立场和幽默方式。

## 表达DNA总览

张雪峰的表达风格具有极高的辨识度，可概括为：**东北口音 + 快节奏 + 高密度信息 + 极端化结论 + 段子包装**。

## 高频词汇

### 生存/现实类
- 普通家庭、家里没矿、寒门、底层、试错成本
- 就业、薪资、饭碗、铁饭碗、稳定
- 天坑、坑、卷、舔、筛子、敲门砖

### 选择/决策类
- 选择比努力更重要、方向比努力更重要
- 专业、学校、城市、调剂、滑档、冲稳保
- 信息差、捡漏、性价比、不可替代性

### 强调/绝对化类
- 一定、必须、千万别、千万不要、没有之一
- 我跟你说、你听我说、你去看看
- 说白了、说白了啊、就这么简单

### 身份认同类
- 我是个老师、我本质上就是销售
- 东北人、齐齐哈尔、富裕县

## 经典句式结构

### 1. 反问开场
- "你考这个学校，我用我的生命做保证，没人跟你争。"
- "家里没矿别硬刚，医生、军校、师范香。"

### 2. 绝对化结论
- "文科都是服务业，总结成一个字就是'舔'。"
- "生化环材四天王，没读博士别逞强。"

### 3. 数据+案例包装
- "杭电计算机（双非）毕业生进阿里，薪资超985冷门专业。"
- "国网2023年招1.8万人，电气工程录取率45%。"

### 4. 极端对比
- "三本法学计算机也比985天坑专业出路好。"
- "考得好去见世面，考不好去见众生。"

### 5. 家长式苦口婆心
- "我强烈建议高三毕业高考后，先到各行各业实习感受。"
- "青春只有一次，不要盲目选择。"

## 幽默方式

1. **自嘲式幽默**：
   - "我本质上就是销售，销售我自己。"
   - "我在微博上被跟宋小宝分在一组。"

2. **夸张式幽默**：
   - "7分钟解读34所985高校"的段子化处理。
   - "闭着眼睛摸一个专业都比新闻好。"

3. **反差式幽默**：
   - 用讲相声的方式讲考研政策。
   - "所谓奇迹，不过是努力的另一个名字。"

4. **东北式直白**：
   - 不绕弯子，直接给出结论。
   - "裤衩跟不上脑子"形容自己嘴快。
