# 荀子 · 长对话与即兴思考

## 稷下学宫的辩论

荀子在稷下学宫三为祭酒，与各家学者反复辩论。这些辩论展现了他即兴思考的特征：

### 与孟子的间接对话（跨越时空）

荀子《性恶》篇直接针对孟子性善论展开驳论：
- 孟子："人之性善" → 荀子："人之性恶，其善者伪也"
- 孟子以"四端之心"为性 → 荀子以"饥而欲饱、寒而欲暖"为性
- 关键分歧：何为"性"？孟子指人之所以异于禽兽者，荀子指人与禽兽之共通属性

### 与秦昭王的对话

荀子入秦考察，与秦昭王、范雎会谈：
- 荀子建议秦"重用儒者，行仁义"
- 秦昭王未采纳——荀子虽肯定秦国"四世有胜"，但认为"县之以王者之功名，则倜倜然其不及远矣"
- 被追问时的回答方式：先肯定对方成就，再指出根本不足

### 与临武君议兵

在赵孝成王面前，荀子与临武君辩论用兵之道：
- 临武君主张"兵之所贵者势利也，所行者变诈也"
- 荀子反驳："凡用兵攻战之本在乎壹民"，强调民心为战争根本
- 即兴类比："弓矢不调则羿不能以中微，六马不和则造父不能以致远"

### 与李斯的师徒对话

《议兵》中记载李斯与荀子的争论：
- 李斯："秦四世有胜，兵强海内，威行诸侯，非以仁义为之也，以便从事而已"
- 荀子："汝所谓便者，不便之便也；吾所谓仁义者，大便之便也"
- 面对弟子的质疑，荀子不回避，直接回应，但坚持立场

## 辩论风格特征

1. **先立后破**：先正面阐述自己的观点，再逐一驳斥对方
2. **善用类比**：用日常经验类比抽象道理（植物生长、产品制作、人工培育）
3. **层层递进**：从前提推导结论，逻辑链条严密
4. **气势磅礴**：排比对偶增强说服力
5. **不回避质疑**：直接面对反对意见，正面回应
