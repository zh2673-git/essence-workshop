| 1898 | 赴南京求学，入水师学堂 | 接触西方新学 | 走异路、求新声 |

| 1902 | 赴日留学 | 开始探索国民性问题 | 国民性批判的起点 |
| 1903 | 剪辫明志，题诗"我以我血荐轩辕" | 确立救国志向 | 立人思想的萌芽 |
| 1904 | 入仙台医专学医 | 希望以医学救国 | 科学救国→精神救国的过渡 |

| 1909 | 回国任教 | 现实的失望 | 沉默期的积累 |
| 1912 | 入北京教育部 | 目睹辛亥革命不彻底 | 对革命的深层怀疑 |
| 1918 | 发表《狂人日记》 | 从沉默转向呐喊 | 铁屋子隐喻 |
| 1921 | 连载《阿Q正传》 | 精神胜利法的经典刻画 | 国民性解剖的巅峰 |
| 1923 | 《呐喊》出版 | 五四启蒙精神的集中体现 | 呐喊作为方法 |
| 1925 | 支持女师大学生运动 | 知识分子的行动选择 | 不沉默的立场 |
| 1926 | 《彷徨》出版，三一八惨案后离京 | 启蒙落潮后的迷茫 | 彷徨中的坚守 |

| 1928 | 与创造社论战 | 大量阅读马克思主义著作 | 从进化论到阶级论 |
| 1930 | 加入左联 | 成为左翼文学领袖 | 横站——联合中的独立 |
| 1931-1935 | 柔石等左联五作家被杀 | "忍看朋辈成新鬼" | 一个也不宽恕 |
| 1935 | 《故事新编》出版 | 古今杂糅的批判 | 历史作为当下的镜像 |
| 1936 | 10月19日病逝上海 | "一个也不宽恕"的遗言 | 至死不妥协 |

## 五大阶段

1. **早年成长**（1881-1898）：家道中落→看透世态→走异路
2. **留学日本**（1902-1909）：弃医从文→立人思想形成→摩罗诗力说
3. **沉默与呐喊**（1909-1926）：辛亥失望→铁屋子→狂人日记→五四高峰
4. **彷徨与转变**（1926-1927）：启蒙落潮→四一二→进化论→阶级论
5. **上海十年**（1927-1936）：杂文战斗→左联领袖→一个也不宽恕
