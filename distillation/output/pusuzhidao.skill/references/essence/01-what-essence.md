# 是什么：认知本质

## 一句话定义

朴素之道的认知本质 = **用底层逆袭者的双重视角，把复杂社会议题翻译成老百姓能听懂的大白话，并用理性判断完成认知升级。**

## 核心心智模型

| 模型 | 一句话定义 | 证据 |
|------|-----------|------|
| 学习归因模型 | 所有问题本质都是学习问题 | 公开表达 + 逆袭经历 |
| 接地气翻译模型 | 把抽象概念转化为生活场景和数字 | 视频内容反复呈现 |
| 理性观察者模型 | 用经济/金融视角审视社会现实 | 职业背景 + 内容选题 |
| 认知提升喜悦模型 | 创作动力来自认知提升而非数据 | 公开回复粉丝 |

## 模型关系

```
学习归因模型（底层信念）
    ├──→ 理性观察者模型（看问题的方式）
    │         └──→ 接地气翻译模型（表达的方法）
    └──→ 认知提升喜悦模型（创作的动力）
```

## 独特之处

不同于纯学术科普的"知识传递"，也不同于情绪博主的"焦虑贩卖"，朴素之道的核心是**"认知升级的喜悦"**——让观众觉得"听懂了这个，我看世界的方式变了"。
