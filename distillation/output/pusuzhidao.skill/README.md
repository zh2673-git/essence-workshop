# 朴素之道 · 认知操作系统

> 把复杂社会议题翻译成老百姓能听懂的大白话，用具体生活场景和数字建立共情，最终给出认知升级的理性判断。

## 核心心智模型

| 模型 | 一句话 | 适用场景 |
|------|--------|---------|
| 学习归因模型 | 所有问题本质都是学习问题 | 困境归因、个人成长 |
| 接地气翻译模型 | 把抽象概念转化为生活场景和数字 | 科普、口播稿、短视频脚本 |
| 理性观察者模型 | 用经济/金融视角看社会现实 | 社会热点、经济议题分析 |
| 认知提升喜悦模型 | 创作动力来自认知提升而非数据 | 内容创作、知识分享 |

## 文件结构

- `SKILL.md` — 完整认知操作系统与表达 DNA
- `references/research/` — 调研原始数据
  - `03-expression-dna.md` — 表达 DNA 与风格特征
  - `06-timeline.md` — 人物时间线
  - `07-root-causes.md` — 根因分析
- `references/essence/` — 三阶蒸馏结果
  - `01-what-essence.md` — 是什么：认知本质
  - `02-why-root-causes.md` — 为什么：根因追溯
  - `03-how-system.md` — 怎么做：系统建模

## 使用方式

在对话中提及「用朴素老师的风格」「像朴素之道那样讲」「朴素之道视角」即可激活此Skill。

主要用于：
- 口播稿 / 短视频脚本创作
- 大众科普内容改写
- 抽象概念的生活化解释
- 社会现实议题的理性评论
