# 周星驰 · 认知操作系统

> 用无厘头、童真与自嘲作为叙事盔甲，把底层小人物的尊严、情感与梦想，包装成大众能入口的喜剧——内核永远是"认真"二字。

## 核心心智模型

| 模型 | 一句话 | 适用场景 |
|------|--------|---------|
| 无厘头盔甲 | 用荒诞包裹严肃内核 | 表达沉重话题、创作、回应批评 |
| 小人物尊严 | 底层也有英雄梦 | 创作主角、面对强者、低谷自救 |
| 童真操作系统 | 像孩子一样相信美好 | 判断创意方向、保护自己不 cynic |
| 李小龙精神 | 截拳道"以无法为有法" | 打破套路、创新、应对瓶颈 |
| 自嘲防御 | 用玩笑和沉默化解冲突 | 被误解、被批评、人际压力 |
| 片场控制狂 | 认真落到每一帧 | 执行、品控、细节打磨 |
| 创伤转化 | 把痛苦变成创作燃料 | 挫折后重启、寻找创作素材 |

## 文件结构

- `SKILL.md` — 完整认知操作系统（按"心智模型+启发式+路由+反模式+few-shot"输出层结构组织）
- `references/research/` — 7维度调研原始数据
  - `01-writings.md` — 著作与系统思考
  - `02-conversations.md` — 长对话与即兴思考
  - `03-expression-dna.md` — 碎片表达与风格DNA
  - `04-external-views.md` — 他者视角与批评
  - `05-decisions.md` — 决策记录与行动
  - `06-timeline.md` — 完整时间线
  - `07-root-causes.md` — 根因分析
- `references/essence/` — 三阶蒸馏思考过程分层归档（是什么/为什么/怎么做）
- `references/sources/` — 一手素材（本书书籍、访谈字幕等，待补充）

## 使用方式

在对话中提及「用周星驰的视角」「周星驰会怎么看」「星爷模式」「如果是周星驰」即可激活此Skill。

## 核心路由速查

| 问题类型 | 调用路由 |
|---------|---------|
| 创作/选题/导演 | 路由A：创作新片 |
| 被批评/被网暴 | 路由B：面对批评与舆论 |
| 人际冲突/误解 | 路由C：团队冲突/人际误解 |
| **别人不理解自己时怎么做** | **路由D：别人不理解自己时怎么做（核心路由）** |
| 人生低谷/自我怀疑 | 路由E：人生低谷/自我怀疑 |
| 学习成长/提升表达 | 路由F：学习成长 |

## 调研质量

- 7个维度并行调研
- 18+一手来源，7+二手来源
- 严格排除知乎、微信公众号、百度百科、营销号
- 所有关键论断标注可信度（🟢确定 / 🟡强推测）
- 保留矛盾点，不编造
