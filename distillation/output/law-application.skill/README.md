# 法条适用 · 认知框架 (law-application-essence)

> 在具体事实与抽象规范之间建立理性连接的推理技术。覆盖民事、刑事、行政及宪法基本权利场景。

## 概述

基于7维度深度调研，通过三阶蒸馏提炼通用法条适用认知框架。包含8个核心认知框架、14条决策启发式、6条场景路由和完整的表达DNA。

## 核心认知框架

| # | 框架名称 | 一句话定义 |
|---|---------|-----------|
| 1 | 事实裁剪框架 | 从生活事实中提炼具有规范意义的法律事实 |
| 2 | 法律关系定性框架 | 判断案件属于民事/刑事/行政/宪法基本权利哪个领域；识别行政行为三要件 |
| 3 | 规范检索框架 | 先检索立法法正式法源；行政行为满足三要件时，再检索行政规范性文件 |
| 4 | 涵摄框架 | 将事实逐一对照法条构成要件 |
| 5 | 冲突裁决框架 | 用位阶与优先规则解决规范冲突；先识别规范性质 |
| 6 | 法律解释框架 | 从文义出发，经体系、历史到目的解释 |
| 7 | 抗辩与效力检视框架 | 检查抗辩事由、违法性阻却、行政行为合法性要件、基本权利限制事由等 |
| 8 | 兜底适用框架 | 穷尽具体规则后再启用兜底条款 |

## 使用方式

当用户提到以下触发词或场景时激活此Skill：

- 「法条适用」「怎么找法条」
- 「这个行为违法吗」「这算侵权吗」「构成犯罪吗」
- 「这个行政行为合法吗」「处罚有没有依据」
- 「该用哪个法条」「特别法优于一般法」
- 「兜底条款怎么用」「法律没规定怎么办」
- 「XX号文是什么」「红头文件能适用吗」「行政规范性文件」
- 用户描述具体事件并请求法律分析

激活后以训练有素的法律人身份回应，使用8个核心认知框架、14条决策启发式和场景路由来分析问题。

## 场景路由速查

| 场景 | 调用的框架组合 |
|------|---------------|
| 用户描述具体事件 | 事实裁剪 + 法律关系定性 + 规范检索 + 涵摄 + 抗辩与效力检视 |
| 问某行为是否违法/侵权/犯罪/行政行为合法 | 法律关系定性 + 涵摄 + 抗辩与效力检视 |
| 问多个法条/文件用哪个（含"XX号文"性质判断） | 冲突裁决 |
| 问没有明确法条怎么办 | 兜底适用 + 法律解释 + 漏洞填补 |
| 问法律概念怎么用 | 法律解释 |
| 默认 | 事实裁剪 + 法律关系定性 + 规范检索 + 涵摄 |

## 方法论

三阶方法论 × 坡度理解 × 类-属性-方法-路由
