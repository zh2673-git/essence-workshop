# 03 多元视角与理论

## 民法视角

**核心关切**：如何在私法关系中确定权利义务。

- **请求权基础方法**：王泽鉴倡导的检视顺序为合同 → 类合同 → 无因管理 → 物权 → 不当得利 → 侵权。
- **法律行为解释**：重视当事人真意，兼顾文义与目的。
- **利益衡量**：在法条不明确时，衡量当事人利益与社会公共利益。

## 刑法视角

**核心关切**：罪刑法定与保障人权。

- **罪刑法定原则**：法无明文规定不为罪，法无明文规定不处罚。
- **禁止类推**：禁止不利于被告人的类推适用。
- **法条竞合**：特别关系、补充关系、吸收关系、择一关系。
- **想象竞合 vs 法条竞合**：从一重罪处断 vs 特别法优于普通法。

## 行政法视角

**核心关切**：依法行政与权力边界。

- **法律保留**：某些事项必须由法律规定，行政机关不得越权。
- **上位法优于下位法**：行政法规、规章不得与法律相抵触。
- **特别授权**：经济特区、自治地方可依法变通适用法律。

## 宪法视角

**核心关切**：基本权利保障与合宪性控制。

- **合宪性解释**：在多种解释可能中，选择最符合宪法的解释。
- **基本权利限制**：限制基本权利的法律应采最窄解释。
- **规范层级**：宪法具有最高法律效力。

## 比较法视角

| 法系 | 法条适用特点 |
|------|-------------|
| **大陆法系** | 成文法为中心，重视法律解释和涵摄 |
| **英美法系** | 判例法为中心，重视类比推理和 distinguishing |
| **北欧法系** | 介于两者之间，重视法律的现实效果 |

## 共识与分歧

**共识**：
- 文义解释是起点
- 法条冲突需要优先规则
- 法律漏洞需要谨慎填补

**分歧**：
- 目的解释与文义解释的边界
- 一般条款具体化的程度
- 法官在法律漏洞填补中的自由度
