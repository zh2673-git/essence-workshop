# 01 定义与核心概念

## 法条适用

**定义**：在面对具体案件事实时，法律人用来确定"哪一条规范可用、如何用、用哪条优先"的推理准则与操作方法。

**核心特征**：
- 领域性：需先判断案件属于民事、刑事、行政或宪法基本权利领域
- 规范性：受法律体系约束，不是任意选择
- 层级性：从宪法到规章、规范性文件，效力有高低
- 程序性：遵循"事实裁剪—法律关系定性—规范检索（先正式法源，行政行为再补规范性文件）—涵摄—抗辩/效力检视"的递进流程
- 冲突性：多个法条可能同时指向同一事实
- 补充性：规则缺失时借助一般条款、类推、原则填补漏洞

## 关键概念

| 概念 | 定义 | 来源 |
|------|------|------|
| **法律渊源** | 宪法、法律、行政法规、地方性法规、自治条例、单行条例、规章、司法解释、国际条约等；**行政规范性文件**虽非正式渊源，但实务中常作为行政行为依据 | 《立法法》；《行政诉讼法》第53条 |
| **效力位阶** | 法律规范之间的效力层级关系 | 《立法法》第98—101条 |
| **行政规范性文件** | 行政机关制定的、涉及公民/法人权利义务、具有普遍约束力并反复适用的文件（俗称"红头文件"），非《立法法》正式渊源 | 《行政诉讼法》第53条；国务院《规章制定程序条例》第2条（排除规章后的剩余） |
| **法律关系定性** | 判断案件属于民事、刑事、行政或宪法基本权利领域的过程 | 部门法划分理论 |
| **法律事实** | 从生活事实中提炼出的、具有规范意义的情节 | 法学方法论 |
| **请求权基础** | 民事领域中支持一方当事人向他方主张权利的特定法律规范 | 王泽鉴请求权基础理论 |
| **规范检索** | 先检索立法法正式法源；行政行为满足三要件时，再检索行政规范性文件 | 通用法条适用方法论 |
| **构成要件（T）** | 法条规定的条件 | 法学三段论 |
| **法律效果（R）** | 满足构成要件后产生的后果 | 法学三段论 |
| **涵摄（Subsumtion）** | 将事实归入构成要件的过程 | 德国法学方法论 |
| **法条竞合** | 同一行为同时符合多个法条，需择一适用 | 刑法/民法理论 |
| **法律解释** | 文义、体系、历史、目的、合宪性、比较法等解释方法 | 法学解释学 |
| **法律漏洞** | 法律对应调整事项未设规范或不应涵盖某类情形 | 法学方法论 |

## 概念边界

法条适用不同于：
- **法律制定**：不是创造新规则，而是在既有规则中寻找和解释
- **事实认定**：不是判断证据真伪，而是基于事实选择规范
- **价值判断**：不是任意道德评价，而是受规范约束的理性推理
