  - 需要确认请求权/抗辩主张的法律基础；

# 规范检索 · 原子技能

## 法律声明

> 本技能输出的是候选法律规范清单及检索路径，**不构成法律意见**。所有法条引用的最终效力与准确性，必须由执业法律专业人员通过官方法律数据库复核确认。

---

## 一、核心概念

### 1.1 正式法源层级

| 位阶 | 制定机关 | 示例 |
|------|----------|------|
| 宪法 | 全国人大 | 《中华人民共和国宪法》 |
| 法律 | 全国人大及其常委会 | 《民法典》《刑法》《行政处罚法》 |
| 行政法规 | 国务院 | 《劳动合同法实施条例》 |
| 地方性法规 | 省级/设区的市人大及其常委会 | 《广东省公路条例》 |
| 部门规章 | 国务院各部委 | 《工资支付暂行规定》 |
| 地方政府规章 | 省级/设区的市政府 | — |
| 司法解释 | 最高法/最高检 | 法释〔2020〕17号 |

### 1.2 两层检索结构

**第一层：立法法正式法源（所有案件必检）**

| 领域 | 检索顺位 |
|------|----------|
| 民事 | 合同/类合同 → 无因管理 → 物权 → 不当得利 → 侵权 |
| 刑事 | 刑法分则罪名 → 刑法总则 → 单行刑法 → 司法解释 |
| 行政 | 法律 → 行政法规 → 地方性法规 → 部门规章/地方政府规章 → 司法解释 |
| 宪法基本权利 | 宪法 → 法律 → 比例原则与法律保留 |

**第二层：行政规范性文件（按条件触发）**

| 情形 | 是否自动检索 |
|------|-------------|
| 行政机关行使职权的案件 | ✅ 自动 |
| 行政机关作为民事主体 | ❌ 不自动 |
| 民事/刑事/宪法案件 | ❌ 不自动（可按需提示） |
| 领域不明或行民交叉 | ⚠️ 提示用户 |

### 1.3 民事请求权九阶检视

```
第一阶：合同上的请求权
第二阶：单方法律行为的请求权
第三阶：类契约请求权
第四阶：身份法上的请求权
第五阶：无因管理请求权
第六阶：物权请求权
第七阶：占有保护请求权
第八阶：不当得利请求权
第九阶：侵权请求权
```

## 二、检索策略：分层降级

### 第一层：已配置的 MCP 工具（优先）

| MCP 服务 | 适用场景 |
|----------|----------|
| 北大法宝 MCP | 法条语义检索、法条溯源、效力校验、案例检索 |
| 秘塔搜索 MCP | 全网法律信息、学术观点、实务文章、地方性规定 |
| 其他法律法规 MCP | 按实际配置调用 |

### 第二层：模型/Agent 自身联网搜索能力

MCP 不可用时，启用联网搜索：

- **法条核验**：国家法律法规数据库（flk.npc.gov.cn）
- **类案检索**：中国裁判文书网、人民法院案例库
- **学说/实务**：学术数据库、政府官网、实务文章

**要求**：所有通过网络搜索获得的法条/案例必须标注来源 URL 与检索时间。

### 第三层：模型内部知识（最后手段）

- 仅用于给出检索方向或初步框架；
- 引用的法条必须标注 `[待校验]`；
- 禁止将未校验法条作为确定性结论的唯一依据。

---

## 三、工作流程

### 阶段一：检索准备

1. 确认法律关系领域（来自 `legal-relationship-typing`）；
2. 确认争议焦点与核心事实；
3. 确认地域信息（涉及地方性规定时）。

### 阶段二：正式法源检索

按领域检索顺位，从高位阶到低位阶检索。

### 阶段三：行政规范性文件检索（条件触发）

若案件涉及行政行为且满足三要件，自动进入第二层检索。

### 阶段四：请求权基础检视（民事案件）

按九阶顺序检视请求权基础，避免遗漏。

### 阶段五：效力与来源校验

- 交叉验证法条现行效力；
- 标注来源、检索时间、效力状态。

## 六、验证规则

| 检查项 | 通过标准 |
|--------|----------|
| 检索顺位正确 | 已按领域顺位从高位阶到低位阶检索 |
| 来源标注完整 | 每条规范均标注来源与检索时间 |
| 效力状态明确 | 已标注 VALID / AMENDED / REPEALED / UNCERTAIN 等 |
| 行政规范性文件条件正确 | 仅在行政行为三要件满足时自动检索 |
| 民事请求权无遗漏 | 已按九阶顺序检视 |

---

## 七、失败模式与降级

| 失败场景 | 处理动作 |
|----------|----------|
| MCP 不可用 | 启用模型/Agent 联网搜索，标注来源 URL |
| 联网搜索受限 | 使用模型内部知识给出检索方向，所有法条标注 `[待校验]` |
| 无法确认法条效力 | 标注 `[效力存疑]`，降低置信度 |
| 地方性规定需要但无地域信息 | 提示用户补充地域信息 |
