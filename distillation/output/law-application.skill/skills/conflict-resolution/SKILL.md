  当法律适用过程中出现法条竞合、证据矛盾或争点优先级冲突时触发本技能。
  典型触发场景：
  - 多个候选法条同时适用或相互冲突；
  - 不同证据之间内容矛盾；
  - 多个争议焦点需要确定审理或分析顺序；
  - 上位法与下位法、特别法与一般法、新法与旧法存在冲突。

  边界：不能替代司法机关的终局裁决；涉及宪法性冲突或重大法律原则冲突时需标注专业复核需求。

# 冲突裁决 · 原子技能

## 法律声明

> 本技能仅提供法律冲突的分析框架与优先适用建议，**不构成法律意见**。复杂冲突（尤其是法源位阶冲突、跨法域冲突）应由执业法律专业人员结合具体情境判断。

---

## 一、核心概念

### 1.1 冲突类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 法条竞合 | 同一事实同时符合多个规范 | 违约与侵权竞合 |
| 法源冲突 | 不同位阶或同等位阶规范不一致 | 部门规章与地方性法规冲突 |
| 证据矛盾 | 不同证据对同一事实的证明方向相反 | 证人A与证人B陈述相反 |
| 争点冲突 | 多个争议焦点需要排序 | 程序性争点与实体性争点并存 |

### 1.2 法条竞合解决规则

| 规则 | 内容 | 适用条件 |
|------|------|----------|
| 上位法优于下位法 | 高位阶规范优先 | 不同位阶规范冲突 |
| 特别法优于一般法 | 特别规定优先 | 同一位阶规范，存在特别与一般关系 |
| 新法优于旧法 | 新规范优先 | 同一位阶，同一事项 |
| 法不溯及既往 | 行为时法优先 | 时间效力冲突 |

### 1.3 争点优先级规则

| 规则 | 含义 |
|------|------|
| 程序先于实体 | 管辖、时效、主体资格等程序问题先解决 |
| 前提先于派生 | 基础法律关系的争点先于后果争点 |
| 主要先于次要 | 核心争点先于附带争点 |
| 法律问题先于事实问题 | 法律适用问题先于证据证明问题 |

## 二、工作流程

### 步骤 1：冲突识别

列出所有冲突点，分类为法条竞合、法源冲突、证据矛盾、争点冲突。

### 步骤 2：冲突评估

评估每个冲突的严重性：
- 是否影响最终结论？
- 是否可以通过解释调和？
- 是否需要外部权威介入？

### 步骤 3：分别解决

按冲突类型适用相应规则。

### 步骤 4：交叉冲突解决

若多个冲突相互影响，综合判断。

### 步骤 5：一致性检验

确保解决方案不自相矛盾，结论可复现。

## 四、示例

**输入**：同一违约行为同时符合《民法典》第577条违约责任与第1165条侵权责任。

**输出**：

```markdown
### 一、冲突清单

| 编号 | 冲突类型 | 冲突描述 | 严重程度 |
|------|----------|----------|----------|
| 1 | 法条竞合 | 违约责任与侵权责任竞合 | 高 |

### 二、法条竞合解决

| 冲突 | 适用规则 | 解决方案 | 理由 |
|------|----------|----------|------|
| 违约责任 vs 侵权责任 | 请求权基础选择 | 由债权人选择行使违约请求权或侵权请求权 | 《民法典》第186条 |

### 五、综合结论
- **最终适用的规范/方案**：建议债权人选择最有利于其利益实现的请求权基础
- **排除的规范/方案**：未被选择的请求权基础
- **置信度**：高
```

## 五、验证规则

| 检查项 | 通过标准 |
|--------|----------|
| 冲突分类正确 | 已正确识别冲突类型 |
| 规则适用正确 | 已按正确顺位适用冲突解决规则 |
| 排除理由充分 | 已说明为何排除其他方案 |
| 一致性 | 解决方案不自相矛盾 |
| 争点排序合理 | 排序符合程序/前提/主要优先规则 |

---

## 六、失败模式与降级

| 失败场景 | 处理动作 |
|----------|----------|
| 冲突无法通过既有规则解决 | 标注"需司法解释或裁判规则"，降低置信度 |
| 证据矛盾无法调和 | 标注"事实真伪不明"，提示举证责任 |
| 涉及宪法性冲突 | 建议提交有权机关处理 |
| 多个冲突相互嵌套 | 分步解决，每步标注依赖关系 |
