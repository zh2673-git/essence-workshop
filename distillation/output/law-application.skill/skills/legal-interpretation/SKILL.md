# 法律解释 · 原子技能

## 法律声明

> 本技能提供的解释结论属于学理论证，**不构成法律意见**。法律解释具有开放性，最终解释结论应由有权机关或执业法律专业人员确定。

---

## 一、核心概念

### 1.1 解释方法顺位

| 顺位 | 方法 | 说明 |
|------|------|------|
| 1 | 文义解释 | 从法律条文的通常含义出发 |
| 2 | 体系解释 | 在上下条文中、在法律整体中理解 |
| 3 | 目的解释 | 探求立法目的与规范目的 |
| 4 | 历史解释 | 参考立法资料、修法过程 |

### 1.2 解释结果类型

| 类型 | 说明 |
|------|------|
| 扩张解释 | 将文义范围内的边缘情形纳入 |
| 限缩解释 | 将文义范围内但不合规范目的的情形排除 |
| 当然解释 | 举重以明轻、举轻以明重 |
| 类比解释 | 法律漏洞填补（民法领域谨慎使用） |

### 1.3 权威支撑顺位

1. 有权解释（立法解释、司法解释）
2. 立法说明与修法资料
3. 最高人民法院指导性案例、公报案例
4. 学者通说
5. 地方法院裁判规则

- 仅用于框架性提示

---

## 三、工作流程

### 步骤 1：明确解释对象

- 需要解释的条文；
- 争议焦点（概念、范围、适用条件）。

### 步骤 2：文义解释

- 通常含义；
- 可能的文义范围。

### 步骤 3：体系解释

- 上下文关系；
- 与其他条文的协调。

### 步骤 4：目的解释

- 规范目的；
- 立法目的。

### 步骤 5：历史解释

- 立法沿革；
- 修法说明。

### 步骤 6：权威支撑检索

按权威支撑顺位检索并标注来源。

### 步骤 7：形成解释结论

- 采用何种解释方法；
- 结论为何；
- 置信度如何。

## 五、示例

**输入**：需要解释《民法典》第1165条"过错"是否包括过失。

**输出**：

```markdown
### 一、解释对象
- **条文**：《民法典》第1165条
- **争议焦点**："行为人因过错侵害他人民事权益"中的"过错"是否包括过失

### 二、文义解释
- **通常含义**："过错"包括故意和过失
- **文义范围**：核心含义明确

### 三、体系解释
- 与《民法典》第1165条第2款（过错推定）并列，过错为一般侵权构成要件

### 七、解释结论
- **采用方法**：文义解释 + 体系解释
- **结论**："过错"包括故意和过失
- **置信度**：高
```

## 六、验证规则

| 检查项 | 通过标准 |
|--------|----------|
| 方法顺位 | 已按文义→体系→目的→历史顺序展开 |
| 权威支撑 | 已检索有权解释、立法说明、案例、学者通说 |
| 来源标注 | 所有支撑来源均已标注 |
| 结论限定 | 已说明解释结论的开放性与局限 |

---

## 七、失败模式与降级

| 失败场景 | 处理动作 |
|----------|----------|
| 文义本身清晰但当事人仍争议 | 指出无解释必要，建议回到涵摄 |
| 权威支撑冲突 | 并列展示不同观点，标注主流意见 |
| 无法检索权威来源 | 结论标注 `[待校验]` |
| 涉及重大法律原则争议 | 建议咨询执业律师或提交有权机关 |
