# 苏格拉底 · 对话与即兴思考

## 对话特征

### 苏格拉底式对话的标准模式（Elenchus）

1. 对话者提出一个论题 P（苏格拉底认为错误的）
2. 苏格拉底获得对辅助前提 Q、R 的同意
3. 苏格拉底论证 Q、R 推导出非 P
4. 苏格拉底宣称 P 被证伪

**核心要求**：
- "说你所信"原则：对话者必须真诚地表达自己的信念
- 目的既是哲学的（发现道德真理），也是治疗的（检验对话者是否过道德生活）

### 被追问时的回答方式

- **从不直接给出答案**：始终以问题回应问题
- **承认无知**：反复声明"我不知道"，但通过追问揭示对方也不知道
- **即兴类比**：频繁使用日常类比（工匠、医生、舵手）来说明观点
- **改变立场的瞬间**：在Apology中，当德尔斐神谕说他最智慧时，他的反应不是接受而是去验证——这展现了他对权威的质疑态度

### 对话中的关键转折

1. **游叙弗伦困境**：虔敬是因为神喜爱才虔敬，还是因为虔敬神才喜爱？（Euthyphro 10a）——这个困境至今仍是哲学核心问题
2. **普罗泰戈拉中的立场变化**：从"德性可教"到"德性即知识"的论证推进
3. **高尔吉亚中面对卡利克勒**：面对"强者正义"的挑战，苏格拉底坚持"宁受不义不行不义"

## 对话风格分析

- **反讽（Eirōneia）**：声称自己无知但始终保持对话主导权
- **助产术比喻**：Theaetetus 148e-151d，将自己的方法比作母亲的助产术
- **牛虻比喻**：Apology 30e，将自己比作雅典城邦的牛虻
- **从不收费**：与智者派的关键区别
- **公共空间对话**：在集市、体育馆等公共场所进行对话，而非私密学院
