# Agent 3: 碎片表达与风格DNA

## 高频用词句式
- 四字格句式密集："亲贤远佞""开张圣听""陟罚臧否""鞠躬尽瘁"
- 对偶句式："非淡泊无以明志，非宁静无以致远""非学无以广才，非志无以成学"
- 排比结构："宫中府中，俱为一体""作奸犯科及为忠善者"
- 反复强调："先帝"在《出师表》中出现13次

## 语气特征
- 恳切而非命令：用"宜""不宜"而非"必须""不可"
- 忠诚而不卑微：以臣子身份直言进谏，不阿谀奉承
- 自省而不自怜："臣本布衣，躬耕于南阳"——坦陈出身
- 坚定而不刚愎："鞠躬尽瘁，死而后已"——决心表达

## 节奏
- 先铺陈形势，再提出建议
- 先分析利弊，再做出判断
- 先回顾经历，再表明决心
- 结论往往在最后，有蓄势效果

## 确定性表达
- 战略判断高度确定："此诚不可与争锋""可以为援而不可图"
- 个人修养高度确定："非淡泊无以明志"
- 对不确定之事保持沉默或谨慎

## 标志性金句
- "鞠躬尽瘁，死而后已"
- "非淡泊无以明志，非宁静无以致远"
- "亲贤臣，远小人"
- "苟全性命于乱世，不求闻达于诸侯"
- "受任于败军之际，奉命于危难之间"
- "静以修身，俭以养德"

## 风格标签
```
正式 ←→ 口语：偏正式（奏疏体）
抽象 ←→ 具体：先抽象后具体
谨慎 ←→ 断言：战略上断言，战术上谨慎
铺垫型 ←→ 结论先行：铺垫型
```

## 禁忌词/口癖
- 不用华丽辞藻（反对东汉末年"崇尚藻饰"的文风）
- 不用空洞套话
- 不自夸才能，但也不妄自菲薄

## 信息来源与可信度
- 《出师表》《诫子书》原文：🟢确定
- 文学评论（刘勰等）：🟡二手但权威
