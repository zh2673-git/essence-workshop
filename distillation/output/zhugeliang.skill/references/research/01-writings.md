# Agent 1: 著作与系统思考

## 核心著作

### 《隆中对》
- 建安十二年(207年)，诸葛亮27岁时与刘备的对话
- 提出三分天下的战略规划：跨有荆益→联吴抗曹→内修政理→待机北伐
- 核心逻辑：环境扫描（曹操不可争锋、孙权可为援不可图）→资源分析（荆益为基）→路径设计（内外并举）→时机判断（天下有变）
- 被誉为中国历史上最伟大的战略构想之一

### 《出师表》
- 建兴五年(227年)北伐前上呈刘禅
- 核心内容：形势判断（危急存亡）→三条建议（开张圣听、执法公平、亲贤远佞）→追叙经历→表明决心
- 文风质朴恳切，六百余字中十三次提及"先帝"
- 刘勰评："志尽文畅，表之英也"

### 《诫子书》
- 写给儿子诸葛瞻的家书，仅八十六字
- 核心观点："夫君子之行，静以修身，俭以养德。非淡泊无以明志，非宁静无以致远。非学无以广才，非志无以成学。"
- 修身哲学的浓缩表达：静→俭→学→志→才的递进关系

### 《便宜十六策》
- 系统阐述治国、治军、用人、外交思想的政论著作
- 核心思想："治国犹于治身，治身之道，务在养神；治国之道，务在举贤"
- 涵盖君臣关系、赏罚制度、军事训练、经济管理、边疆治理
- 辩证思维："欲思其利，必虑其害；欲思其成，必虑其败"

### 《将苑》
- 治军用将之道
- "将不可骄，骄则失礼，失礼则人离"
- "贵而不骄，胜而不悖，贤而能下"

## 反复出现的核心论点（≥3次=真信念）

1. **法治为本**：《便宜十六策》论赏罚、《蜀科》制定、斩马谡实践——至少4次出现
2. **举贤任能**：《便宜十六策》论举贤、《出师表》荐人才、实际用人实践——至少5次出现
3. **审时度势**：《隆中对》分析天下大势、联吴抗曹决策、北伐时机选择——至少4次出现
4. **静以修身**：《诫子书》核心论点、自身廉洁实践、"静"的哲学——至少3次出现
5. **德刑并施**：《便宜十六策》论恩威并济、《答法正书》论法治与德政、治蜀实践——至少4次出现

- "攻心为上"——南征战略的核心概括
- "亲贤臣，远小人"——政治判断的简洁准则
