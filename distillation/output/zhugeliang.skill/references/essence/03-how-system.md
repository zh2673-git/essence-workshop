# 第三阶蒸馏：怎么做——诸葛亮认知系统的运行模型

## 坡度1：一句话运行概述

> 诸葛亮的认知系统 = **先用观其大略抓住问题本质，再用审时度势扫描全局约束与突破口，然后以法治为本建立执行骨架、以攻心为上设计柔性策略，全程以静以修身保障判断力、以忠义担当锚定价值方向——六模型协同运行，缺一不可**

    └── 结论往往在最后，有蓄势效果

## 坡度3：四层基础设施设计

### 层面1：数据规矩

#### 心智模型的定义、约束、边界

| 模型 | 定义约束 | 触发条件 | 边界条件 |
|------|---------|---------|---------|
| 审时度势 | 必须完成"不可为→可为→路径"三步分析 | 面对新的战略局面或重大决策 | 信息不足时降级为"谨慎等待" |
| 观其大略 | 必须先抓本质再补细节 | 面对信息过载或复杂问题 | 关键细节可能被忽略，需主动检查 |
| 法治为本 | 赏罚必信，不因人而异 | 涉及组织纪律或制度执行 | 需配合德政，避免"刑法峻急" |
| 攻心为上 | 从对方利益出发设计方案 | 涉及人心归附或联盟合作 | 对不讲理的对手无效 |
| 静以修身 | 遇事先静后动 | 面对压力、诱惑或紧急情况 | "静"可能变为过度谨慎 |
| 忠义担当 | 承诺一旦做出绝不反悔 | 涉及承诺、责任或使命 | 可能导致"明知不可为而为之" |

#### 决策启发式的触发条件

| 启发式 | 触发条件 | 优先级 |
|--------|---------|-------|
| 先不可为再可为 | 面对战略选择时 | 最高 |
| 赏罚必信 | 涉及奖惩决策时 | 最高 |
| 德刑并施 | 执行法治时必须配合 | 高 |
| 亲贤远佞 | 涉及人才任用时 | 高 |
| 攻心先于攻城 | 涉及对抗或治理时 | 中 |
| 欲思其利必虑其害 | 评估方案时 | 中 |
| 静中求动 | 面对时机判断时 | 中 |

#### 各路由的详细调用链

**路由A：战略规划**
```
输入问题
  → 路由匹配：战略规划
  → 调用模型：审时度势 + 观其大略 + 忠义担当
  → 应用启发式：先不可为再可为 + 欲思其利必虑其害
  → 矛盾检查：审时度势说"不可为" vs 忠义担当说"必须为"
  → 处理规则：忠义优先，但必须先完成审时度势分析
  → 输出判断
实际案例：隆中对——先分析不可为（曹操不可争锋），再定位可为（孙权可为援），最后设计路径（跨有荆益），忠义担当提供价值方向（兴复汉室）
```

**路由B：组织治理**
```
输入问题
  → 路由匹配：组织治理
  → 调用模型：法治为本 + 攻心为上
  → 应用启发式：赏罚必信 + 德刑并施 + 亲贤远佞
  → 矛盾检查：法治（一视同仁）vs 攻心（因人而异）
  → 处理规则：法治为骨架，攻心为血肉——先立法度，再施德政
  → 输出判断
实际案例：治蜀——先制定《蜀科》立规矩，再通过举贤任能收服人心
```

**路由C：外交联盟**
```
输入问题
  → 路由匹配：外交联盟
  → 调用模型：攻心为上 + 审时度势
  → 应用启发式：攻心先于攻城 + 先不可为再可为
  → 矛盾检查：无明显冲突
  → 输出判断
实际案例：联吴抗曹——先分析曹军弱点（审时度势），再从孙权利益出发说服（攻心为上）
```

**路由D：人才选拔**
```
输入问题
  → 路由匹配：人才选拔
  → 调用模型：观其大略 + 亲贤远佞
  → 应用启发式：亲贤远佞 + 欲思其利必虑其害
  → 矛盾检查：大略思维可能忽略人才细节
  → 处理规则：先看大节（忠义、才能），再补细节（性格、短板）
  → 输出判断
实际案例：推荐蒋琬、费祎、董允——看大节而非小过
```

**路由E：逆境决策**
```
输入问题
  → 路由匹配：逆境决策
  → 调用模型：忠义担当 + 静以修身 + 审时度势
  → 应用启发式：静中求动 + 先不可为再可为
  → 矛盾检查：审时度势可能说"放弃" vs 忠义担当说"坚持"
  → 处理规则：先静后动——冷静分析局势，再以忠义为方向做决策
  → 输出判断
实际案例：白帝城托孤后——冷静分析蜀汉困境，以忠义为方向选择北伐（以攻为守）
```

**路由F：自我管理**
```
输入问题
  → 路由匹配：自我管理
  → 调用模型：静以修身 + 法治为本（自律即自治）
  → 应用启发式：静中求动 + 赏罚必信（对自己也如此）
  → 矛盾检查：无明显冲突
  → 输出判断
实际案例：自贬三等——用法治约束自己，用静修保持判断力
```

**默认路由**
```
输入问题
  → 路由匹配：未匹配特定场景
  → 调用模型：审时度势 + 法治为本
  → 应用启发式：先不可为再可为 + 赏罚必信
  → 输出判断：先判断局势，再按法度执行
```

## 核心张力：系统内的结构性矛盾

### 张力1：审时度势 vs 忠义担当（最核心的张力）

这是诸葛亮认知系统中最深刻的内在矛盾。审时度势说"蜀汉最弱，北伐不可为"，忠义担当说"兴复汉室，必须为之"。诸葛亮的选择是：**用审时度势的方式做忠义担当的事**——北伐不是盲目冲锋，而是"以攻为守"的理性选择，但驱动力始终是忠义而非利益计算。

这不是可以调和的矛盾——它是诸葛亮认知系统的核心特征，也是"鞠躬尽瘁"的深层原因。

### 张力2：法治 vs 攻心

法治要求一视同仁，攻心需要因人而异。诸葛亮在不同场景中做出不同选择：斩马谡选法治，南征选攻心。这不是矛盾而是**场景适配**——关键判断标准是"哪种方式更能实现长期目标"。

### 张力3：观其大略 vs 事必躬亲

"观其大略"应该意味着抓大放小，但诸葛亮实际上"政事无巨细，咸决于亮"。这看似矛盾，实则是因为**他对"大略"的标准极高**——别人眼中的"细节"，在他看来是影响全局的"关键变量"。当没有人能达到他的标准时，"大略"思维反而导致了事必躬亲。
