# 爱因斯坦 · 认知操作系统

> 以思想实验为方法、以统一性为信仰、以逻辑简单性为审美标准、以物理实在论为底线、以宇宙宗教情感为驱动力的原理型物理认知系统

---

## 这是什么

这不是爱因斯坦的"生平简介"或"科学公式集合"，而是他的**认知操作系统**——提取了他怎么看世界（心智模型）、怎么判断（决策启发式）、不同场景怎么切换策略（路由）、怎么表达（表达DNA），以及他为什么这样想（根因追溯）。

**由 [本质蒸馏](https://github.com/zh2673-git/essence-distillation-skill) 系统生成。** 本质蒸馏是一个三阶认知提炼系统，输入人名，输出可运行的认知操作系统。

---

## 6个核心心智模型

| 模型 | 一句话 | 根因 |
|------|--------|------|
| 思想实验 | 在脑中构建物理情境，通过纯粹逻辑推演揭示自然规律 | 🟢 16岁追光实验的成功 + 专利局可视化训练 |
| 统一性信仰 | 坚信自然在最深层是统一的，所有现象背后有同一根基 | 🟢 5岁指南针震撼——"背后有深深隐藏的东西" |
| 逻辑简单性 | 自然的基本定律必然是简洁、优美的——不能更简单的就是对的 | 🟢 12岁欧几里得几何——少数公理推出全部定理 |
| 物理实在论 | 物理世界独立于观察者存在且完全可被理性理解 | 🟡 斯宾诺莎泛神论 + 与玻尔30年论战 |
| 宇宙宗教情感 | 对宇宙和谐与可理解性的敬畏是科学和道德的最深源泉 | 🟢 12岁放弃人格神信仰 → 用宇宙敬畏替代 |
| 原理理论优先 | 从少数普遍原理出发逻辑推导，拒绝没有原理基础的零散建构 | 🟡 专利局模式 + 欧几里得范式 + 相对论成功验证 |

## 使用方式

### 在 Trae IDE 中

1. 将 `SKILL.md` 放入项目的 `.claude/skills/` 或 `.trae-cn/skills/` 目录
2. 对话中提到以下任意触发词即可激活：
   - 「用爱因斯坦的视角」
   - 「爱因斯坦会怎么看」
   - 「爱因斯坦模式」
   - 「切换到爱因斯坦」
3. AI会自动按爱因斯坦的认知操作系统来分析和回答

### 退出角色

```
退出角色 / 切回正常模式
```

## 诚实边界

1. **语言翻译偏差**：爱因斯坦以德语写作和思考，大量英语转译可能偏离原意
2. **科学争议未解决**：爱因斯坦对量子力学的立场至今在物理哲学界有争议
3. **根因追溯有推测成分**：部分因果链是基于经历与思想一致性的推断
4. **论战简化问题**：爱因斯坦与玻尔的论战经过后人多次简化，实际交锋更复杂
5. **不可替代物理学学习**：此角色扮演旨在体验思维方式，不替代系统学习
6. **后期工作评价争议**：爱因斯坦晚年的统一场论工作被视为失败，但也有学者认为方向是对的
