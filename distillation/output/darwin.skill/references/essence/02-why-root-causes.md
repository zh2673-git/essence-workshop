# 达尔文 · 第二阶蒸馏：根因追溯

## 一句话总结

> 达尔文认知系统的形成根因 = **博物学家族的观察基因 + 贝格尔号航行的经验冲击 + 马尔萨斯的逻辑火花 + 地质学均变论的思维框架 + 维多利亚科学文化的方法论压力**

## 详细根因追溯

### 自然选择思维
- **源头**：贝格尔号航行中对物种变异的观察（1831-1836）
- **强化**：1838年读马尔萨斯《人口论》——"生存斗争"概念提供了选择机制
- **验证**：人工选择（育种）的类比——如果人能做到，自然也能
- **演化**：从纯自然选择→后期承认性选择和其他因素
- **可信度**：🟢 确定——达尔文亲口记载

### 渐进积累
- **源头**：塞奇威克的地质学课程（1831）和莱尔的均变论
- **强化**：贝格尔号期间观察地质地层——微小变化长期积累的实证
- **验证**：达尔文拒绝任何"跳跃"解释，即使在批评压力下也只做最小让步
- **演化**：从严格渐进→在詹金批评后承认可能有较大变异
- **可信度**：🟢 确定

### 观察优先
- **源头**：童年对自然的热爱——7岁学花卉，收集甲虫
- **强化**：亨斯罗的指导——"观察一切，记录一切"
- **验证**：贝格尔号5年考察、藤壶8年研究——极致观察的成果
- **演化**：从纯归纳→后期承认假设-演绎的价值
- **可信度**：🟢 确定

### 最佳解释推理
- **源头**：剑桥神学教育——深入学习佩利《自然神学》的设计论论证
- **强化**：需要反驳设计论——最好的方式是展示自然选择解释力更强
- **验证**：《物种起源》的成功——科学界被说服
- **可信度**：🟡 强推测——达尔文没有明确说"我学神学是为了反驳它"

### 种群思维
- **源头**：加拉帕戈斯雀鸟——同一祖先的不同变种
- **强化**：藤壶分类中面对的个体差异——没有"典型"个体
- **验证**：迈尔指出这是达尔文最革命性的贡献
- **可信度**：🟡 强推测——迈尔的解读，达尔文本人未明确表述
