# 达尔文 · 认知操作系统 (darwin-essence)

> "我的大脑似乎变成了一台从大量收藏中研磨出一般规律的机器。" —— 查尔斯·达尔文

## 概述

基于7维度深度调研，通过三阶蒸馏提炼达尔文的认知操作系统。包含5个核心心智模型、7条决策启发式、7条场景路由和完整的表达DNA。

## 核心心智模型

| # | 模型名称 | 一句话定义 |
|---|---------|-----------|
| 1 | 自然选择思维 | 通过变异-选择-保留的循环解释一切复杂适应 |
| 2 | 渐进积累 | 微小变化长期积累可产生巨大结果 |
| 3 | 观察优先 | 先大量观察再归纳规律，事实先于理论 |
| 4 | 最佳解释推理 | 通过对比竞争假说的解释力来选择最佳理论 |
| 5 | 种群思维 | 关注群体中的变异而非寻找"典型" |
