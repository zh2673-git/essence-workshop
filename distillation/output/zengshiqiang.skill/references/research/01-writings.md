# 曾仕强 · 著作与系统思考

> 本文件调研曾仕强（1934年8月28日—2018年11月11日）的著作体系、反复出现的核心论点、自创术语、推荐书单与思想源头、各领域核心方法论。
>
> **勘误遵守**：学历以"杜鲁门大学行政管理硕士 + 莱斯特大学管理哲学博士 + 牛津大学管理哲学荣誉博士"为准（详见 `06-timeline.md`）；生卒以"1934年8月28日生于福建漳州 / 2018年11月11日19时54分逝于台湾三军总医院"为准；兴国管理学院：曾仕强为首任校长（赵景霖创办）。
>
> **信源黑名单遵守**：未引用知乎、微信公众号、百度百科、百度知道作为主要信源。涉及"百科"仅指 baike.com（微信百科）；自媒体/读书会文章仅用于交叉验证已由权威信源确认的事实。

## 核心论点（≥3次出现视为真信念）

> 标注出现次数的方法：在 ≥3 个独立载体（书籍/讲座/语录/访谈）中出现方列为真信念。来源后括号标注：[曾自述] / [别人转述] / [推断]

### 1. **持经达变是最有效的管理方式**（出现 ≥6 次，真信念）

- 出处1：《中国式管理》（2005）"持经达变是最有效的管理方式，有原则，却必须因人、因时、因事、因地而应变，以求制宜"——曾自述
- 出处2：《中道》（2018，北京联合出版公司）"持经达权而生生不息"——曾自述
- 出处3：《中国式管理语录50条》第18条、第19条"经是方的，规规矩矩，实实在在。权是变动的意思，要持经达权，合理应变，才能圆通而安人"——曾自述（转述于CSDN/2007年发布）
- 出处4：头条文章引述"《易经》没有讲过要求新求变，《易经》只说继旧开新，要从旧的开出新的来……中国人在变与不变上的原则就只有4个字：持经达变"——曾自述（转述）
- 出处5：2016年9月确诊癌症扩散后，公开视频中"它跟我协商，我让它活着，它也让我活着"——曾自述，是其以"持经达变"应对生死考验的实践
- 出处6：搜狐文章比较"曾仕强持经达变"vs"戴欣明动态平衡"——别人转述/评论

> **本质提炼**：经=不变的法则（道、伦理、良心）；变=方法论的灵活应变。持经达变是变与不变的兼顾，反对"求新求变"和"刻板不变"两个极端。

### 2. **中国式管理三大主轴：以人为主、因道结合、依理应变**（出现 ≥4 次，真信念）

- 出处1：《中国式管理》（2005）系统提出
- 出处2：《中国式管理语录50条》第22条"中国式管理具有三大主轴，那就是以人为主，因道结合，以及依理应变"
- 出处3：《中道》（2018）三向度展开
- 出处4：podcasts.apple.com/cn 正版作品大合集介绍："《中国式管理》——以人为主，因道结合，依理而变"

> **三主轴含义**：①以人为主：管理应该以人为主，因为事在人为；②因道结合：理念相同才能密切配合，"道不同，不相为谋"；③依理应变：凡事依据原则，因人、因事、因时、因地而应变，以求合理。

### 3. **管理是修己安人的历程**（出现 ≥4 次，真信念）

- 出处1：《中国式管理》（2005）核心定义
- 出处2：《中道》（2018）安人之道展开
- 出处3：《中国式管理语录50条》第1条、第10-15条
- 出处4：podcasts大合集介绍："《中道》——明道进德，持经达变，修己安人"

> **修己安人逻辑**：起点是修己（管理者自我修养），终点是安人（让组织成员安心、安宁、安顿）。计划、组织、领导、控制所有管理职能都围绕"安人"展开。

### 4. **美式管理=个人主义、日式管理=集体主义、中式管理=交互主义**（出现 ≥3 次，真信念）

- 出处1：《中国式管理语录50条》第20条"美国式管理的哲学基础是个人主义，日本式管理的哲学基础是集体主义，中国式管理则是我们常用的交互主义"
- 出处2：第21条"日本人拿中国的管理哲学，来运用西方的管理科学"
- 出处3：第28条"中国式管理的交互主义，秉持'二合一'的态度，将个人主义和集体主义这两种极端的说法，合在一起，形成'在集体中完成个人'的合理主义"

### 5. **中西差异：法理情（西）vs 情理法（中）**（出现 ≥3 次，真信念）

- 出处1：《中国式管理语录50条》第33条"法是过去的产物，情是未来的埋伏，只有理才是现在的指标，中国式管理主张依理应变"
- 出处2：podcasts大合集介绍："中华文化的三阶智慧，从法理情到情理法"
- 出处3：相关文章总结"西方是法理情，中国是情理法：先讲人情，再讲道理，最后讲法律"

### 6. **圆通 ≠ 圆滑**（出现 ≥3 次，真信念）

- 出处1：《圆通的人际关系》（2008）
- 出处2：《人际的奥秘》编辑推荐语"光讲技巧，容易导致'圆滑'，只有注入伦理道德的因素，才能变'圆滑'为'圆通'"
- 出处3：《人性管理》"管理者外圆内方的领导艺术"

> **区分**：圆滑=只有技巧无伦理；圆通=技巧+伦理道德。差异在于是否注入"伦理道德"的因素。

### 7. **《易经》三义：变易、不易、简易**（出现 ≥3 次，真信念）

- 出处1：CCTV百家讲坛《易经的奥秘》第六集"易有三义"（CCTV-10，2009年，节目单原文确证）
- 出处2：docin《百家讲坛—易经奥秘笔记》"易有三义：简易、变易、不易——出自《易纬·乾凿度》"
- 出处3：曾仕强品三国、相关文章等多处引述

> **三义含义**：①变易：世界唯一不变的就是变化本身；②不易：万变之中有永恒不变的"道"（良心、伦理、道德、自然规律）；③简易：最高深的道理往往最简单（阴阳平衡、情理兼顾、心安理得）。

### 8. **大同必须包容小异，世界大同并非世界一同**（出现 ≥3 次，真信念）

- 出处1：《中国式管理语录50条》第31条"大同必须包容小异，世界大同并非世界一同"
- 出处2：半月谈"共筑中国梦"访谈（曾仕强本人亲口）："'世界大同'他们所讲的是'地球村'……这两个非常接近，方向都对，方法不一样。我们是'一统天下'，他们是'统一天下'，就这么'一统'和'统一'不同而已"——曾自述（一手）
- 出处3：podcasts大合集介绍："中华文化的三阶智慧"

### 9. **中国人擅长把二看成三，以二合一代替二选一**（出现 ≥3 次，真信念）

- 出处1：《中国式管理语录50条》第16条
- 出处2：podcasts大合集"二合一为何大于二选一？融合与取舍的智慧揭秘"
- 出处3：语录第32条"凡事未定案之前，十分民主，一旦拍板定案，相当独裁。这种把民主和独裁合起来想，称为专制"——把二看成三的运用

> **本质**：西方人"二选一"（要/不要、好/不好），中国人"二合一"（好就是不好，不好就是好——因为好里有不好的部分，不好里有好的部分）。这是中国式管理的方法论根基。

### 10. **《易经》"没有好坏"，好会变坏，坏会变好**（出现 ≥3 次，真信念）

- 出处1：曾仕强亲口引述"'《易经》里面是没有好坏的，好会变坏，坏会变好，意思就是好就是坏，坏就是好。因为好的里面有坏的因素，随时会发生的；坏的里面，它也有很多好的因素，它会随时翻过来，把所有坏的成分统统取代掉"——曾自述（头条文章引用）

## 自创术语与概念体系

### A. 核心管理学概念

## 核心方法论（按领域：管理 / 易经 / 人际 / 情绪 / 领导）

### 1. 管理领域：M理论 × 六字真诀 × 修己安人

**M理论三向度**（《中道》2018系统提出；其雏形见《中道管理—M理论及其应用》2007）：

| 向度 | 对应 | 核心要义 |
|------|------|---------|
| **安人之道**（仁） | 仁 | 管理的最终目标是"安人"——让员工身安心乐、视员工有如家人；安顾客第一优先、安员工以厂为家、安股东持续发展、安社会形象良好 |
| **经权之道**（义） | 义 | 经=不变的法则，权=应变的手段；变时要以安为前提；随时都应将心比心 |
| **絜矩之道**（礼） | 礼 | 絜矩=将心比心；让员工自动自发；制度要自生自长、适时修订；务求人人都能安 |

**六字真诀**（2004年国家机关事务管理局司局级干部培训首提）：

| 字 | 含义 |
|----|------|
| **两难** | 凡事相对，存在矛盾——看到问题的两面 |
| **兼顾** | 统一矛盾，合起来想——不二选一 |
| **合理** | 寻找此时此地的合理决策——依理应变 |

**修己安人逻辑链**：
- 修己 = 修心（凭良心）+ 修行（守本分）+ 修言（会说话——直话曲说、硬话软说、坏话好说）
- 安人 = 安顾客 → 安员工 → 安股东 → 安社会（按层次顺序求安）
- 安人的历程：开心 → 交心 → 共同关心 → 同心

**树状组织精神**：
- 根部吸收水份源源不断供应树干；树干毫不保留让枝叶予取予求
- "我支持你，你放手去做"——符合中国人"你办事，我放心"的心理需求
- 反对"上侵下职"——上司侵犯下属职责会妨害员工学习成长

### 2. 易经领域：三义 × 三圣 × 持经达变 × 善易不卜

**易经三义**（CCTV百家讲坛《易经的奥秘》第六集，2009）：

| 义 | 含义 |
|----|------|
| **简易** | 最高深的道理往往最简单；筷子2支一动一不动；毛笔1支能粗能细——简单中含阴阳 |
| **变易** | 世界唯一不变的是变化本身；《易经》最早叫《变经》；变是阳，常是阴 |
| **不易** | 万变之中有永恒不变的"道"——良心、伦理、道德、自然规律 |

**易经的"三把钥匙"**：
1. 伏羲八卦——阴阳是构成宇宙万物最基本的元素
2. 文王六十四卦——宇宙的64个密码
3. 孔子《十翼》（《易传》）——把易经变成非常严肃的哲学

**善易不卜**：
- 孔子："不占而已矣"；荀子："善易者不占"
- 真正读懂《易经》的人不用它来占卜算卦
- 卜卦是"充分引发自己潜能的过程，同时可以寻找更多的资讯，作为判断事物、做出决定的参考"

**超越吉凶**（《易经的奥秘》第十五集）：
- 人生是吉凶悔吝的循环往复——"无奈的必然律"
- 唯谦卦能跳出吉凶悔吝的必然律，立于不败之地

**孔子一生的六阶段卦象**（《易经的奥秘》第八集、第十一集）：
- 64卦每卦6爻的共同规律："初难知，上易知；二多誉，五多功；三多凶，四多惧"
- 孔子一生分六阶段：十五而学 → 三十而立 → 四十不惑 → 五十知天命 → 六十耳顺 → 七十从心所欲不逾矩

### 3. 人际领域：十大要领 × 圆通分是非 × 沟通四层次

**人际十大要领**（《人际的奥秘》）：

| 序号 | 要领 | 心法 |
|------|------|------|
| 一 | 一表人才 | 不要以貌取人；提升第一印象；正确认识自己 |
| 二 | 两套西装 | 外在表现与内在表现同样重要；外表应符合社会文化背景 |
| 三 | 三杯酒量 | 爱好并非坏事；凡事要适可而止；酒桌交际的技巧 |
| 四 | 四圈麻将 | 赢到后来总是输；人际关系要经营 |
| 五 | 五方交游 | 多个朋友多条路；严于律己获益多 |
| 六 | 六出祁山 | 明知不可为而为之；朋友之间可以通财 |
| 七 | 七术打马 | 多说恭维话；少去讨好人 |
| 八 | 八口吹牛 | 要自己肯定自己；让别人来赞扬你 |
| 九 | 九分努力 | 努力没有用；运气很重要 |
| 十 | 十分忍耐 | 会忍才能赢；欲速则不达 |

**圆通分是非**（《人际的奥秘》编辑推荐语）：
- 西方人"二分法"——对就是对，错就是错
- 中国人已摆脱"二分法"——"错，绝对不可以；对，常常没有用"
- "水至清则无鱼，人至察则无徒"——在圆通中分是非
- 把是非分得大家都有面子，不得罪人，但也不讨好人

**沟通四层次**：
- 不沟不通 → 沟而不通 → 沟而能通 → 不沟而通（最高境界）

**沟通艺术**：
- "使对方听得进去"——先察言观色
- 站在对方立场——不明言为基础
- 中国人听的不是道理，是态度

### 4. 情绪领域：情绪四步管理法 × 反求诸己

**情绪的基本观念**（《情绪的奥秘》）：
- 情绪没有好坏、优劣之分，只是人对环境的一种反应
- 情绪反应是"警钟"——告诉人们"你的反应有点问题"
- 情绪有理性和非理性两大类——非理性情绪源于"夸大"与"不符合实际"

**情绪四步管理法**（《情绪的奥秘》，搜狐读书文章归纳）：

| 步骤 | 要点 |
|------|------|
| 觉察 | 像旁观者一样观察自己的情绪波动——喉咙收缩、肩膀紧张、脑子混乱、心情恶劣时去查清这些感觉的产生 |
| 命名 | 准确说出此刻的感受（是"委屈"还是"失望"？） |
| 溯源 | 找到情绪触发的深层原因 |

- 自己是情绪负债的制造者

### 5. 领导领域：外圆内方 × 总裁魅力学 × 老板干部配合

### 保留的矛盾

1. **生年矛盾**：雪梅读书会文章（meipian.cn/5cqv0diz）与部分自媒体误称"1935年10月20日生"——权威信源（mbalib、半月谈、文汇网讣告、新浪讣告）一致确认为"1934年8月28日生于福建漳州"。详见 `06-timeline.md` 勘误。
2. **学历矛盾**：任务描述原称"美国UMass MBA"——权威信源一致指向"美国杜鲁门大学（Truman State University）行政管理硕士 + 英国莱斯特大学管理哲学博士 + 牛津大学管理哲学荣誉博士"。详见 `06-timeline.md` 勘误。
3. **兴国管理学院角色矛盾**：任务描述原称"2000年创办兴国管理学院"——兴国管理学院百科条目明确：赵景霖创办，曾仕强为**首任校长**（2000年9月—2003年7月31日）。
4. **任务描述提及的部分著作找不到独立条目**：《中国式管理思维》《中国管理哲学》《超国界管理》《二十四孝》——分别可能是误称、领域统称、未出版独立著作、反思性解读（详见上文"任务描述中提及但未找到独立条目的著作"小节）。
5. **《曾仕强品三国》（2022）的作者身份**：豆瓣条目署名"曾仕强"，但该书出版时曾仕强已去世4年（2018年11月11日去世），应为**遗稿整理版**，由时代光华整理出版。
