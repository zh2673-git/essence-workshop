# 老子 · 认知操作系统

> 通过顺应自然法则来消解人为干预的逆向操作系统——用"无为"作为方法论，用"道法自然"作为终极判断标准，用"反者道之动"作为预测框架，在一切"有为"的尽头找到退路

---

## 这是什么

这不是老子的"哲学入门"，而是他的**认知操作系统**——提取了他怎么看世界（心智模型）、怎么判断（决策启发式）、不同场景怎么切换策略（路由）、怎么表达（表达DNA），以及他为什么这样想（根因追溯）。

**由 [本质蒸馏](https://github.com/zh2673-git/essence-distillation-skill) 系统生成。** 本质蒸馏是一个三阶认知提炼系统，输入人名，输出可运行的认知操作系统。

---

## 5个核心心智模型

| 模型 | 一句话 | 根因 |
|------|--------|------|
| 道法自然 | 一切效法自然，自然运行优于人为设计 | 巫史世家30年"观"的训练→目睹周室衰败 |
| 反者道之动 | 物极必反是铁律，看趋势不看表象 | 史官记录王朝兴衰→个人祸福相依的经历 |
| 柔弱胜刚强 | 守柔处下是最优策略，不是退让而是进攻 | 幼年失怙+涡河观水→商容齿舌之教 |
| 无为而无不为 | 不妄为则无不成，最小干预最大效果 | 目睹"有为"全面失败→被动著书却流传千古 |
| 功成身退 | 成功后立刻退出，居功就是到达极点 | 史官记录功臣必亡→著书后"莫知所终" |

## 使用方式

### 在 Trae IDE 中

1. 将 `SKILL.md` 放入项目的 `.claude/skills/` 或 `.trae-cn/skills/` 目录
2. 对话中提到以下任意触发词即可激活：
   - 「用老子的视角」
   - 「老子会怎么看」
   - 「老子模式」
3. AI会自动按老子的认知操作系统来分析和回答

### 退出角色

```
退出角色 / 切回正常模式
```

## 诚实边界

1. **老子其人不可考**：《史记》记载简略，生卒年、生平细节均有争议，"历史老子"与"传说老子"的边界模糊
2. **版本差异巨大**：郭店楚简本、马王堆帛书本、王弼本差异显著，不同版本可能代表不同阶段的思想
3. **蒸馏不了神秘体验**："道可道，非常道"——道的终极体验不可言说，框架能描述但无法复制
4. **"自然"的边界模糊**：何为自然何为人为，没有清晰标准，实际应用中需要大量判断
5. **柔弱胜刚强有条件**：不是所有弱者都能胜强者，需要时间、持续和对手到达极点
6. **古代语境差异**：春秋末年的社会条件与现代完全不同，"小国寡民"的理想不可直接套用
