# Agent 6: 人物时间线

## 完整时间线

| 时间 | 事件 | 思想影响 |
|------|------|---------|
| **1924.1.1** | 出生于内布拉斯加州奥马哈市 | 大萧条时期成长，塑造节俭和韧性 |
| **1930s** | 在巴菲特祖父的杂货店打工 | 与巴菲特童年有交集但未相遇 |
| **1941** | 进入密歇根大学学习数学 | 奠定数学和概率思维基础 |
| **1943** | 19岁生日后退学，加入陆军航空队 | 军事纪律+气象学训练 |
| **1943-1945** | 在Caltech学习气象学，晋升少尉 | 科学思维、概率分析、数据解读 |
| **1945** | 与Nancy Huggins结婚 | 第一段婚姻 |
| **1946** | 通过GI Bill进入哈佛法学院（无本科学位） | Roscoe Pound推荐，破格录取 |
| **1948** | 哈佛法学院magna cum laude毕业 | 法律训练：精确论证、先例分析、伦理框架 |
| **1948-1950s** | 在加州Wright & Garrett律师事务所执业 | 公司法、交易结构、尽职调查 |
| **1953** | 离婚，失去房产 | 人生低谷，重建 |
| **1954** | 9岁儿子Teddy确诊白血病 | 自费治疗，一年后去世，人生最痛苦经历 |
| **1956** | 与Nancy Barry再婚 | 最终育有8个孩子 |
| **1959** | 在奥马哈晚宴上首次遇见巴菲特 | "一见如故"，开始长期合作 |
| **1962** | 创立Munger, Tolles & Olson律师事务所 | 加州顶级律所之一 |
| **1962** | 与Jack Wheeler创立Wheeler, Munger & Co. | 投资合伙公司，年化19.8%回报 |
| **1960s初** | 通过房地产赚到第一个100万美元 | "树套利" — 集中投资帕萨迪纳公寓 |
| **1972** | 说服巴菲特收购See's Candies（2500万美元） | 伯克希尔投资哲学转折点 |
| **1973-1974** | 熊市亏损53%+，几乎破产 | 确立"避免杠杆"铁律 |
| **1975** | 辞去Wheeler, Munger领导职务 | 公司1976年清盘 |
| **1978** | 成为伯克希尔·哈撒韦副主席 | 与巴菲特正式合作 |
| **1984** | 发表著名演讲《人类误判心理学》 | 系统阐述25种认知偏差 |
| **1984-2011** | 担任Wesco Financial董事长兼CEO | 独立投资平台 |
| **1991** | 处理所罗门兄弟丑闻 | 危机管理 |
| **1993** | 首次登上福布斯400富豪榜 | 财富公开 |
| **1997-2023** | Costco独立董事 | 长期价值投资典范 |
| **2005** | 《穷查理宝典》出版 | 思想系统化整理 |
| **2008** | 投资比亚迪（通过李录推荐） | 中国投资里程碑 |
| **2021** | 每日期刊重仓阿里巴巴 | 晚年最大投资失误 |
| **2023.10** | Acquired Podcast首秀（唯一播客） | 生前最后深度访谈 |
| **2023.11.28** | 在加州医院安详去世，享年99岁 | 距100岁生日差34天 |

## 最近12个月动态（2022.11-2023.11）

- **2022.5**: 获得连任伯克希尔·哈撒韦副主席
- **2023年初**: 每日期刊年会，继续批评加密货币和SPAC
- **2023.10**: Acquired Podcast访谈，谈Costco、比亚迪、日本投资机会
- **2023.11.28**: 去世，巴菲特发表声明："没有查理的灵感、智慧和参与，伯克希尔不可能达到今天的地位"

## 思想转折点

1. **1954年儿子去世**: 从"正常生活"转向"极度理性"的防御性思维
2. **1959年遇见巴菲特**: 从"独立投资者"转向"合作伙伴"
3. **1972年See's Candies**: 从"格雷厄姆式便宜货"转向"优质企业"
4. **1973-74年几乎破产**: 从"使用杠杆"转向"极度厌恶风险"
5. **1984年人类误判心理学**: 从"纯投资"转向"认知科学+投资"
