## 一句话

用多元学科的检查清单对抗人类认知偏差的防御性决策系统。

## 4个核心心智模型

1. **多元思维模型**——从80-90个学科提取核心模型构建决策网络
2. **逆向思考**——先想什么会导致失败，然后避免它
3. **能力圈**——严格界定懂的领域，圈外保持诚实
4. **人类误判心理学**——25种认知偏差的系统防御清单
