## 一、孔子核心著作与文本源流

### 1.1 《论语》——最可靠的孔子言行记录

- **性质**：孔子与弟子的对话录，由其弟子及再传弟子编纂，非孔子本人所著。
- **篇幅**：共20篇，约1.2万字（A）。
- **形成时间**：战国初期即已流传，西汉时定型（A）。
- **文本地位**：汉代被视为辅翼《五经》的"传"；南宋朱熹将其收入《四书》，成为科举必读（A）。

### 1.2 "五经"——孔子整理/编纂的古代经典

传统认为孔子"删《诗》《书》，定《礼》《乐》，赞《易》，作《春秋》"。

| 经典 | 传统说法 | 现代学术质疑 | 可信度 |
|------|---------|-------------|-------|
| 《诗》 | 孔子从3000篇删至305篇 | 不可信；可能是孔子用作教材 | A/C |
| 《书》 | 孔子编纂 | 可能部分由孔子整理 | C |
| 《礼》 | 孔子"定礼" | 《仪礼》《周礼》《礼记》非一时之作 | C |
| 《乐》 | 孔子"正乐" | 已失传，仅存残篇 | A/C |
| 《易》 | 孔子作"十翼"（《易传》） | 多数学者认为《易传》成于战国至汉初，非孔子亲作 | B |
| 《春秋》 | 孔子"作《春秋》" | 可能是孔子据鲁国史记编纂，但"微言大义"为后人发挥 | A/B |

**关键区分**（A）：
- 孔子自述："述而不作，信而好古"（《论语·述而》）——他自称传述者而非创作者。
- 这与后世"孔子作《春秋》"说存在张力，矛盾保留。

### 1.3 重要的辅助文献

- **《孔子家语》**：传统被视为王肃伪作，近年出土文献证明其部分内容属实，但需审慎对待（B/C）。
- **《礼记》**中的《大学》《中庸》篇：传统归入儒家经典系统，但成书在孔子之后（A/B）。
- **郭店楚简**（1993年出土）：包含大量早期儒家文献，可与传世文献互校（A）。

> 注："知"在《论语》中多数作"知道/了解"义，作"智"（wisdom）义的需根据上下文甄别。

### 2.2 孔子对核心术语的"重新定义"

以下术语不是孔子凭空创造，而是他赋予了全新内涵：

#### 仁——孔子思想的核心（A）

| 维度 | 孔子原话 | 解读 |
|------|---------|------|
| 根本定义 | "爱人"（《颜渊》） | 对他人的关怀 |
| 实践纲领 | "克己复礼为仁"（《颜渊》） | 约束自己使言行符合礼 |
| 具体表目 | "非礼勿视，非礼勿听，非礼勿言，非礼勿动"（《颜渊》） | 四勿 |
| 推己及人（积极） | "己欲立而立人，己欲达而达人"（《雍也》） | 忠 |
| 推己及人（消极） | "己所不欲，勿施于人"（《颜渊》） | 恕 |
| 五德条目 | "恭、宽、信、敏、惠"（《阳货》） | 实践仁的五种品德 |
| 道德基础 | "孝弟也者，其为仁之本与"（《学而》——有子所言） | 血缘亲情是仁的根基 |

**孔子所说 vs 后人解读**：
- 孔子从未给"仁"下过一个统一定义。他对不同学生问"仁"给出不同回答（因材施教——A）。
- 孔子说"不知其仁也"评价子路、冉求、公西赤（《公冶长》），连最亲近的学生都不轻易以"仁"许之。这说明"仁"在孔子这里是极高的境界，并非人人可轻易达到（A）。
- 后世将"仁"泛化为"全德之名"（朱熹等），是后人的理论化概括，孔子自己未做此提炼（B）。

#### 君子——从身份到品德的转化（A）

- 原义：国君之子/贵族（身份概念）。
- 孔子赋予新义：道德高尚的人（品德概念）。
- "君子去仁，恶乎成名？君子无终食之间违仁"（《里仁》）。
- 对位概念"小人"也经历了类似的语义转化。

#### 礼——从仪式规范到社会秩序（A）

- 孔子继承周礼，但将礼从"仪节"上升为"社会秩序的根基"（B）。
- "人而不仁，如礼何？"（《八佾》）——礼若无仁为基础则空（A）。
- 在孔子那里，仁是内容（内在），礼是形式（外在），二者互为表里（由学者总结的逻辑，孔子本人未做此明确表述——B）。

#### 中庸——作为至高德性（A）

- "中庸之为德也，其至矣乎！民鲜久矣"（《雍也》）。
- "过犹不及"（《先进》）。
- 孔子未对"中庸"做系统阐发，其方法论由后儒（子思《中庸》篇）发扬（A/B）。

#### 忠恕——孔子"一以贯之"之道（A）

- 曾子总结："夫子之道，忠恕而已矣"（《里仁》）。
- 忠：尽己（积极推己及人）；恕：推己（消极的不施于人）。
- 这是孔子自己明确的方法论——"吾道一以贯之"（《里仁》）。

### 2.3 概念之间的关系（系统框架）

学者分析总结的核心逻辑链（B/C）：

```
         仁（内在价值核心）
           ↕
    义（道德判断/应当）
           ↕
礼（外在规范/社会秩序）
           ↕
   乐（和谐/教化完成）

志于道 → 据于德 → 依于仁 → 游于艺（《述而》）
```

**孔子本人明确说过的关系**（A）：
1. "人而不仁，如礼何？人而不仁，如乐何？"（《八佾》）——仁是礼乐的前提。
2. "克己复礼为仁"（《颜渊》）——践行礼就是仁的实践。
3. "君子义以为上"（《阳货》）"君子义以为质"（《卫灵公》）——义是君子的本质。
4. "道之以德，齐之以礼，有耻且格"（《为政》）——德礼结合优于政刑。

### 2.4 矛盾与张力（保留）

1. **孔子"罕言仁" vs 《论语》中"仁"出现110次**（A）：
   - 原文："子罕言利与命与仁"（《子罕》）。
   - 事实：《论语》中"仁"是最常见概念之一。
   - 可能解释：孔子平日教学不常谈仁，但弟子认为重要，所以记录了下来。
   - 或："仁"字110次中含弟子所言、他处引用，孔子本人说"仁"的频率较低。

2. **"述而不作" vs "作《春秋》"**（A）：
   - 孔子自称"述而不作"（《述而》）。
   - 但《孟子》说"孔子成《春秋》而乱臣贼子惧"，强调创作性。
   - 现代学者认为："述"指对已有传统的阐述，"作"未违反；也有认为《春秋》是将鲁国史记加以裁剪，本质上仍是"述"。

3. **仁与礼何者为重**（B）：
   - 孔子既说"克己复礼为仁"（礼先），又说"人而不仁如礼何"（仁先）。
   - 学者倾向认为"仁"是本质、"礼"是表现，但二者在孔子体系中是循环互证的，并非直线因果。

## 三、孔子推荐的书目与评价

### 3.1 直接推荐（《论语》原文——A）

| 书名 | 孔子原话 | 原文出处 |
|------|---------|---------|
| 《诗》 | "《诗》三百，一言以蔽之，曰'思无邪'" | 《为政》 |
| 《诗》 | "不学《诗》，无以言" | 《季氏》 |
| 《诗》 | "兴于《诗》，立于礼，成于乐" | 《泰伯》 |
| 《诗》 | "诗，可以兴，可以观，可以群，可以怨。迩之事父，远之事君，多识于鸟兽草木之名" | 《阳货》 |
| 《周南》《召南》 | "人而不为《周南》《召南》，其犹正墙面而立也与？" | 《阳货》 |
| 礼 | "不学礼，无以立" | 《季氏》 |
| 《易》 | "加我数年，五十以学《易》，可以无大过矣" | 《述而》 |

### 3.2 间接论及（A）

- **《书》**："子所雅言，《诗》《书》、执礼，皆雅言也"（《述而》）——孔子用标准语（雅言）读《诗》《书》和行礼。
- **《乐》**："吾自卫反鲁，然后乐正，《雅》《颂》各得其所"（《子罕》）——孔子对音乐做过整理。
- **《春秋》**：孔子未在《论语》中提及自己与《春秋》的关系；"作《春秋》"的说法出自《孟子》。

### 3.3 孔子读书习惯（A）

- **韦编三绝**：孔子晚年读《易》极为用功，穿竹简的牛皮绳断了很多次（《史记·孔子世家》，非《论语》原文——C级转述）。
- **"学而不厌"**（《述而》）：终身学习的自我描述。
- **"温故而知新"**（《为政》）：学习方法论。

## 五、研究建议与注意事项

1. **首选读本**：
   - 原文：《论语》杨伯峻《论语译注》（现代最通行）。
   - 学术：刘宝楠《论语正义》（清）或朱熹《论语集注》（宋）。
2. **研究原则**：
   - 引孔子原文时优先以《论语》为据，其次《左传》《孟子》中的转述。
   - 区分"孔子说的" vs "传统认为孔子说的"（如"十翼"为孔子作已被现代学术基本否定）。
   - 区分"前孔子时代的经典原貌"与"经孔子整理后的面貌"。
3. **出土文献关注**：
   - 郭店楚简、上博简中有与《论语》风格相近的文献，可互校。
   - 《孔子诗论》（上博简）对理解孔子《诗》学有重要价值。
