# Agent 2: 长对话与即兴思考

## 核心对话场景

### 1. 龙场悟道后的讲学（1508-1509，贵阳）
- **场景**：贵州龙场，简陋的龙冈书院
- **对话对象**：当地夷人、贵州提学副使席书、贵阳诸生
- **核心内容**：首揭"知行合一"之说
- **即兴表现**：
  - 被追问"何为第一等事"时，少年即答"读书学圣贤"而非"读书登第"
  - 龙场期间被问"圣人处此更有何道"，自答自悟"圣人之道，吾性自足"
  - 教学者静坐体悟，"只教学者存天理、去人欲，为省察克治实功"

### 2. 天泉桥论学（1527，绍兴）
- **场景**：赴广西平叛前夜，与弟子钱德洪、王畿泛舟天泉桥
- **对话对象**：钱德洪（保守解）、王畿（激进解）
- **核心内容**：传"四句教"
- **即兴表现**：
  - 当场面对两弟子的分歧，给出调和："二者相资为用，不可偏废"
  - 嘱咐钱德洪："以后与朋友讲学，切不可失了我的宗旨"
  - 展现了对自己学说可能被曲解的预见性

### 3. 平定宁王后的政治应对（1519-1520）
- **场景**：宁王被擒后，明武宗要求"亲擒叛贼"
- **对话对象**：太监张永、明武宗近臣
- **核心内容**：功成身退的政治智慧
- **即兴表现**：
  - 将宁王交付张永，称病辞官归隐
  - 面对荒唐的"放归宁王重擒"提议，选择低调保全大局
  - 体现了"此心不动"在政治危机中的运用

### 4. 南赣剿匪期间的教导（1516-1518）
- **场景**：军营中、战后反思
- **对话对象**：将领、士兵、地方官员
- **核心内容**："破山中贼易，破心中贼难"
- **即兴表现**：
  - 首创"教卒之制"，亲自训练骨干
  - 将军事行动与民生疾苦直接关联："叛军越早一日出南昌城，百姓就越早一日遭受战乱"
  - 用"致良知"激发士兵内在动力，而非单纯命令

### 5. 与湛若水的定交（1505前后）
- **场景**：北京，学术交往
- **对话对象**：湛若水（甘泉）
- **核心内容**：共以倡明圣学为事
- **即兴表现**：
  - 一见定交，成为终身学术挚友
  - 湛若水后来撰写《阳明先生墓志铭》，提出"五溺"说
  - 展现了阳明在学术圈的影响力与人格魅力

## 被追问时的回答方式

### 模式1：回归本心
- **典型**：当弟子问"如何格物"时，答"格者，正也。正其不正以归于正"
- **特点**：不向外求答案，而是引导对方反观内心

### 模式2：事上验证
- **典型**：被问"知行如何合一"时，举"知孝必行孝"的例子
- **特点**：用具体生活场景说明抽象道理

### 模式3：直指核心
- **典型**："吾生平讲学，只是致良知三字"
- **特点**：化繁为简，一句话点出本质

## 即兴类比

### 1. 军事与修心的类比
- "破山中贼易，破心中贼难"
- 将剿匪战术与克己功夫相类比

### 2. 镜子与心的类比
- "此心如明镜止水"
- 心不动时能照见万物真相

### 3. 舵与马的类比
- "志不立，如无舵之舟，无衔之马，漂荡奔逸，终亦何所底乎"
- 用日常事物说明立志的重要性

## 改变立场的瞬间

### 1. 从程朱理学到心学
- **触发**：格竹失败 + 龙场悟道
- **转变**："始知圣人之道，吾性自足，向之求理于事物者误也"
- **标志**：从向外求理转向向内求理

### 2. 从佛老到儒学
- **触发**：弘治十八年（1505），34岁
- **转变**：从"出入佛老"回归"圣贤之学"
- **标志**：开始专志授徒讲学

### 3. 教学方法的变化（教亦三变）
- **第一阶段**（贵阳，1509）：首揭"知行合一"
- **第二阶段**（滁州，1514）：多教学者"静坐"
- **第三阶段**（江右，1520）：专提"致良知"
