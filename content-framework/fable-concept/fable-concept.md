# 寓言故事框架

**寓言化的认知方法**（三阶合一框架的叙事式表达应用）：

```
寓言结构
  │ 方法：角色A（代表概念主体）想做X
  │       → 遇到阻碍Y（代表概念要解决的核心矛盾）
  │       → 产生后果Z（代表概念的运行结果）
  │       → 点破"这个故事讲的，其实是{概念}"
  ↓
取象方法（选择寓言载体）
  │ 方法：按概念类型选载体
  │       动物拟人→行为/心理类，器物拟人→系统/机制类
  │       自然现象→规律/法则类，抽象拟人→元概念/思维类
  ↓
映射回溯校验
  │ 方法：每个寓言元素必须能回溯到本质定义
  │       冲突必须映射反常识根因，而非任意冲突
  │       角色行为必须映射概念核心要素
```

**映射铁律**：
1. 寓言元素找不到概念对应物 → 冗余装饰，删除
2. 寓言元素能找到对应物但无法回溯本质定义 → 表面描述，重新映射
3. 反差/反转必须对应本质定义展开的反常识根因，而非任意冲突

**禁止**：没有冲突的纯描述、角色直接说出道理、旁白式说教。
