# 五蕴法口播稿框架

> 用佛教「五蕴」（色·受·想·行·识）作为口播稿的内容骨架，把认知产物组织成一条从「生活现象」到「认知升级」的完整叙述链。
> 本框架只规定结构，不含实质内容与具体平台交付流程。

## 口播化核心原则

口播稿不是书面文章，是给嘴巴和耳朵用的。自然感的来源不是辞藻，而是**节奏、停顿、语气和人味**。

### 必须区分"书面稿"与"口播稿"

| 书面稿 | 口播稿 |
|---|---|
| 句子完整、逻辑严密 | 允许短句、重复、小修正 |
| 抽象词、连接词多 | 口语词、停顿多 |
| 读者可以回看 | 听众只能听一遍，必须给呼吸点 |
| 信息密度高 | 一次只给一个概念，留消化时间 |

### 停顿与节奏标记系统

为便于录制，口播稿可用以下标记：

| 标记 | 含义 | 示例 |
|---|---|---|
| `，` | 短停顿，约0.5秒 | 你看，锅里的水烧开的时候， |
| `。` | 句间停顿，约1秒 | 中间咕嘟咕嘟往上冒。 |
| `——` | 拉长或强调后的停顿 | 它更像地球打的一个嗝—— |
| `/` | 可换气的呼吸点 | 夏天海洋吃撑了，/热得受不了， |
| `//` | 段落间重停顿，约1.5-2秒 | 。//说白了， |
| `【重】` | 重读 | 它真正的使命【重】是散热。 |
| `【轻】` | 轻读、带过 | 这个现象，【轻】你可能天天见。 |
| `【笑】` | 可带笑意的语气 | 小时候我也怕，【笑】觉得它像个怪兽。 |
| `【停顿】` | 故意留白，让观众反应 | 你觉得呢？【停顿】 |

> 最终发布给观众的文字版可删除这些标记，但录制用稿建议保留。

### 语气词与口头禅词库

适当使用，增加人味，但不过度：

- **引出话题**：你看、你想、说白了、说实话、其实
- **过渡**：那、然后呢、接着、这时候、你会发现
- **强调**：其实、说到底、关键是、特别注意
- **共情**：对吧、是不是、你也这样、我也一样
- **留白**：嗯、啊、这个、那个（少量，自然即可）

### 人声化改写 checklist

生成后逐句检查：

- [ ] 这句话我念出来会憋气吗？（长句拆分）
- [ ] 这里如果停顿一下，观众更好理解吗？（加标点或 `/`）
- [ ] 这个书面词，我日常会怎么说？（替换为口语）
- [ ] 这里是不是太顺了，像背书？（加入小重复或自问）
- [ ] 有没有跟观众直接对话？（每30秒至少出现一次"你"）

## 真人感：从"结构正确"到"像真人说话"

### 真人口播 vs AI口播的关键差异

| AI口播（机械） | 真人口播（自然） |
|---|---|
| 每段职责明确，推进高效 | 允许废话、跑题、突然想到的补充 |
| 比喻精确、术语规范 | 比喻粗糙但好懂，会说“大概就是这么个意思” |
| 逻辑严密，一二三清楚 | 逻辑不一定完美，会说“反正”“怎么说呢” |
| 只在结尾反问 | 中间不断“对吧”“你发现没”“是不是” |
| 信息密度均匀 | 有松有紧，重点处慢，铺垫处快 |

### 增加“人味”的六种方法

1. **允许情绪废料**
   - 加入“真的”“其实吧”“怎么说呢”“我当时就想”
   - 例："我当时就想，这台风跟锅里的水，是不是一回事？"

2. **允许自我修正**
   - 先说一个不太准确的说法，再圆回来
   - 例："也不是怪兽吧，反正就是那种……特别大的气旋。"

3. **允许口语化重复**
   - 重要的话多说一遍，像现场强调
   - 例："它要散热，它得散热，不然海洋就热爆了。"

4. **插入即时互动**
   - 不只是结尾，中间也要不断抛给观众
   - 例："你发现没？每次台风来之前，新闻都这么说。"

5. **加入个人锚点**
   - 每篇至少一个“我”的具体经历或观察
   - 例："我去年看台风路径图，盯了一晚上，就看它怎么拐。"

6. **让比喻保持粗糙**
   - 不要追求完美对应，生活化更重要
   - 例："地球自转把它掰弯"比"科里奥利力使其旋转"更像人话

### 反模式：过度口播化

避免为了自然而自然，以下会显得做作：

- 每句都加“对吧”
- 故意结巴或频繁“嗯嗯啊啊”
- 重复同一个词超过3次
- 自我修正过于夸张

## 核心结构

```
色（现象切入）
  → 受（情感共鸣）
    → 想（理论点亮）
      → 行（行动落点）
        → 识（主题升华）
```

| 蕴 | 内容职责 | 观众心理 | 时长占比 |
|---|---|---|---|
| 色 | 用身边普遍现象/故事/具体场景开场，引出问题 | 代入 | 20% |
| 受 | 说出自己的感受或身边人的感受，建立情绪连接 | 共情 | 15% |
| 想 | 联想到一本/两本有名书里的理论，把现象翻译成认知模型 | 理解 | 25% |
| 行 | 给出可执行的认知行动或思考方向，不要空喊口号 | 落地 | 20% |
| 识 | 主题升华到个人成长、家庭和睦、社会进步、国家富强中的一个或多个层次 | 升华 | 20% |

## 一、色 · 现象切入

### 方法

- 从一个**具体、可观察的生活场景**开始，而不是一个抽象概念
- 优先选择观众**普遍经历过**或**能在身边看到**的现象
- 用一句话把场景和问题同时抛出来

### 组织要点

| 要素 | 说明 | 示例 |
|---|---|---|
| 场景 | 时间、地点、人物、动作 | 周末聚餐，朋友小李又迟到了半小时 |
| 反差 | 表面合理 vs 底下有问题 | 大家都笑着说“没事”，但心里其实都不舒服 |
| 问题 | 由场景自然引出的疑问 | 为什么我们越包容，越容易被当成好欺负？ |

### 朴素之道风格注入

- 用**具体数字**替代模糊词：不是“很多人”，而是“我认识的五个人里，三个都这样”
- 用**大白话**描述场景：像跟朋友聊天，不是新闻联播
- 用**反问或自嘲**结尾，拉近距离
- **开场必须带"你看/你想"**：直接跟观众建立对话感
- **第一句不要解释概念**：先给画面，再给问题

---

## 二、受 · 情感共鸣

### 方法

- 说出**自己真实的感受**，或**身边人的真实反应**
- 不要直接评价对错，先描述“我当初也这样”“我朋友当时挺难受的”
- 用感受承接场景，让观众觉得“他懂我”

### 组织要点

| 类型 | 用法 | 示例 |
|---|---|---|
| 自我感受 | 我曾有过同样的情绪 | 说实话，我以前也觉得，越忍让越显得有格局 |
| 他人感受 | 身边人怎么想的 | 但后来我才发现，那个总迟到的朋友，其实根本没把大家当回事 |
| 群体感受 | 把这个感受放大到普遍 | 很多人不是不生气，是不好意思生气 |

### 朴素之道风格注入

- 承认复杂性：先别急着下结论
- 用口语转折：“说白了”“说到底”“其实不是……而是……”
- 允许 vulnerability：暴露自己的小困惑、小尴尬
- **用"我以前也……"建立同频**：不是居高临下地说教，是一起成长
- **允许小重复**：比如“真的，我当时就是这么想的”，增加真实感

---

## 三、想 · 理论点亮

### 方法

- 从现象和感受中，**自然联想到一本或两本有名书中的理论**
- 用书中的概念把前面的现象“点亮”，让观众从“我也有这种感觉”变成“原来是这样”
- 理论不是用来炫的，是用来解释生活现象的

### 组织要点

| 步骤 | 任务 | 示例 |
|---|---|---|
| 点出理论 | 一句话介绍书名和核心概念 | 《被讨厌的勇气》里有个说法：所有烦恼，都来自人际关系 |
| 翻译到生活 | 把理论对应回前面的场景 | 放到小李这件事上，就是我们太在意“别人怎么看我” |
| 给出认知模型 | 用一个简单结构概括 | 可以把人际关系分成两类：课题分离和课题混淆 |

### 朴素之道风格注入

- **抽象概念先比喻**：把理论比作生活中的东西
- **不要堆砌书名**：最多两本，每本只取一个核心概念
- **大白话翻译**：不是“阿德勒心理学强调课题分离”，而是“说白了，就是你的事归你，我的事归我”
- **理论前加停顿**：说完感受后，用“这让我想起……”或“后来我读到一本书……”自然过渡
- **用“你可以理解为”替代“也就是说”**：更像聊天，不像讲课

## 四、行 · 行动落点

### 方法

- 给出一个**可执行、不空洞**的认知行动或思考方向
- 不是“我们应该努力”“我们要改变”，而是“下次遇到这种情况，你可以先问自己一个问题”
- 行动要小到能在生活中立刻试

### 组织要点

| 类型 | 用法 | 示例 |
|---|---|---|
| 提问式 | 给观众一个自问 | 下次别人再让你不舒服时，先问自己：这是谁的课题？ |
| 场景式 | 在具体场景中怎么做 | 聚会迟到这件事，你可以提前说清楚：我等二十分钟，到点先吃 |
| 认知式 | 改变一个看问题的角度 | 别把忍让当格局，那可能只是不敢表达 |

### 朴素之道风格注入

- **禁用说教腔**：不用“我们应该”“必须认识到”
- **用具体动作替代抽象号召**：不说“要学会拒绝”，而说“你可以先说‘我考虑一下’”
- **承认边界**：不是所有情况都适用，给观众留判断空间
- **行动前加设问**：“那你可以怎么做呢？”“落到生活里，其实就一件事……”
- **用"先试试"降低压力**：不是要求观众立刻改变，而是给一个小动作

## 五、识 · 主题升华

### 方法

- 把前面的讨论提升到更大的格局
- 升华方向四选一或多层递进：**个人成长 → 家庭和睦 → 社会进步 → 国家富强**
- 升华不是喊口号，是用前面的逻辑自然推导出的结论

### 组织要点

| 层次 | 核心问题 | 示例 |
|---|---|---|
| 个人成长 | 这件事对我意味着什么 | 学会课题分离，不是让你冷漠，是让你少内耗 |
| 家庭和睦 | 这件事对家庭关系有什么启发 | 很多家庭矛盾，其实都是因为边界不清 |
| 社会进步 | 这件事反映了什么社会现象 | 一个越讲边界的社会，越能让人安心合作 |
| 国家富强 | 这件事和国家发展有什么联系 | 每个人都少一点无谓的情绪内耗，社会的运行成本就会低很多 |

### 朴素之道风格注入

- **结尾给认知升级或反问**：不是强行号召，而是引发共鸣
- **常用收尾句式**：“你说是不是这个理？”“这个观点，你认同吗？”“说到底，……”
- **避免大词堆砌**：升华可以用大白话完成
- **升华前加"说到底"或"所以"**：让观众知道你要总结了
- **最后一句抛给观众**：用反问收尾，而不是自己下结论

## 整体节奏与衔接

### 推荐时长结构（以3分钟口播稿为例）

| 蕴 | 时长 | 字数参考 |
|---|---|---|
| 色 | 30-40秒 | 80-120字 |
| 受 | 20-30秒 | 60-90字 |
| 想 | 50-70秒 | 130-180字 |
| 行 | 30-40秒 | 80-120字 |
| 识 | 30-40秒 | 80-120字 |

### 段间衔接词库

- 色 → 受：说实话，我当时…… / 你可能也有这种感觉……
- 受 → 想：后来我想明白一件事…… / 这让我想起一本书里说的……
- 想 → 行：那落到生活里，我们可以怎么做？
- 行 → 识：说白了，这不只是一个人的事……

### 口播节奏控制

- **每30秒一个呼吸点**：要么停顿，要么换段，要么抛出一个问题
- **关键句前后留空**：重要结论前停顿0.5秒，结论后再停0.5秒
- **不要连续两个长句**：长句后必须接一个短句或问句
- **数字和比喻要慢下来**：观众需要反应时间
- **结尾必须收住**：最后一句不要太长，留给观众回味

## 风格约束（与朴素之道表达DNA对齐）

### 必须遵守

1. **短句为主**：每句尽量不超过15字（口播比书面更短）
2. **口语转折优先**：用“说白了”“说到底”“其实不是……而是……”
3. **具体数字替代模糊表述**：用“月薪五千、房租三千”而不是“生活成本高”
4. **禁用说教腔**：不用“我们应该”“必须认识到”“由此可见”
5. **结尾用反问或认知升级**：不用“点赞关注”“大家一定要”
6. **先承认复杂性，再给出判断**：不武断
7. **每段开头用"你"或"我"开场**：保持对话感
8. **关键信息前加停顿**：给观众反应时间

### 必须避免

| 反模式 | 示例 | 替代 |
|---|---|---|
| 抽象开场 | 今天我们来聊聊人际关系 | 你有没有遇到过这种人：每次都迟到，你还不好意思说他 |
| 堆砌术语 | 阿德勒心理学强调课题分离 | 说白了，就是你的事归你，我的事归我 |
| 空喊口号 | 我们要学会拒绝 | 下次你可以先说“我考虑一下” |
| 强行升华 | 所以我们要为实现中华民族伟大复兴而努力 | 每个人少一点内耗，社会的运行成本就低一点 |
| 连续长句 | 台风是热带气旋在特定海温条件下通过水汽凝结释放潜热形成并维持的低压系统 | 台风怎么来的？/海水烧热，水汽往上冒，/空气转圈，/越转越快。 |
| 缺少停顿 | 水烧开了中间往上冒周围往下沉形成一个旋转的圈 | 你看，水烧开了。/中间往上冒，周围往下沉，/转成一个圈。 |
| 书面连接词 | 因此、由此可见、综上所述 | 所以、说白了、说到底 |

## Few-shot 示例

### 主题：为什么越忍让，越被当成好欺负

**色**：
你有没有遇到过这种人？/每次聚会都迟到半小时，/大家笑着说“没事没事”，/但心里其实都不舒服。//我朋友小李就这样，/五年了，没人真正跟他翻过脸。

**受**：
说实话，/我以前也觉得，/忍一忍显得自己有格局。//可后来发现，/那个总迟到的人，/根本不会因为你包容他就改。//他只是默认：/你好说话，/所以可以一直消耗你。

**想**：
这让我想起一本书，/叫《被讨厌的勇气》。//里面有个说法，/叫“课题分离”。//说白了，/就是你的事归你，/我的事归我。//他迟到是他的事，/我表达不满是我的事。//我们把这两个事搅在一起，/才会又委屈，/又不敢出声。

**行**：
那落到生活里，/你可以怎么做呢？//先做一件小事：/把底线说清楚。//比如，“我等二十分钟，/到点先吃”。//不是绝情，/是让别人知道，/你的边界在哪。

**识**：
说到底，/学会课题分离，/不是让你变得冷漠，/是让你少内耗。//一个边界清晰的人，/反而更容易交到真朋友。//你说，是不是这个理？【停顿】

### 主题：消费降级，其实是消费理念的升级

**色**：
iPhone15出了，我没换。身边朋友问我：你怎么不换手机？我说还能用。他愣了一下，好像我不换手机是件很奇怪的事。

**受**：
前几年我不是这样。出新手机必买，新款球鞋必抢，好像拥有了这些东西，生活就更有质感。可后来我发现，那些东西带来的快乐，撑不过三天。

**想**：
《稀缺》这本书里有个说法：人穷的时候，会把注意力全部集中在眼前稀缺的东西上，反而做出很多长期并不划算的决定。放到消费上就是，我们越想在东西上找存在感，越容易买一堆没用的。

**行**：
我现在买东西前先问自己一个问题：没有它，我的生活会差多少？如果答案是不会差，那就不买。这个习惯让我一年少花了不少钱，但生活质量反而高了。

**识**：
说到底，消费降级不是因为我们变穷了，是我们终于开始把钱花在真正重要的地方。这对个人是清醒，对家庭是节流，对社会也是一种更可持续的消费观。你认同吗？

## 使用说明

1. 先确定一个**主题**和**目标观众**
2. 按五蕴顺序填空，每一蕴只回答该蕴的核心问题
3. **先生成纯净朗读版**：自然段落流，不显示五蕴标记
4. **再生成录制标记版**：加入 `/` `//` `【重】` `【停顿】` 等标记，用于实际录制
5. **人声化改写**：对照 checklist 逐句检查停顿、语气、长句、对话感
6. 如需使用朴素之道风格，同时调用 `workflow/distillation/output/pusuzhidao.skill/SKILL.md`
