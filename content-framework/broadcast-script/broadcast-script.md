# 五蕴素材整合框架

> 给你一个话题，快速拆出一组**可选素材**。
> 不生成口播成稿，只帮你把最切中要害的经历、感受、认知、行动、升华整合到一张清单里，省掉到处找素材的时间。

---

## 五蕴骨架

```
色（现象切入）
  → 受（情感共鸣）
    → 想（认知映射）
      → 行（行动落点）
        → 识（主题升华）
```

| 蕴 | 素材类型 | 核心问题 | 时间占比 |
|---|---|---|---|
| 色 | 经历、故事、问题、现象 | 这件事最普遍、最具体的入口是什么？ | 20% |
| 受 | 真实感受 | 自己或大家对此的真实情绪是什么？ | 15% |
| 想 | 类似经历、故事、理论、观点 | 什么经历或认知模型能解释这个现象？ | 25% |
| 行 | 可执行的认知行动或思考方向 | 落到生活里，可以怎么做、怎么问？ | 20% |
| 识 | 主题升华 | 这件事对个人、家庭、社会意味着什么？ | 20% |

---

## 使用流程

1. **定话题**：写下你要讨论的核心问题或主题。
2. **逐蕴填空**：按色、受、想、行、识的顺序，在对应表格里填入素材。
3. **做选择**：每一蕴只选最有画面感、最有共鸣、最能支撑观点的 1-2 条。
4. **串起来**：把选出的素材按五蕴顺序排列，形成一条从现象到认知的完整叙述链。
5. **optional**：如果需要成稿，再在这个骨架上做口播化或文章化改写。

## 一、色 · 现象切入

### 这蕴做什么
给出一个**具体、可观察、有画面感**的入口，让观众立刻觉得“这我熟”。

### 核心问题
这个话题最常见的现象、经历或矛盾是什么？

### 清单式提问

- 观众最可能在什么场景里遇到这件事？
- 有没有一个具体的人、具体时间、具体地点的画面？
- 表面看起来正常，但底下有问题的反差是什么？
- 这件事有没有一个让人印象深刻的数字或细节？
- 你身边最近有谁经历过类似的事？
- 如果要用一句话把场景和问题同时抛出来，怎么说？
- 这个现象的普遍性有多强？是少数人才有，还是大家都有？
- 有没有一个反常识的观察？（比如“越……反而越……”）

### 示例：为什么越忍让，越被当成好欺负

| 数据/细节 | “我认识的五个人里，三个都这样” |

## 二、受 · 情感共鸣

### 这蕴做什么
说出**真实的感受**，建立“我懂你”的连接。感受可以是自己的，也可以是身边人的。

### 核心问题
这件事带来的真实情绪是什么？

### 清单式提问

- 我第一次遇到这件事时，真实感受是什么？
- 身边人对这件事最常见的反应是什么？
- 这种感受里有没有矛盾？（比如既生气又不好意思说）
- 有没有一个“我以前也这样”的瞬间？
- 这种感受被忽视过吗？被压抑过吗？
- 如果直接说出感受，会不会太说教？怎么表达更自然？
- 这种感受有没有普遍性？多少人会共鸣？
- 感受背后藏着什么未被满足的期待？

### 素材库模板

| 感受来源 | 具体感受 | 是否选用 | 理由 |
|---|---|---|---|
| 自己的感受 | | | |
| 身边人的感受 | | | |
| 群体普遍感受 | | | |
| 感受中的矛盾 | | | |

### 示例：为什么越忍让，越被当成好欺负

| 感受来源 | 具体感受 |
|---|---|
| 自己的感受 | 我以前也觉得，忍一忍显得自己有格局 |
| 身边人的感受 | 那个总迟到的人，根本不会因为你包容他就改 |
| 群体普遍感受 | 很多人不是不生气，是不好意思生气 |
| 感受中的矛盾 | 越忍让越委屈，越委屈越不敢说 |

## 三、想 · 认知映射

### 这蕴做什么
用**相似经历、故事、理论或观点**把现象“点亮”，让观众从“我也有这感觉”变成“原来是这样”。

### 核心问题
什么经历、故事或认知模型能解释前面的现象？

### 清单式提问

- 我自己有没有类似的经历，可以作为类比？
- 身边人有没有讲过类似的故事？
- 哪本书、哪个理论、哪个观点能解释这个现象？
- 这个现象的本质是什么？能不能用一句话概括？
- 有没有一个简单模型可以把复杂现象讲清楚？
- 能不能用一个生活化的比喻来说明？
- 这个认知模型有哪些边界？不能解释什么？
- 反过来看，如果没有这个认知，我们会怎么误判？

### 素材库模板

| 认知来源 | 具体内容 | 是否能解释现象 | 是否选用 | 理由 |
|---|---|---|---|---|
| 自己的类似经历 | | | | |
| 他人的故事 | | | | |
| 书名/理论/观点 | | | | |
| 比喻/模型 | | | | |

### 示例：为什么越忍让，越被当成好欺负

| 认知来源 | 具体内容 |
|---|---|
| 书名/理论/观点 | 《被讨厌的勇气》里的“课题分离”：你的事归你，我的事归我 |
| 比喻/模型 | 把人际关系分成两类：课题分离 vs 课题混淆 |
| 自己的类似经历 | 我以前也把“表达不满”和“破坏关系”混为一谈 |
| 他人的故事 | 一个朋友学会了课题分离后，反而交到了更真的朋友 |

## 四、行 · 行动落点

### 这蕴做什么
给出一个**小到能立刻试**的认知行动或思考方向，不喊口号。

### 核心问题
落到生活里，可以怎么做、怎么问、怎么改变视角？

### 清单式提问

- 下次再遇到这种情况，可以先问自己一个什么问题？
- 有没有一个具体的小动作，而不是抽象的口号？
- 这个行动需要多长时间？能不能在 5 分钟内完成？
- 行动的边界在哪里？哪些情况不适用？
- 有没有一个话术或表达方式可以直接用？
- 这个行动如何降低心理压力？（不是“必须改”，而是“先试试”）
- 怎么判断这个行动有效？有没有反馈信号？
- 如果失败，最可能的原因是什么？

### 素材库模板

| 行动类型 | 具体行动 | 是否选用 | 理由 |
|---|---|---|---|
| 提问式 | | | |
| 场景式 | | | |
| 话术式 | | | |
| 认知式 | | | |

### 示例：为什么越忍让，越被当成好欺负

| 行动类型 | 具体行动 |
|---|---|
| 提问式 | 别人再让你不舒服时，先问自己：这是谁的课题？ |
| 场景式 | 聚会迟到这件事，提前说清楚：“我等二十分钟，到点先吃” |
| 话术式 | 不说“你必须准时”，而说“我会等你二十分钟” |
| 认知式 | 别把忍让当格局，那可能只是不敢表达 |

## 五、识 · 主题升华

### 这蕴做什么
把讨论提升到更大格局，让主题有重量。方向可选：**个人成长、家庭和睦、社会进步、国家富强**，也可以多层递进。

### 核心问题
这件事对个人、家庭、社会意味着什么？

### 清单式提问

- 这件事对我个人成长有什么启发？
- 对家庭关系有什么启发？（边界、沟通、期待）
- 它反映了什么社会现象或趋势？
- 它和更大的时代议题有什么关系？
- 升华是不是前面逻辑的自然推导？还是硬拔高？
- 能不能用大白话说出升华，而不是堆大词？
- 结尾是抛给观众一个问题，还是给一个认知升级？
- 有没有一个“说到底……”的总结句式？

### 素材库模板

| 升华层次 | 核心问题 | 可选素材 | 是否选用 | 理由 |
|---|---|---|---|---|
| 个人成长 | 这件事对我意味着什么？ | | | |
| 家庭和睦 | 对家庭关系有什么启发？ | | | |
| 社会进步 | 反映了什么社会现象？ | | | |
| 国家富强 | 和国家发展有什么联系？ | | | |

### 示例：为什么越忍让，越被当成好欺负

| 升华层次 | 可选素材 |
|---|---|
| 个人成长 | 学会课题分离，不是让你冷漠，是让你少内耗 |
| 家庭和睦 | 很多家庭矛盾，其实都是因为边界不清 |
| 社会进步 | 一个越讲边界的社会，越能让人安心合作 |
| 国家富强 | 每个人都少一点无谓的情绪内耗，社会的运行成本就会低很多 |

## 素材串联示例：为什么越忍让，越被当成好欺负

从每一蕴里选出一条素材，按顺序串起来，就形成一条从现象到认知的完整叙述链：

**色**：聚会总有人迟到，大家笑着说“没事”，但心里不舒服。朋友小李就这样，五年了没人真正跟他翻过脸。

**受**：我以前也觉得，忍一忍显得自己有格局。可后来发现，那个总迟到的人根本不会因为你包容他就改。他只是默认：你好说话，所以可以一直消耗你。

**想**：《被讨厌的勇气》里有个说法叫“课题分离”。说白了，就是你的事归你，我的事归我。他迟到是他的事，我表达不满是我的事。把这两件事搅在一起，才会又委屈又不敢出声。

**行**：下次遇到这种情况，先做一件小事：把底线说清楚。比如，“我等二十分钟，到点先吃”。不是绝情，是让别人知道你的边界在哪。

**识**：说到底，学会课题分离不是让你变得冷漠，是让你少内耗。一个边界清晰的人，反而更容易交到真朋友。

## 最后提醒

- 这个框架的价值不是生成完美稿件，而是**帮你把散落的素材快速归位**。
- 每一蕴可以多写几个候选，但最后用的时候一定要做减法。
- 五蕴顺序固定，但每一蕴内部的素材由你自己挑选和组合。
