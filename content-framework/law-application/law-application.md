# 法律适用框架

8个核心原子方法按顺序推进，方法固定，执行灵活。

```
事实裁剪（fact-trimming）
  │ 方法：整理事实，裁剪与法律无关的情节
  │ 执行：裁剪粒度、事实组织方式 → 按案情决定
  ↓
法律关系定性（legal-relationship-typing）
  │ 方法：判断法律关系领域
  │ 执行：定性深度、领域细分 → 按案情决定
  ↓
规范检索（norm-retrieval）
  │ 方法：检索适用法条
  │ 执行：检索来源、检索范围 → 按可用工具决定
  ↓
涵摄（subsumption）
  │ 方法：判断要件是否满足
  │ 执行：要件拆解方式、论证详略 → 按案情决定
  ↓
冲突裁决（conflict-resolution）
  │ 方法：解决规范/证据/争点冲突
  │ 执行：冲突处理路径、裁决依据 → 按冲突类型决定
  ↓
法律解释（legal-interpretation）
  │ 方法：解释模糊规范
  │ 执行：解释方法选择（文义/目的/体系/历史）→ 按模糊类型决定
  ↓
抗辩与效力检视（defense-and-validity-review）
  │ 方法：审查抗辩与效力
  │ 执行：审查范围、检视深度 → 按案情决定
  ↓
兜底适用（fallback-application）
  │ 方法：穷尽具体规则后兜底
  │ 执行：兜底依据、适用条件 → 按案情决定
```

**硬性约束**：8个方法是法律适用的骨架，缺一不可；具体执行按案情灵活决定。
