# 连载小说写作框架

六步推进——从类型定位到单章写作，方法固定，执行灵活：

```
类型定位（定调）
  │ 方法：确定小说类型（知识/行业/历史/科幻/现实）+ 回答定位三问
  │       （读者是谁·核心冲突是什么·每期带走什么）
  │ 执行：类型混合比例、定位侧重 → 按创作意图决定
  ↓
世界观构建（立世）
  │ 方法：三阶思考——是什么（时代/地点/规则/技术）→
  │       为什么（核心议题/普遍冲突/读者连接点）→
  │       怎么做（开场场景/代入感/信息释放节奏）
  │ 执行：世界观详略、设定文档形式 → 按类型决定
  ↓
人物设定（塑人）
  │ 方法：主角（身份/欲望/恐惧/核心冲突/成长弧线）+
  │       反派（动机/对立点/复杂性）+ 配角（功能/关系/关键场景）
  │ 执行：人物数量、配角详略 → 按故事规模决定
  ↓
故事主线（织线）
  │ 方法：三幕式骨架——建置25%（介绍世界/人物/日常，触发核心冲突）→
  │       对抗50%（主角面对障碍，展开主要情节）→
  │       结局25%（高潮/解决/人物变化）
  │ 执行：幕的比例、冲突类型组合 → 按故事节奏决定
  ↓
分季分章（切章）
  │ 方法：一季对应一个完整故事弧线（8-20章），季末阶段性结局+下季钩子
  │       每章大纲含：编号/标题/场景/人物/冲突/推进/钩子/字数目标
  │ 执行：季的长度、章的切分粒度 → 按连载节奏决定
  ↓
单章写作（成章）
  │ 方法：思考层走认知层三阶（每章一个本质定义+顿悟触发三件套），
  │       输出层按故事节奏（场景切入→冲突升级→人物选择→转折→钩子收尾）
  │ 执行：开场方式（动作/对话/场景/独白）、章节篇幅 → 按故事节奏决定
```

**思考层 vs 输出层分离**：每章本质定义和顿悟触发器是思考层脚手架（写作前思考必走），章节外形由故事节奏决定，不强制章节开头抛本质，也不强制按三阶分节。

**硬性约束**：每章一个本质定义（核心概念内核）；顿悟触发三件套（一句话点破+反常识根因+视角升级）每章必备，位置由故事节奏决定；每章结尾必须有钩子。
