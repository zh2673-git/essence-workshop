**议论文写作全流程框架**


## 一、标题

**立意维度：** 个人 → 社会 → 国家

**吸引人三法：**
- **数字**：三次选择，一生追问
- **反差**：最快的慢功夫
- **动词**：让传统活起来


## 二、开头

| 步骤 | 名称 | 任务 |
| :---: | :--- | :--- |
| ① | 引入 | 现象/故事/引用，吸引注意 |
| ② | 界定"是什么" | 排比+反差+比喻，为核心词画像 |
| ③ | 蓄势"为什么" | 一句话点明重要性或紧迫性 |
| ④ | 亮剑"怎么做" | 给出中心论点（立场/方向） |

**禁止项：**
- 不要用"不是...而是..."这种明显的AI味表述，改用更自然的表达方式。
- **语气自检规则**：如果文章主题是反对某种行为（如反对说教），文章本身不能犯同样的错误。例如，反对说教的文章不能采用居高临下的论断式语气，而应该用故事化、反思式、对话式表达，让读者自己得出结论而非被告知结论——这是文章主题的自我践行。

**框架标记规则**：框架中的编号（如01/02/03）、标签（如【引】【议】【联】【结】）、结构名称（如"开局""结尾""中间"）仅为写作过程的内部指引，**最终成品中必须全部移除**。读者不应看到任何框架脚手架痕迹。成品应呈现为自然段落流，引议联结的逻辑隐含在行文中，而非显式标注。


## 三、中间

**第一步：分论点框架**

| 类型 | 逻辑 | 适用 |
| :---: | :--- | :--- |
| 并列式 | A＝B＝C，同层不同面 | 美德、品质、作用 |
| 递进式 | A→B→C，由浅入深 | 哲理、思辨、成长 |
| 对照式 | A vs B，正反对比 | 争议性话题 |

**第二步：每段"引·议·联·结"**

| 环节 | 任务 | 占比 |
| :---: | :--- | :---: |
| 引 | 引述论据（事实/名言/数据），概括不展开 | 1/4 |
| 议 | 分析议论，回答"为什么能证明分论点" | 重点 |
| 联 | 联系主旨/现实，从事例到普遍规律 | — |
| 结 | 回扣分论点关键词，闭合段落 | — |

**第三步：段间衔接**

递进：上段铺垫"底色"，下段以"如果说……是底色，那么……则是锋芒"承接
对照：正段末埋伏笔，反段首用"然而""与之相对"转折


## 四、结尾

| 步骤 | 任务 |
| :---: | :--- |
| ① | 总结全文，重申论点 |
| ② | 呼应开头（照应非重复） |
| ③ | 留有余响（设问/意象/格局拉升） |

**三种方法：** 首尾呼应式 / 总结排比式 / 呼吁展望式

**进阶点睛：** 巧用设问 / 意象落笔 / 格局拉升（个人→社会→文明/时代）

**字数：** 100字以内，不节外生枝


## 五、Few-shot 示例

### 【标题】
从"倍速"到"留白"——慢下来的力量（反差）

### 【开头】
①引入：倍速追剧、速读名著、三天学会编程，"快"正成为时代通行证。
②是什么：真正的成长，是思想反刍的消化力，是潜入深水的静默生长——信息填鸭的饱腹感，浮光掠影的到此一游，终将随风消散。
③为什么：当大脑被碎片信息塞满，灵魂反而因饥饿而枯萎。
④怎么做：因此，在狂奔的时代学会"慢下来"，才是智者真正的加速。

### 【中间】
**分论点框架：递进式**
分论点一：慢下来，是对信息过载的清醒抵抗（个人层面，引·议·联·结）
分论点二：慢下来，是从量变到质变的必要酝酿（社会层面，引·议·联·结）
分论点三：慢下来，是文明积淀的深层逻辑（国家/历史层面，引·议·联·结）

**段间衔接：**
段一结尾：这是个体觉醒的第一步。
段二开头：当个体选择慢下来，一种更深刻的变化正在社会肌理中悄然发生。
段二结尾：量的积累，终将等待质的跳跃。
段三开头：而这种从个体到社会的"慢"，放眼文明的长河，正是文化得以沉淀、精神得以扎根的根本逻辑。

**单段范本（分论点一）：**
【引】德国学者罗尔夫·多贝里在《清醒思考的艺术》中提醒我们：信息摄入越多，判断力反而可能越弱。
【议】因为大脑处理信息的带宽是有限的，无节制的快节奏输入，本质上是一种认知过载，它挤占了反思与消化的空间。
【联】在这个每秒都有无数信息争抢眼球的时代，选择性"慢下来"，是为判断力留出呼吸的间隙，看似效率低下，实则智慧之举。
【结】抵抗过载，从为大脑留白开始。

### 【结尾】
①总结排比收束：慢下来，是个体在信息洪流中打捞清醒的定力；慢下来，是社会从野蛮生长转向精耕细作的智慧；慢下来，更是一个文明拒绝速朽、沉淀经典的自觉。
②呼应开头：从"倍速"到"留白"，我们终将领悟——
③意象收束：河流最快之处泥沙俱下，而深潭静水，才照见天光云影。
