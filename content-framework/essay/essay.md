# 高考作文框架

五步法来自高考满分作文，方法是固定的，但具体执行灵活：

```
题目（定火）
  │ 方法：找到核心张力 + 写出可证伪的判断句
  │ 执行：格式、长度、编排 → 模型自行决定
  ↓
结构（垒山）
  │ 方法：开头（点题+悬念）→ 主体（2-3层次，每层四环推进）→ 结尾（视角微升）
  │ 执行：层次划分、篇幅分配 → 模型自行决定
  ↓
选材（添薪）
  │ 方法：优先有解读空间的素材 + 每个素材三种用法
  │ 执行：素材选择、呈现方式 → 模型自行决定
  ↓
炼句（显形）
  │ 方法：高光打磨关键句 + 控制修辞密度
  │ 执行：句式、节奏、金句位置 → 模型自行决定
  ↓
复盘（判读）
  │ 方法：以读者视角重读 + 亮点测试
  │ 执行：自检方式、修改决策 → 模型自行决定
```

### 题目（定火）：找到核心张力 + 写出判断句

**方法固定**

- 找到核心张力：矛盾、转折、追问
- 写出可证伪的判断句：一句话定调
- 点题+悬念：让读者知道"讲什么"+点燃继续读的欲望

**执行灵活**

- 格式、长度、编排按需要决定

### 结构（垒山）：三部分骨架

**方法固定**

- 开头：点题+悬念/反差
- 主体：2-3个层次，每层按「论点句→阐释→素材→回扣」四环推进
- 结尾：回到题目+视角微升

**执行灵活**

- 每个层次的篇幅、编排按需要决定

### 选材（添薪）：优先有解读空间的素材

**方法固定**

- 优先有解读空间的素材：经典、个人经历、边缘案例
- 每个素材三种用法：正面论证、反面衬托、类比迁移
- 案例必须具体：有场景、有动作、有转折

**执行灵活**

- 素材选择、呈现方式按认知产物和实际需要决定

### 炼句（显形）：高光打磨关键句

**方法固定**

- 高光打磨关键句：开头第一句、每层论点句、结尾句
- 控制修辞密度：比喻、排比集中在亮点处，避免通篇燃烧

**执行灵活**

- 句式、节奏、金句位置按风格和实际需要决定

### 复盘（判读）：以读者视角重读

**方法固定**

- 以读者视角重读：开头测试、转折测试、亮点测试
- 回溯检验：每步能否回溯到本质定义

**执行灵活**

- 自检方式、修改决策按实际情况决定
