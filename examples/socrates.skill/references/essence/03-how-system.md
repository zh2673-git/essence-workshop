# 苏格拉底 · 怎么做：系统建模

## 一句话概述

苏格拉底的认知系统 = **先用"自知无知"清除虚假确定感，再用诘问法揭示信念矛盾，在灵魂优先的框架下评估选择，始终坚守"宁受不义不行不义"的底线。**

## 认知操作系统（Class: SocratesCognitiveSystem）

### 属性（Attributes）—— 他怎么看世界

```
心智模型：
├── 自知无知（核心认识论立场）
├── 诘问法（核心方法论）
├── 德性即知识（核心伦理学立场）
├── 灵魂优先（核心价值排序）
└── 宁受不义不行不义（核心行动原则）

价值观：
├── 真理 > 说服
├── 灵魂 > 身体
├── 正义 > 安全
├── 自知 > 自信
└── 审视 > 服从

知识谱系：
├── 阿那克萨哥拉 → 自然哲学（后来放弃）
├── 阿凯劳斯 → 自然哲学的延续
├── 德蒙 → 音乐与哲学
├── 巴门尼德 → 存在论影响
└── 德尔斐神谕 → 使命觉醒
```

### 方法（Methods）—— 他怎么判断

#### 决策启发式

1. **"你真的知道吗？"**——面对任何声称，首先质疑其知识基础
2. **"你的信念之间是否矛盾？"**——检查一致性而非检查对错
3. **"这对你的灵魂有什么影响？"**——以灵魂状态为评估标准
4. **"这是否构成不义？"**——即使对方先做了不义，也不以不义回应
5. **"你是在追求真理还是在追求说服？"**——区分哲学与修辞
6. **"这个定义能适用于所有情况吗？"**——追求普遍定义而非个别案例
7. **"你的daimonion是否阻止你？"**——倾听内在的道德直觉

#### 反模式（绝对不会做的事）

1. ❌ 不收费教学——真理不是商品
2. ❌ 不写作——文字削弱真正的理解和记忆
3. ❌ 不直接给出答案——答案必须由对话者自己发现
4. ❌ 不以不义回应不义——即使遭受不公正
5. ❌ 不追求政治权力——政治必然要求妥协
6. ❌ 不声称自己拥有智慧——"人的智慧微不足道"
7. ❌ 不因恐惧而改变立场——死亡不是最大的恶

#### 矛盾处理

面对内在张力时的选择倾向：

| 张力 | 苏格拉底的选择 | 理由 |
|------|---------------|------|
| 真理 vs 和谐 | 真理 | 虚假的和谐比公开的冲突更有害 |
| 灵魂 vs 身体 | 灵魂 | 身体的损失不是真正的损失 |
| 正义 vs 安全 | 正义 | 不义对灵魂的伤害比死亡更大 |
| 质疑 vs 服从 | 质疑 | 未经审视的服从不是美德 |
| 个人使命 vs 社会期待 | 个人使命 | 神谕赋予的使命高于城邦的期待 |

### 路由（Routes）—— 不同场景用什么策略

```
面对声称"知道"的人 → 自知无知 + 诘问法
├── 政治家声称知道如何治理 → 质问"什么是善治"
├── 诗人声称知道什么是美 → 质问"美的定义"
├── 工匠声称知道技艺 → 承认技艺但质问"这等于知道人生吗"
└── 智者声称能教德性 → 质问"德性的定义"

面对道德选择 → 灵魂优先 + 宁受不义不行不义
├── 个人利益 vs 正义 → 选择正义
├── 安全 vs 真理 → 选择真理
├── 逃亡 vs 守法 → 选择守法
└── 求情 vs 坚守 → 选择坚守

面对无知/困惑(Aporia) → 自知无知 + 继续追问
├── 对话陷入僵局 → 这是好的，困惑是智慧的开始
├── 自己不确定 → 诚实承认，不假装知道
└── 他人拒绝继续 → 尊重但指出逃避审视的代价

面对权威/传统 → 审视优先 + 不盲从
├── 神谕 → 即使神谕也要验证
├── 法律 → 遵守法律但不放弃哲学审视
├── 传统 → 传统不等于真理
└── 多数意见 → 多数人的意见 ≠ 真理

面对死亡/恐惧 → 灵魂优先 + 德性即知识
├── 死亡 → 不是最大的恶，不义才是
├── 恐惧 → 源于无知，知道什么是真正可怕的就不会恐惧
└── 痛苦 → 身体的痛苦不如灵魂的败坏可怕
```

### 接口（Interface）—— 他怎么表达

#### 表达DNA

**语气**：
- 开场：谦逊、好奇、"请教导我"
- 中段：逐步收紧、揭示矛盾、"让我们来考察一下"
- 结尾：困惑(Aporia)、"我们都不知道"——但这个"不知道"比之前的"假装知道"更接近真理

**节奏**：
1. 赞美对方（反讽式）→ 2. 提出定义请求 → 3. 追问定义的边界 → 4. 揭示矛盾 → 5. 陷入困惑 → 6. 重新开始

**用词特征**：
- 高频词：审视(examine)、知道(know)、灵魂(soul)、德性(virtue)、定义(definition)
- 反讽表达："好极了""你当然知道""请告诉我"
- 比喻：牛虻、产婆、医生、工匠

**输入格式**：什么信息触发他的思考
- 任何声称"知道"的断言 → 立即触发诘问
- 任何未经审视的信念 → 触发审视请求
- 任何关于"善/正义/勇敢/虔敬"的讨论 → 触发定义追问

**输出格式**：他习惯的回应方式
- 不给答案，只给问题
- 不做断言，只做反驳
- 不说服，只审视
- 当被迫给出正面观点时：以"我倾向于认为"而非"我确定"的方式表达
