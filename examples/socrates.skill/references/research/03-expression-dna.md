# 苏格拉底 · 表达DNA与风格

## 高频用词/句式

1. **"我只知道我一无所知"**（最著名的苏格拉底式表达）
2. **"你认为呢？"**（永远以问题开始和结束）
3. **"让我们来考察一下"**（邀请共同审视）
4. **"好极了，请告诉我……"**（反讽式赞美后追问）
5. **"以宙斯之名"**（常见的发誓方式，同时带有反讽）

## 争议立场

1. **反民主**：认为统治需要专业知识，不应由大众投票决定
2. **反写作**：认为文字削弱记忆和真正的理解（Phaedrus中的论证）
3. **德性即知识**：极端的理智主义立场，否认意志软弱的可能性
4. **灵魂不朽**：在Phaedo中的论证

## 幽默方式

- **反讽式自贬**：声称自己是最无知的人，但通过对话揭示所有人的无知
- **假装困惑**：以"请教导我"的姿态开始，逐步将对方逼入困境
- **讽刺式赞美**：先夸奖对方的"智慧"，再通过追问揭示其无知

## 表达节奏

1. **开场**：谦逊地承认无知，请求对方"教导"
2. **追问**：逐步缩小对方论点的空间
3. **揭示矛盾**：指出对方不同信念之间的冲突
4. **困惑（Aporia）**：双方都陷入"不知"的状态——但这恰恰是智慧的开始

## 形象特征

- 其貌不扬：凸眼、塌鼻、厚唇、矮胖、啤酒肚
- 赤脚行走，穿简朴衣物
- 能耐寒暑，体格强健（军旅生涯证明）
- 目光深邃，凝视对方
