# 荀子 · 著作与系统思考

## 核心著作体系

《荀子》32篇，经刘向校定，唐杨倞作注。其中22篇为荀子亲笔，5篇为弟子记录言行，5篇为引用材料。

### 核心篇章与论点

| 篇章 | 核心论点 | 反复出现频率 |
|------|---------|------------|
| 《性恶》 | 人之性恶，其善者伪也 | 全书反复论证≥10次 |
| 《礼论》 | 礼者，治辨之极也 | 全书反复论证≥8次 |
| 《天论》 | 天行有常，制天命而用之 | 全书反复论证≥5次 |
| 《劝学》 | 学不可以已，青取之于蓝 | 全书反复论证≥6次 |
| 《解蔽》 | 蔽于一曲而闇于大理 | 全书反复论证≥4次 |
| 《正名》 | 制名以指实，约定俗成 | 全书反复论证≥4次 |
| 《王制》 | 隆礼尊贤而王，重法爱民而霸 | 全书反复论证≥5次 |
| 《富国》 | 强本而节用，开源节流 | 全书反复论证≥3次 |

### 自创术语体系

| 术语 | 含义 | 出处 |
|------|------|------|
| 性伪之分 | 天性（性）与人为（伪）的根本区分 | 《性恶》 |
| 化性起伪 | 通过后天教化改造先天本性 | 《性恶》 |
| 天人相分 | 天道自然与人道治理的分离 | 《天论》 |
| 制天命而用之 | 掌握自然规律加以利用 | 《天论》 |
| 虚壹而静 | 心的认知方法：虚心、专一、沉静 | 《解蔽》 |
| 明分使群 | 明确分工使社会群体有序 | 《富国》《王制》 |
| 积善成德 | 通过持续积累善行形成德性 | 《劝学》 |
| 隆礼重法 | 以礼为根本、以法为辅助 | 《王制》《君道》 |

### 核心逻辑链

```
性恶（起点）→ 顺性则乱 → 必须约束 → 礼义教化（化性起伪）→ 积善成德 → 圣人境界
                                    ↓
                              法度刑罚（辅助约束）→ 隆礼重法 → 社会有序
```

## 信息来源与可信度

- 一手来源：《荀子》原文32篇（最高可信度）
- 二手来源：《史记·孟子荀卿列传》、刘向《孙卿书录》
- 学术研究：郭沫若《荀子的批判》、冯友兰《中国哲学史》、王先谦《荀子集解》
