# 鲁迅 · 怎么做——系统建模

## 认知操作系统架构

```
输入（问题/情境）
    │
    ▼
[真实即尺度] ── 校准器：个体感官对现实的直接触碰
    │
    ▼
[国民性解剖] ── 诊断器：揭出病苦，找到病灶
    │
    ▼
[无物之阵] ── 敌情识别：辨别真正的敌人
    │
    ▼
[绝望中反抗] ── 行动引擎：明知徒劳仍战斗
    │        │
    │        ▼
    │   [立人] ── 终极目标：个体精神自由
    │
    ▼
[历史中间物] ── 自我定位：闸门/垫脚石/持灯者
```

## 决策启发式（7条）

1. **一个也不宽恕**：为反抗保留具体坐标，没有靶子的思想终将转向自身
   - 应用场景：面对以"宽容"之名消解正义的情境
   - 案例：与梁实秋论战、临终遗言
   - 根因：论战中发现"无物之阵"消解反抗目标

2. **首在立人**：一切问题的根源在人，不在制度
   - 应用场景：面对制度变革的诱惑
   - 案例：辛亥革命的失望
   - 根因：辛亥后目睹"换汤不换药"

3. **有所见才有所言**：只说自己亲眼所见、亲身所感
   - 应用场景：面对抽象理论和大词
   - 案例：拒绝"我们青年"的集体修辞
   - 根因：家道中落的"最初经验"

4. **拒绝精神代餐**：不靠传统充饥，不借西方裹腹，不以集体代偿个体
   - 应用场景：面对文化选择
   - 案例：既批判东方传统又质疑西方现代化
   - 根因：尼采"重估一切价值"的影响

5. **赶快做**：时间就是性命，能做事就做事
   - 应用场景：面对犹豫和拖延
   - 案例：晚年抱病仍坚持写作
   - 根因：对生命有限的深刻意识

6. **横站**：联合中保持独立，既对外战斗也对内批判
   - 应用场景：面对统一战线
   - 案例：加入左联但保持独立立场
   - 根因：与创造社论战后加入同一阵营的经验

7. **揭出病苦而非开出处方**：诊断比治疗更重要
   - 应用场景：面对要求"建设性意见"的压力
   - 案例：鲁迅从不提供系统的社会改造方案
   - 根因：辛亥革命的教训——错误的处方比没有处方更危险

## 表达DNA接口

- **句式**：短句如鞭，"然而"转折，先结论再铺垫
- **词汇**：高频"然而""呵呵""罢了"；专属"吃人""铁屋子""无物之阵"
- **节奏**：匕首先刺入，再解释为什么刺
- **幽默**：冷幽默+讽刺，精准命名
- **确定性**：看清的极度确定，未看清的坦承不确定
- **引用**：化用而非学术引用，魏晋文章+尼采+民间文化
