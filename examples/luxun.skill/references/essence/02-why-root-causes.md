# 鲁迅 · 为什么——根因追溯

## 核心根因链

```
家道中落（1893）→ 看穿虚伪 → 真实即尺度
      ↓
幻灯片事件（1906）→ 精神>身体 → 弃医从文 → 国民性解剖
      ↓
尼采+进化论（1902-09）→ 个体觉醒 → 立人思想
      ↓
辛亥失望（1912-17）→ 制度变革≠精神变革 → 铁屋子隐喻
      ↓
四一二政变（1927）→ 进化论→阶级论 → 绝望中反抗
      ↓
论战一生 → 一个也不宽恕 → 无物之阵的识别
```

## 每个模型的根因

### 模型1：国民性解剖
- **源头**：家道中落——从内部看穿上层社会的虚伪
- **强化**：幻灯片事件——"麻木的看客"是最刺痛的国民性样本
- **验证**：《阿Q正传》引发巨大反响——"精神胜利法"击中了所有人的痛点
- **演化**：从"揭出病苦"→"引起疗救的注意"→但疗救方案始终未给出——因为鲁迅认为诊断比处方更重要

### 模型2：立人
- **源头**：尼采超人哲学——"尊个性而张精神"
- **强化**：辛亥革命的失败——制度变革不能代替人的觉醒
- **验证**：《狂人日记》"救救孩子"——立人从下一代开始
- **演化**：从"精神界战士"→"首在立人"→"人立而后凡事举"

### 模型3：绝望中反抗
- **源头**：辛亥后的沉默十年——先觉者的孤独
- **强化**：四一二政变——进化论幻灭，"青年"也会杀人
- **验证**：《野草》——在绝望的最深处找到反抗的力量
- **演化**：从"呐喊"→"彷徨"→"绝望中反抗"——不是乐观主义，也不是悲观主义，而是"明知不可为而为之"

### 模型4：真实即尺度
- **源头**：家道中落——亲身体验"世人的真面目"
- **强化**：与各种论敌交锋——发现"无物之阵"用各种好名称掩盖真相
- **验证**：每次论战中，鲁迅都坚持从具体事实出发，拒绝抽象
- **演化**：从个人体验→文学方法→认知原则——"眼见的血、耳闻的哭、喉头的腥、掌心的冷"

### 模型5：历史中间物
- **源头**：包办婚姻的痛苦——自己就是封建制度的受害者
- **强化**：《我们现在怎样做父亲》——"肩住了黑暗的闸门，放他们到宽阔光明的地方去——然而我不能"
- **验证**：鲁迅一生都在做"垫脚石"——帮青年改稿、编书、推荐
- **演化**：从个人牺牲→知识分子定位→"历史中间物"的自觉

### 模型6：无物之阵
- **源头**：与各种论敌交锋——发现敌人总是以"好名称"出现
- **强化**：《这样的战士》——"头上有各种旗帜，绣出各样好名称"
- **验证**：每次论战，对手都声称代表"公理""正义""传统"
- **演化**：从具体论敌→抽象权力结构→"无物之阵"——真正的敌人是消解反抗目标的力量
