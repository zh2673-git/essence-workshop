# 达尔文 · 人物时间线

| 时间 | 事件 | 对思维的影响 | 形成的模型 |
|------|------|-------------|-----------|
| 1809.2.12 | 出生于什鲁斯伯里 | 家族传统：祖父伊拉斯谟是博物学家和进化思想先驱 | 遗传的好奇心 |
| 1817 | 母亲去世（8岁） | 早期失去母亲，由姐姐抚养 | 内向、独立 |
| 1825-1827 | 爱丁堡大学学医 | 对医学无兴趣，但被博物学吸引；结识格兰特（拉马克主义者） | 博物学兴趣萌芽 |
| 1827 | 在普林尼学会宣读两篇论文 | 第一次科学发现：纠正前人错误 | 观察与证伪 |
| 1828-1831 | 剑桥大学基督学院 | 结识亨斯罗（植物学教授），成为密友 | 导师制学习模式 |
| 1831 | 跟随塞奇威克学习地质学 | 学会地质学思维：深层时间、地层分析 | 均变论思维 |
| 1831.12-1836.10 | 贝格尔号环球航行 | 观察到加拉帕戈斯雀鸟变异、南美化石与现存物种的关系 | 变异与共同祖先 |
| 1837.3 | 开始写"物种演变"笔记本 | 正式开始思考物种变化问题 | 系统性思考 |
| 1838.9.28 | 阅读马尔萨斯《人口论》 | "生存斗争"的概念——自然选择的关键灵感 | 自然选择 |
| 1839 | 出版《贝格尔号航海记》 | 将观察系统化 | 观察优先 |
| 1839.1 | 与表姐埃玛结婚 | 家庭支持系统 | 长期深耕 |
| 1842 | 写出《物种起源》简要提纲 | 理论框架初步成型 | 系统建模 |
| 1844 | 扩展为230页的"论文" | 理论进一步发展，写遗嘱要求死后发表 | 谨慎但坚定 |
| 1846-1854 | 藤壶研究（8年） | 深入理解变异和分类 | 耐心与彻底性 |
| 1856 | 开始写"大书" | 系统性论证 | 证据优先 |
| 1858.6 | 收到华莱士论文 | 震惊，但承认独立发现 | 诚实优先 |
| 1858.7.1 | 林奈学会联合发表 | 确立优先权 | 科学诚信 |
| 1858.7-1859.11 | 写作《物种起源》"摘要" | 13个月完成划时代著作 | 高效输出 |
| 1859.11.24 | 《物种起源》出版 | 首日售罄，引发激烈争论 | 影响力 |
| 1860 | 牛津辩论（赫胥黎vs威尔伯福斯） | 理论获得公开辩护 | 代理人辩护 |
| 1862-1877 | 出版十余部补充专著 | 持续深化理论 | 持续产出 |
| 1871 | 《人类的起源》出版 | 将人类纳入进化框架 | 勇气 |
| 1882.4.19 | 在达恩村家中逝世 | 葬于威斯敏斯特大教堂 | 历史地位 |

## 最新动态（1882年后）
- 达尔文已逝世，但其理论持续发展
- 现代综合进化论（1930s-1940s）将达尔文理论与遗传学结合
- 分子生物学和基因组学进一步验证和发展了进化论
- 达尔文主义仍然是生物学的基础框架
