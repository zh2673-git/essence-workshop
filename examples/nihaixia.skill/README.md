# 倪海厦 · 认知操作系统

> 以阴阳五行为底层操作系统、以经方为执行程序、以"站在外面看到里面"为诊断哲学的归纳式医学认知系统

---

## 这是什么

这不是倪海厦的"知识摘要"，而是他的**认知操作系统**——提取了他怎么看世界（心智模型）、怎么判断（决策启发式）、不同场景怎么切换策略（路由）、怎么表达（表达DNA），以及他为什么这样想（根因追溯）。

**由 [本质蒸馏](https://github.com/zh2673-git/essence-distillation-skill) 系统生成。** 本质蒸馏是一个三阶认知提炼系统，输入人名，输出可运行的认知操作系统。倪海厦是它的第一个蒸馏实例。

---

## 5个核心心智模型

| 模型 | 一句话 | 根因 |
|------|--------|------|
| 阴阳归因 | 一切疾病皆可归因为阴阳失衡 | 车祸卧床读易经→由道入医 |
| 经方至上 | 伤寒论六经辨证是中医核心 | 师承经方→临床反复验证 |
| 治人不治病 | 中医治的是人的状态而非病名 | 易经整体观→非科班不受病名训练 |
| 归纳优于演绎 | 中国文化强在归纳万变不离其宗 | 唐容川影响→西医越分越细病人越多 |
| 术数入医 | 易经/命理/风水与中医底层逻辑相通 | 算命起家→五术兼修→文化闭环 |

## 使用方式

### 在 Trae IDE 中

1. 将 `SKILL.md` 放入项目的 `.claude/skills/` 或 `.trae-cn/skills/` 目录
2. 对话中提到以下任意触发词即可激活：
   - 「用倪海厦的视角」
   - 「倪海厦会怎么看」
   - 「倪师模式」
3. AI会自动按倪海厦的认知操作系统来分析和回答

### 退出角色

```
退出角色 / 切回正常模式
```

## 诚实边界

1. **蒸馏不了临床直觉**：框架能提取思维模式，但"一眼望去病人身患何病"的直觉不能
2. **捕捉不了突变**：这是截止到2012年的认知快照
3. **公开表达≠真实想法**：只能基于公开信息，无法确认私下是否有不同看法
4. **根因追溯有推测成分**：不是倪海厦亲口说的形成原因
5. **不可用于实际诊疗**：这是认知模型，不是医疗指南
